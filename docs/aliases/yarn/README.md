# Yarn Aliases

Shortcuts for the Yarn JavaScript package manager.

## Overview

These aliases are defined in `yarn.aliases.sh` and loaded automatically by Chezmoi.

## Reference

| Alias | Description |
|:---|:---|
| `y` | Yarn shortcut |
| `yi` | Install dependencies |
| `ya` | Add dependency |
| `yad` | Add dev dependency |
| `yrun` | Run script |
| `ystem` | Start project |
