# Python Aliases

Shortcuts for Python development. Interpreter, packages, and virtual environments.

## Overview

These aliases are defined in `python.aliases.sh` and loaded automatically by Chezmoi.

## Reference

| Alias | Command | Description |
|:---|:---|:---|
| `py` | `python` | Python interpreter |
| `pip` | `pip3` | Package installer |
| `v` | `virtualenv` | Virtual environment |
| `uv` | `uv` | Fast package manager |
| `uvpi` | `uv pip install` | Install with uv |
