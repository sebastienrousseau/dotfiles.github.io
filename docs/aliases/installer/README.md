# Installer Aliases

Tools for bootstrapping and remote deployment.

## Overview

The installer aliases provide streamlined commands for setting up your dotfiles environment locally and deploying your configuration to remote servers.

## Reference

| Alias | Description |
|-------|-------------|
| `dot-install` | Run the `install.sh` bootstrap script locally. |
| `dot-teleport` | Teleport your config to a remote server (`dot-teleport user@host`). |
