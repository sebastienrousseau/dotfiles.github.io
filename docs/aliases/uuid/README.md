# UUID Aliases

Generate universally unique identifiers.

## Overview

These aliases are defined in `uuid.aliases.sh` and loaded automatically by Chezmoi.

## Reference

| Alias | Description |
|:---|:---|
| `uuid` | Generate a UUID and copy to clipboard |
