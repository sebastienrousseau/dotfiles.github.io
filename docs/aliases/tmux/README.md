# Tmux Aliases

Manage terminal sessions, windows, and panes with tmux.

## Overview

These aliases are defined in `tmux.aliases.sh` and loaded automatically by Chezmoi.

## Reference

| Alias | Description |
|:---|:---|
| `tm` | Start tmux |
| `tma` | Attach to last session |
| `tmat` | Attach to specific session |
| `tmks` | Kill all sessions except current |
| `tmka` | Kill all sessions |
| `tml` | List all sessions |
| `tmn` | New unnamed session |
| `tms` | New named session |
| `tmr` | Reload tmux configuration |
| `tmls` | List windows |
| `tmlp` | List panes |
| `tmi` | Show tmux info |
