# Terraform Aliases

Shortcuts for Infrastructure as Code with Terraform and Ansible.

## Overview

These aliases are defined in `terraform.aliases.sh` and loaded automatically by Chezmoi.

## Reference

### Terraform

| Alias | Description |
|:---|:---|
| `tf` | Terraform shortcut |
| `tfi` | Initialise configuration |
| `tfp` | Plan changes |
| `tfa` | Apply changes |
| `tfaa` | Apply with auto-approve |
| `tfd` | Destroy infrastructure |
| `tff` | Format configuration |
| `tfv` | Validate configuration |

### Ansible

| Alias | Description |
|:---|:---|
| `ans` | Ansible shortcut |
| `ansp` | Run Ansible Playbook |
| `anslint` | Lint Ansible files |
