# Алиасы Permission

Ярлыки для управления правами файлов и каталогов в вашей shell-среде.

## Обзор

Эти алиасы определены в `permission.aliases.sh` и загружаются автоматически chezmoi. Они обеспечивают быстрый доступ к типичным шаблонам прав и командам владения.

## Справочник

### Числовые ярлыки прав

| Алиас | Описание |
|-------|----------|
| `000` | Установить права: нет чтения, записи или выполнения для пользователя, группы и остальных |
| `400` | Установить права: только чтение для владельца |
| `444` | Установить права: только чтение для всех |
| `600` | Установить права: чтение и запись для владельца |
| `644` | Установить права: чтение для всех, но запись только для владельца |
| `666` | Установить права: чтение и запись для всех |
| `755` | Установить права: чтение, запись и выполнение для владельца, но только чтение и выполнение для группы и остальных |
| `764` | Установить права: чтение и запись для владельца и группы, но только чтение для остальных |
| `777` | Установить права: чтение, запись и выполнение для всех |

### Команды режима файлов

| Алиас | Описание |
|-------|----------|
| `chmod` | Изменить биты режима файла |
| `chmodr` | Рекурсивно изменить биты режима файла |
| `chmodu` | Рекурсивно изменить биты режима файла на текущего пользователя |
| `chmox` | Сделать файл исполняемым |

### Команды владения

| Алиас | Описание |
|-------|----------|
| `chown` | Изменить владельца и группу файла |
| `chownr` | Рекурсивно изменить владельца и группу файла |
| `chownu` | Рекурсивно изменить владельца и группу файла на текущего пользователя |
| `chgrp` | Изменить групповое владение файлами или каталогами |
| `chgrpr` | Рекурсивно изменить групповое владение файлами или каталогами |
| `chgrpu` | Рекурсивно изменить групповое владение файлами или каталогами на текущего пользователя |
