# Go-псевдонимы

Оптимизируйте рабочий процесс разработки на Go с помощью интуитивных ярлыков оболочки.

![Баннер Dotfiles](https://kura.pro/dotfiles/v2/images/titles/title-dotfiles.svg "Баннер Dotfiles")

## Обзор

Эти псевдонимы определены в `go.aliases.sh` и автоматически загружаются chezmoi. Они обеспечивают быстрый доступ к распространённым командам Go для запуска, сборки, тестирования и управления модулями.

## Справочник

| Псевдоним | Команда | Описание |
|----------|---------------|------------------------------------|
| `go` | `go` | Ярлык Go |
| `gor` | `go run` | Запустить Go-программу |
| `gob` | `go build` | Собрать Go-пакет |
| `got` | `go test` | Запустить Go-тесты |
| `gomt` | `go mod tidy` | Очистить зависимости Go-модуля |
| `goget` | `go get` | Скачать и установить Go-пакеты |
