# AI-псевдонимы

Ярлыки для AI и LLM-инструментов. Ускорьте рабочий процесс с AI-поддержкой.

## Обзор

Эти псевдонимы определены в `ai.aliases.sh` и автоматически загружаются Chezmoi.

## Справочник

### GitHub Copilot

| Псевдоним | Описание |
|:---|:---|
| `ghcp` | Ярлык Copilot |
| `ghs` | Предложить код |
| `ghe` | Объяснить код |

### Ollama

| Псевдоним | Описание |
|:---|:---|
| `ol` | Ярлык Ollama |
| `olr` | Запустить модель |
