# Алиасы Update

Поддерживайте систему и пакеты в актуальном состоянии.

## Обзор

Эти алиасы определены в `update.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

### macOS

| Алиас | Описание |
|-------|----------|
| `upd` | Обновить ПО macOS, Homebrew, Casks, App Store, Rust, Ruby, Python и Node.js |

### Linux

| Алиас | Описание |
|-------|----------|
| `open` | Открыть файл или URL в предпочитаемом приложении |
| `pbcopy` | Скопировать в буфер обмена |
| `pbpaste` | Вставить из буфера обмена |
| `upd` | Обновить пакеты Linux, pnpm, Rust, Ruby, Python и Node.js |
