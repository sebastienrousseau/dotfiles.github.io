# Алиасы Tmux

Управляйте терминальными сессиями, окнами и панелями с tmux.

## Обзор

Эти алиасы определены в `tmux.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

| Алиас | Описание |
|-------|----------|
| `tm` | Запустить tmux |
| `tma` | Присоединиться к последней сессии |
| `tmat` | Присоединиться к конкретной сессии |
| `tmks` | Завершить все сессии кроме текущей |
| `tmka` | Завершить все сессии |
| `tml` | Список всех сессий |
| `tmn` | Новая безымянная сессия |
| `tms` | Новая именованная сессия |
| `tmr` | Перезагрузить конфигурацию tmux |
| `tmls` | Список окон |
| `tmlp` | Список панелей |
| `tmi` | Показать информацию tmux |
