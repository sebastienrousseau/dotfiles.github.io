# Find-псевдонимы

Мощные ярлыки поиска файлов с использованием fd, современной альтернативы find.

![Баннер Dotfiles][banner]

## Обзор

Эти псевдонимы предоставляют упрощённый интерфейс к `fd`, быстрой и удобной альтернативе традиционной команде `find`. Определённые в `find.aliases.sh`, они автоматически загружаются chezmoi и предлагают цветной вывод, интуитивный синтаксис и запоминающиеся команды для распространённых операций поиска файлов.

Когда `fd` доступен в вашей системе, команда `find` автоматически использует `fd`, обеспечивая улучшенную производительность и более простой синтаксис по умолчанию.

## Справочник

### Основные команды

| Псевдоним | Команда | Описание |
|-------|---------|-------------|
| `fd` | `fd --color always` | Поиск файлов с цветным выводом |
| `find` | `fd` | Использовать fd как замену find по умолчанию |

### Параметры поиска

| Псевдоним | Команда | Описание |
|-------|---------|-------------|
| `fda` | `fd --absolute-path` | Показать результаты с абсолютными путями |
| `fdc` | `fd --ignore-case` | Выполнить поиск без учёта регистра |
| `fdd` | `fd --list-details` | Показать детальную информацию о файлах |
| `fde` | `fd --extension` | Фильтровать по расширению файла |
| `fdf` | `fd --follow` | Следовать символическим ссылкам при поиске |
| `fdh` | `fd --hidden` | Включить скрытые файлы в результаты |
| `fdn` | `fd --glob` | Сопоставлять файлы с glob-шаблонами |
| `fdo` | `fd --owner` | Фильтровать по владельцу файла |
| `fds` | `fd --size` | Фильтровать по размеру файла |
| `fdu` | `fd --exclude` | Исключить файлы, соответствующие шаблону |
| `fdx` | `fd --exec` | Выполнить команду для каждого результата |

### Служебные команды

| Псевдоним | Команда | Описание |
|-------|---------|-------------|
| `fdv` | `fd --version` | Показать информацию о версии fd |
