# Псевдонимы соответствия

Инструменты оболочки для соответствия нормативным требованиям и обеспечения конфиденциальности.

## Обзор

Псевдонимы соответствия предоставляют необходимые элементы управления для обеспечения конфиденциальности и ведения журналов аудита в вашей оболочке. Эти псевдонимы помогают отключить телеметрию в распространённых CLI-инструментах и отслеживать изменения конфигурации для требований соответствия.

## Справочник

### Элементы управления

| Псевдоним | Описание |
|-------|-------------|
| `privacy-mode` | Отключает телеметрию для CLI-инструментов (Homebrew, Dotnet, Azure и др.) для текущей сессии. |
| `dot-audit` | Отображает последний журнал аудита конфигурации (история Git или журнал аудита). |
