# Псевдонимы использования диска

Быстро отслеживайте и анализируйте потребление хранилища по всей файловой системе.

## Обзор

Псевдонимы использования диска предоставляют оптимизированные команды для мониторинга потребления хранилища в Unix-подобных системах. Определённые в `disk-usage.aliases.sh`, эти ярлыки упрощают распространённые операции `du` и автоматически загружаются chezmoi как часть конфигурации Dotfiles.

## Справочник

В следующей таблице перечислены все доступные псевдонимы использования диска и их функции.

| Псевдоним | Описание |
|-------|-------------|
| `du` | Показать использование диска текущей директории |
| `du1` | Показать размер файлов и директорий в текущей директории |
| `duh` | Показать размер файлов и директорий в удобочитаемом формате |
| `ducks` | Показать 10 самых больших файлов и директорий в текущей директории |
| `dus` | Показать размер в удобочитаемом формате, отсортированный по размеру |
| `dusym` | Показать размер файлов и директорий, включая символические ссылки |
| `dut` | Показать общий размер текущей директории |
