# Clear-псевдонимы

Оптимизируйте рабочий процесс в терминале с помощью быстрых ярлыков очистки экрана.

## Обзор

Псевдонимы clear предоставляют эффективные способы сброса отображения терминала с опциональным показом полезного контекста, такого как содержимое директории, история команд или дерево файлов. Эти псевдонимы определены в `clear.aliases.sh` и автоматически загружаются chezmoi.

## Справочник

| Псевдоним | Описание |
|-------|-------------|
| `c` | Очистить экран терминала |
| `cdw` | Перейти в рабочую директорию |
| `cl` | Очистить экран терминала (псевдоним для `c`) |
| `clc` | Очистить экран и показать содержимое директории |
| `clh` | Очистить экран и показать историю команд |
| `clp` | Очистить экран и показать текущую директорию |
| `clt` | Очистить экран и показать дерево директорий |
| `cpl` | Очистить экран и показать рабочую директорию с содержимым |
| `cplt` | Очистить экран и показать рабочую директорию с деревом |
