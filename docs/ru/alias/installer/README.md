# Псевдонимы установщика

Инструменты для начальной настройки и удалённого развёртывания.

## Обзор

Псевдонимы установщика предоставляют оптимизированные команды для настройки вашего окружения dotfiles локально и развёртывания конфигурации на удалённые серверы.

## Справочник

| Псевдоним | Описание |
|-------|-------------|
| `dot-install` | Запустить скрипт начальной настройки `install.sh` локально. |
| `dot-teleport` | Перенести вашу конфигурацию на удалённый сервер (`dot-teleport user@host`). |
