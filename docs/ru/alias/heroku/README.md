# Heroku-псевдонимы

Ярлыки оболочки для Heroku CLI, определённые в `heroku.aliases.sh` и автоматически загружаемые chezmoi.

## Обзор

Эта коллекция предоставляет более 200 псевдонимов для [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli), охватывая управление приложениями, аддоны, базы данных, пайплайны, контейнеры и многое другое.

## Справочник

### Общие команды

| Псевдоним | Описание |
|-------|-------------|
| `hkk` | Ярлык команды Heroku CLI |
| `hka` | Добавить новых пользователей в приложение |
| `hkh` | Показать справку по Heroku |
| `hkl` | Показать все команды |
| `hkla` | Показать, кто имеет доступ к приложению |
| `hklg` | Показать последние логи |
| `hkn` | Показать уведомления |
| `hks` | Показать текущий статус платформы Heroku |
| `hku` | Обновить Heroku CLI |

### Аддоны

| Псевдоним | Описание |
|-------|-------------|
| `hkada` | Присоединить существующий ресурс аддона к приложению |
| `hkadc` | Создать новый ресурс аддона |
| `hkadd` | Отсоединить ресурс аддона от приложения |
| `hkadi` | Показать детальную информацию об аддоне |
| `hkadk` | Уничтожить ресурс аддона |
| `hkadl` | Показать ваши аддоны и присоединения |
| `hkado` | Открыть панель аддона в браузере |

### Приложения

| Псевдоним | Описание |
|-------|-------------|
| `hkapc` | Создать новое приложение |
| `hkape` | Просмотреть ошибки приложения |
| `hkapfav` | Показать избранные приложения |
| `hkapi` | Показать детальную информацию о приложении |
| `hkapk` | Уничтожить приложение |
| `hkapl` | Показать ваши приложения |
| `hkapo` | Открыть приложение в браузере |
| `hkapr` | Переименовать приложение |
| `hkapt` | Передать приложение другому пользователю или команде |

### Переменные конфигурации

| Псевдоним | Описание |
|-------|-------------|
| `hkcfe` | Интерактивно редактировать переменные конфигурации |
| `hkcfg` | Показать значение одной переменной конфигурации |
| `hkcfs` | Установить переменные конфигурации |
| `hkcfu` | Удалить переменные конфигурации |
| `hkcfv` | Показать переменные конфигурации приложения |

### Контейнеры

| Псевдоним | Описание |
|-------|-------------|
| `hkct` | Использовать контейнеры для сборки и развёртывания |
| `hkctin` | Войти в Heroku Container Registry |
| `hkctout` | Выйти из Heroku Container Registry |
| `hkctpush` | Собрать и отправить Docker-образы |
| `hkctrelease` | Развернуть ранее отправленные образы |

### PostgreSQL

| Псевдоним | Описание |
|-------|-------------|
| `hkpg` | Показать информацию о базе данных |
| `hkpgbk` | Показать резервные копии базы данных |
| `hkpgbkc` | Создать новую резервную копию |
| `hkpgbkr` | Восстановить резервную копию |
| `hkpgpsql` | Открыть psql-оболочку к базе данных |
| `hkpgpull` | Загрузить базу данных Heroku локально |
| `hkpgpush` | Загрузить локальную базу на Heroku |
| `hkpgreset` | Удалить все данные в базе данных |

### Пайплайны

| Псевдоним | Описание |
|-------|-------------|
| `hkpipe` | Показать доступные пайплайны |
| `hkpipea` | Добавить приложение в пайплайн |
| `hkpipec` | Создать новый пайплайн |
| `hkpipei` | Показать приложения в пайплайне |
| `hkpipek` | Уничтожить пайплайн |
| `hkpipep` | Продвинуть релиз в downstream |
| `hkpiper` | Удалить приложение из пайплайна |

### Релизы

| Псевдоним | Описание |
|-------|-------------|
| `hkrel` | Показать релизы приложения |
| `hkreli` | Показать детальную информацию о релизе |
| `hkrelr` | Откатить к предыдущему релизу |

### Запуск

| Псевдоним | Описание |
|-------|-------------|
| `hkrun` | Выполнить одноразовый процесс в dyno Heroku |
| `hkrund` | Выполнить отсоединённый dyno |
