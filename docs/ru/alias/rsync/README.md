# Алиасы Rsync

Синхронизируйте файлы и каталоги между локациями с отслеживанием прогресса.

## Обзор

Эти алиасы определены в `rsync.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

| Алиас | Описание |
|-------|----------|
| `rs` | Rsync с подробным выводом и отображением прогресса |
