# Алиасы Subversion

Ярлыки для операций контроля версий Subversion.

## Обзор

Эти алиасы определены в `subversion.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

| Алиас | Описание |
|-------|----------|
| `sad` | Добавить файлы и каталоги под контроль версий |
| `sau` | Управление кэшированными учётными данными |
| `sbl` | Показать когда каждая строка была изменена |
| `scg` | Связать changelist с файлами |
| `sci` | Закоммитить изменения в репозиторий |
| `scl` | Очистить рабочую копию |
| `sco` | Извлечь рабочую копию |
| `scp` | Копировать файлы и каталоги |
| `sct` | Вывести содержимое файлов или URL |
| `sdi` | Показать локальные изменения или различия |
| `sdl` | Удалить файлы из контроля версий |
| `shp` | Показать справку по командам |
| `sin` | Показать информацию об элементе |
| `sip` | Импортировать неверсионированный файл в репозиторий |
| `slg` | Показать сообщения лога |
| `slk` | Заблокировать пути рабочей копии |
| `sls` | Список записей каталога |
| `smd` | Создать новый каталог под контролем версий |
| `smg` | Слить изменения в рабочую копию |
| `smgi` | Показать информацию о слиянии |
| `smv` | Переместить или переименовать элемент |
| `sp` | Установить значение свойства |
| `spdl` | Удалить свойство |
| `spdt` | Редактировать свойство внешним редактором |
| `spgt` | Вывести значение свойства |
| `sph` | Применить патч |
| `spls` | Список всех свойств |
| `srl` | Переместить рабочую копию на другой URL |
| `srs` | Разрешить конфликты |
| `srsd` | Удалить состояние конфликта |
| `srv` | Восстановить чистое состояние рабочей копии |
| `sst` | Показать статус рабочей копии |
| `ssw` | Переключить рабочую копию на другой URL |
| `sulk` | Разблокировать пути рабочей копии |
| `sup` | Обновить рабочую копию из репозитория |
| `supg` | Обновить формат хранения метаданных |
| `sxp` | Создать неверсионированную копию дерева |
