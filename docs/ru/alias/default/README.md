# Стандартные псевдонимы

Основные ярлыки оболочки для повседневных задач в терминале.

![Баннер Dotfiles][banner]

## Обзор

Стандартные псевдонимы предоставляют набор универсальных ярлыков оболочки, которые упрощают распространённые операции в терминале. Эти псевдонимы определены в `default.aliases.sh` и автоматически загружаются chezmoi во всех поддерживаемых оболочках.

Эти ярлыки охватывают общие утилиты, команды выхода, сетевую диагностику, мониторинг системы и навигацию по файловой системе.

## Справочник

### Общие

| Псевдоним | Команда | Описание |
|-------|---------|-------------|
| `da` | `date "+%Y-%m-%d %A %T %Z"` | Отобразить текущую дату и время |
| `p` | `pwd` | Показать текущую директорию |
| `path` | `echo ${PATH//:/\\n}` | Отобразить переменную PATH построчно |
| `r` | `reload` | Перезагрузить оболочку |
| `t` | `tail -f` | Следить за выводом файла в реальном времени |
| `wk` | `date +%V` | Показать номер текущей недели |

### Выход и завершение

| Псевдоним | Команда | Описание |
|-------|---------|-------------|
| `q` | `exit` | Выйти из оболочки |
| `quit` | `exit` | Выйти из оболочки |
| `bye` | `exit` | Выйти из оболочки |
| `:q` | `exit` | Выйти из оболочки (стиль vim) |
| `halt` | `sudo /sbin/halt` | Остановить систему |
| `poweroff` | `sudo /sbin/shutdown` | Выключить систему |
| `reboot` | `sudo /sbin/reboot` | Перезагрузить систему |

### История

| Псевдоним | Команда | Описание |
|-------|---------|-------------|
| `h` | `history` | Просмотреть историю команд |
| `history` | `fc -il 1` | Показать историю с временными метками ISO 8601 |

### Сеть

| Псевдоним | Команда | Описание |
|-------|---------|-------------|
| `ifconfig` | `sudo ifconfig` | Настроить сетевые интерфейсы |
| `ipinfo` | `ipconfig getpacket en0` | Получить сетевые параметры для en0 |
| `nls` | `sudo lsof -i -P \| grep LISTEN` | Показать активные сетевые прослушиватели |
| `op` | `sudo lsof -i -P` | Показать открытые порты |
| `ping` | `ping -c 5` | Ping с ограничением в 5 пакетов |
| `ports` | `netstat -tulan` | Показать все прослушиваемые порты |

### Мониторинг системы

| Псевдоним | Команда | Описание |
|-------|---------|-------------|
| `top` | `sudo btop` | Интерактивный монитор процессов |
| `spd` | `sudo rm -rf /private/var/log/asl/*` | Очистить файлы журнала ASL |

### Утилиты

| Псевдоним | Команда | Описание |
|-------|---------|-------------|
| `ctf` | `ls -1 \| wc -l` | Подсчитать файлы в текущей директории |
| `qfind` | `find . -name` | Быстрый поиск файлов |
| `reload` | `exec $SHELL -l` | Перезагрузить оболочку |
| `wth` | `curl -s "wttr.in/?format=3"` | Получить текущую погоду |
