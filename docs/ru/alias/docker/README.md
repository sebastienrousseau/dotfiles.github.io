# Docker-псевдонимы

Оптимизируйте рабочий процесс с контейнерами с помощью интуитивных ярлыков для Docker, Compose и Swarm.

![Баннер Dotfiles][banner]

## Обзор

Эти псевдонимы определены в `docker.aliases.sh` и автоматически загружаются chezmoi. Они предоставляют комплексный набор ярлыков для разработки с Docker, охватывая контейнеры, образы, тома, сети, операции Docker Compose и Docker Swarm.

## Справочник

### Базовые команды

| Псевдоним | Описание |
|-------|-------------|
| `dk` | Ярлык Docker |
| `dkv` | Показать версию Docker |
| `dki` | Показать системную информацию |
| `dkl` | Войти в реестр Docker |
| `dklo` | Выйти из реестра Docker |

### Операции с контейнерами

| Псевдоним | Описание |
|-------|-------------|
| `dkps` | Показать запущенные контейнеры |
| `dkpsa` | Показать все контейнеры |
| `dkr` | Запустить команду в новом контейнере |
| `dkri` | Запустить интерактивный контейнер |
| `dkrd` | Запустить контейнер в фоне |
| `dks` | Запустить контейнер |
| `dkst` | Остановить контейнер |
| `dkrs` | Перезапустить контейнер |
| `dkp` | Приостановить контейнер |
| `dkup` | Возобновить контейнер |
| `dkrm` | Удалить контейнер |
| `dkrma` | Удалить все контейнеры |
| `dkrmf` | Принудительно удалить контейнер |

| `dkst` | Показать использование ресурсов контейнера |

| `dkpl` | Загрузить образ из реестра |
| `dkex` | Выполнить команду в контейнере |
| `dkeit` | Выполнить интерактивную команду |
