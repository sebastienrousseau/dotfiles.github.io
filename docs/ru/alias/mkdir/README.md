# Алиасы Mkdir

Упрощайте создание каталогов с помощью интеллектуальных соглашений об именовании.

![Баннер Dotfiles][banner]

## Обзор

Алиасы mkdir предоставляют ярлыки для создания каталогов с автоматическими метками даты и времени. Эти алиасы определены в `mkdir.aliases.sh` и загружаются автоматически chezmoi.

## Справочник

| Алиас | Описание |
|-------|----------|
| `md` | Создать каталог |
| `mdd` | Создать каталог с датой |
| `mdt` | Создать каталог со временем |
| `mde` | Создать каталог примеров с текущей датой |
| `mdn` | Создать каталог заметок с текущей датой |
| `mdw` | Создать рабочий каталог с текущей датой |
