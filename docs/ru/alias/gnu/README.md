# GNU Coreutils-псевдонимы

Ярлыки для основных Unix-утилит для ускорения работы в командной строке.

![Баннер Dotfiles][banner]

## Обзор

Псевдонимы GNU Coreutils предоставляют сокращённые команды для стандартных Unix-утилит. Определённые в `gnu.aliases.sh` и автоматически загружаемые chezmoi, эти псевдонимы упрощают операции с файлами, обработку текста и задачи проверки данных.

## Справочник

### Управление файлами

| Псевдоним | Описание |
|-------|-------------|
| `basename` | Извлечь имя файла из пути |
| `cp` | Копировать файлы и директории |
| `dirname` | Извлечь путь директории |
| `ln` | Создать ссылки между файлами |
| `loname` | Показать имя ссылки |
| `ls` | Показать содержимое директории |
| `mkdir` | Создать директории |
| `mkfifo` | Создать именованные каналы (FIFO) |
| `mknod` | Создать блочные или символьные специальные файлы |
| `mv` | Переместить или переименовать файлы или директории |
| `pathchk` | Проверить валидность имени файла |
| `pwd` | Показать имя рабочей директории |
| `readlink` | Показать разрешённые символические ссылки |
| `realpath` | Показать разрешённый физический путь |
| `rm` | Удалить файлы или директории |
| `rmdir` | Удалить пустые директории |
| `unlink` | Удалить файлы или директории |

### Обработка текста

| Псевдоним | Описание |
|-------|-------------|
| `awk` | Язык сканирования и обработки шаблонов |
| `cat` | Объединить и отобразить файлы |
| `csplit` | Разбить файл на части по контексту |
| `cut` | Удалить секции из каждой строки файлов |
| `diff` | Сравнить файлы построчно |
| `fold` | Перенести каждую строку по указанной ширине |
| `grep` | Вывести строки, соответствующие шаблону |
| `head` | Вывести первую часть файлов |
| `nl` | Пронумеровать строки файлов |
| `paste` | Объединить строки файлов |
| `patch` | Применить diff-файл к оригиналу |
| `ptx` | Создать перестановочный индекс содержимого файла |
| `sed` | Потоковый редактор для фильтрации и преобразования текста |
| `sort` | Сортировать строки текстовых файлов |
| `split` | Разбить файл на части |
| `tail` | Вывести последнюю часть файлов |
| `tr` | Преобразовать или удалить символы |

### Утилиты контрольных сумм

| Псевдоним | Описание |
|-------|-------------|
| `b2sum` | Вычислить или проверить дайджесты BLAKE2 |
| `cksum` | Вычислить контрольную сумму CRC и размер |
| `sha1sum` | Вычислить или проверить дайджесты SHA1 |
| `sha224sum` | Вычислить или проверить дайджесты SHA224 |
| `sha256sum` | Вычислить или проверить дайджесты SHA256 |
| `sha384sum` | Вычислить или проверить дайджесты SHA384 |
| `sha512sum` | Вычислить или проверить дайджесты SHA512 |

### Утилиты кодирования

| Псевдоним | Описание |
|-------|-------------|
| `base32` | Вывести или преобразовать данные base32 |
| `base64` | Закодировать или декодировать данные base64 |
| `basenc` | Закодировать или декодировать данные в различных форматах |
