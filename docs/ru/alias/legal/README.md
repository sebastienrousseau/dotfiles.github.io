# Юридические псевдонимы

Инструменты оболочки для управления соответствием открытому исходному коду, сканирования лицензий и атрибуции.

## Обзор

Юридические псевдонимы предоставляют оптимизированный рабочий процесс для поддержания соответствия открытому исходному коду в ваших проектах. Эти инструменты обрабатывают сканирование лицензий, управление заголовками авторских прав и атрибуцию зависимостей.

## Справочник

### Сканирование лицензий

| Псевдоним | Описание | Тип |
|-------|-------------|------|
| `fossology-start` | Запустить локальный сервер FOSSology на порту 8081 | Docker |
| `fossology-stop` | Остановить сервер FOSSology | Docker |
| `license-scan` | Быстрое сканирование лицензий текущей директории через Trivy | Binary |

### Заголовки авторских прав

| Псевдоним | Описание | Тип |
|-------|-------------|------|
| `add-headers` | Рекурсивно добавить заголовки лицензии MIT ко всем исходным файлам | Docker (google/addlicense) |

### Атрибуция

| Псевдоним | Описание | Тип |
|-------|-------------|------|
| `gen-notice` | Сгенерировать файл NOTICE для зависимостей (первоначально поддержка Go) | Docker |

### Вклад

| Псевдоним | Описание | Тип |
|-------|-------------|------|
| `check-cla` | Отслеживать проверки GitHub PR, включая статус CLA | CLI (gh) |

## Требования

- **Docker** - Требуется для изоляции инструментов соответствия
- **GitHub CLI (gh)** - Требуется для проверки PR и CLA
- **Trivy** - Автоматически устанавливается или предлагается для быстрого сканирования
