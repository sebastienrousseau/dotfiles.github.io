# Алиасы Pnpm

Ярлыки для быстрого, экономящего диск пакетного менеджера.

![Баннер Dotfiles][banner]

## Обзор

Эти алиасы определены в `pnpm.aliases.sh` и загружаются автоматически chezmoi. Они обеспечивают быстрый доступ к типичным командам [pnpm](https://pnpm.io/) для управления пакетами Node.js.

## Справочник

### Инициализация проекта

| Алиас | Описание |
|-------|----------|
| `pni` | Инициализировать новый проект |
| `pnc` | Создать новый проект |
| `pnin` | Установить зависимости проекта |
| `pnun` | Удалить зависимости проекта |

### Разработка

| Алиас | Описание |
|-------|----------|
| `pnb` | Собрать проект |
| `pnrun` | Запустить скрипт из проекта |
| `pnd` | Запустить проект в режиме dev |
| `pns` | Запустить проект в режиме serve |
| `pnst` | Запустить проект |
| `pnsv` | Запустить проект в режиме server |

### Тестирование и качество

| Алиас | Описание |
|-------|----------|
| `pnt` | Тестировать проект |
| `pntc` | Тестировать проект с покрытием |
| `pnln` | Lint проекта |
| `pndoc` | Сгенерировать документацию проекта |

### Управление зависимостями

| Алиас | Описание |
|-------|----------|
| `pnls` | Список всех зависимостей |
| `pnout` | Проверить устаревшие зависимости |
| `pnwhy` | Проверить почему установлена зависимость |
| `pnad` | Добавить dev зависимость |
| `pnap` | Добавить peer зависимость |
| `pnrm` | Удалить зависимость |
| `pnup` | Обновить зависимость |
| `pnui` | Интерактивно обновить зависимость |
| `pnuil` | Интерактивно обновить зависимость до последней |

### Глобальные пакеты

| Алиас | Описание |
|-------|----------|
| `pnga` | Добавить глобальную зависимость |
| `pngls` | Список всех глобальных зависимостей |
| `pngrm` | Удалить глобальную зависимость |
| `pngu` | Обновить глобальную зависимость |

### Публикация и утилиты

| Алиас | Описание |
|-------|----------|
| `pnpub` | Опубликовать проект |
| `pnx` | Сокращение для pnpx |
| `pnh` | Показать справку |
