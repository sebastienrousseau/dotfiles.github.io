# Алиасы Terraform

Ярлыки для инфраструктуры как кода с Terraform и Ansible.

## Обзор

Эти алиасы определены в `terraform.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

### Terraform

| Алиас | Описание |
|-------|----------|
| `tf` | Сокращение Terraform |
| `tfi` | Инициализировать конфигурацию |
| `tfp` | Планировать изменения |
| `tfa` | Применить изменения |
| `tfaa` | Применить с автоподтверждением |
| `tfd` | Уничтожить инфраструктуру |
| `tff` | Форматировать конфигурацию |
| `tfv` | Проверить конфигурацию |

### Ansible

| Алиас | Описание |
|-------|----------|
| `ans` | Сокращение Ansible |
| `ansp` | Запустить Ansible Playbook |
| `anslint` | Проверить Ansible файлы |
