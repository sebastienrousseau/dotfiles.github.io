# Псевдонимы редакторов

Ярлыки для запуска и настройки предпочитаемого текстового редактора.

![Баннер Dotfiles](https://kura.pro/dotfiles/v2/images/titles/title-dotfiles.svg "Баннер Dotfiles")

## Обзор

Псевдонимы редакторов предоставляют единый интерфейс для редактирования текста в различных приложениях. Определённые в `editor.aliases.sh` и автоматически загружаемые chezmoi, эти ярлыки работают совместно с `editor.sh` для определения и настройки лучшего доступного редактора в вашей системе.

Система поддерживает Neovim, Visual Studio Code, Vim, Nano, Emacs, Sublime Text и Atom. Универсальные псевдонимы адаптируются к активному редактору, а специализированные псевдонимы предлагают функциональность для конкретного редактора.

## Справочник

### Универсальные псевдонимы

Эти псевдонимы работают с любым редактором, настроенным через `editor.sh`.

| Псевдоним | Описание |
|-------|-------------|
| `e` | Быстрая команда редактирования |
| `edit` | Стандартная команда редактирования |
| `editor` | Полная команда редактора |
| `mate` | Команда в стиле TextMate |
| `n` | Короткий псевдоним редактирования |
| `v` | Псевдоним редактирования в стиле Vim |

### Псевдонимы Neovim

Доступны, когда Neovim является основным редактором.

| Псевдоним | Описание |
|-------|-------------|
| `vi` | Перенаправлен на Neovim |
| `vim` | Перенаправлен на Neovim |
| `nvimrc` | Редактировать конфигурацию Neovim Vimscript |
| `nvimlua` | Редактировать конфигурацию Neovim Lua |
| `nvimconf` | Открыть директорию конфигурации Neovim |

### Псевдонимы Visual Studio Code

Доступны, когда установлен VS Code.

| Псевдоним | Описание |
|-------|-------------|
| `vsc` | Запустить VS Code |
| `vsca` | Добавить папку в текущее окно |
| `vscd` | Сравнить два файла |
| `vscn` | Открыть новое окно |
| `vscr` | Переиспользовать окно при открытии файлов |
| `vscex` | Установить расширение VS Code |
| `vsclist` | Показать установленные расширения |

### Псевдонимы Vim

Доступны, когда Vim является основным редактором.

| Псевдоним | Описание |
|-------|-------------|
| `vi` | Перенаправлен на Vim |
| `vimrc` | Редактировать файл конфигурации Vim |
| `vimconf` | Открыть директорию конфигурации Vim |

### Псевдонимы Nano

Доступны, когда Nano является основным редактором.

| Псевдоним | Описание |
|-------|-------------|
| `nanorc` | Редактировать файл конфигурации Nano |
| `ne` | Улучшенный Nano с номерами строк и плавной прокруткой |

### Псевдонимы Emacs

Доступны, когда установлен Emacs.

| Псевдоним | Описание |
|-------|-------------|
| `em` | Запустить Emacs |
| `emacs-nw` | Запустить Emacs в терминальном режиме |
| `emacsc` | Запустить клиент Emacs |
| `emacsrc` | Редактировать файл конфигурации Emacs |
| `et` | Быстрый терминальный Emacs |

### Псевдонимы Sublime Text

Доступны, когда установлен Sublime Text.

| Псевдоним | Описание |
|-------|-------------|
| `st` | Запустить Sublime Text |
| `stt` | Открыть текущую директорию в Sublime Text |
| `stn` | Открыть в новом окне Sublime Text |
