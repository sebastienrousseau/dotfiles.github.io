# Алиасы UUID

Генерируйте универсально уникальные идентификаторы.

## Обзор

Эти алиасы определены в `uuid.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

| Алиас | Описание |
|-------|----------|
| `uuid` | Сгенерировать UUID и скопировать в буфер обмена |
