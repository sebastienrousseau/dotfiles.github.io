# Алиасы Python

Ярлыки для разработки на Python. Интерпретатор, пакеты и виртуальные окружения.

## Обзор

Эти алиасы определены в `python.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

| Алиас | Команда | Описание |
|-------|---------|----------|
| `py` | `python` | Интерпретатор Python |
| `pip` | `pip3` | Установщик пакетов |
| `v` | `virtualenv` | Виртуальное окружение |
| `uv` | `uv` | Быстрый пакетный менеджер |
| `uvpi` | `uv pip install` | Установить с помощью uv |
