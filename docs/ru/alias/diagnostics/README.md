# Псевдонимы диагностики

Инструменты для самовосстановления и проверки состояния системы.

## Обзор

Псевдонимы диагностики предоставляют команды для мониторинга состояния системы, обнаружения отклонений конфигурации и автоматического восстановления вашего окружения dotfiles. Эти утилиты интегрируются с chezmoi для обеспечения синхронизации локальных файлов с управляемым состоянием.

## Справочник

### Состояние и восстановление

| Псевдоним | Описание |
|-------|-------------|
| `doc`, `dot-doctor` | Запустить скрипт проверки состояния системы (`doctor.sh`). |
| `drift`, `dot-drift` | Проверить, отклонились ли локальные файлы от управляемого состояния (`chezmoi verify`). |
| `heal`, `dot-heal` | Применить управляемое состояние для исправления отклонений (`chezmoi apply`). |
| `doc-full` | Запустить doctor с расширенной отладочной информацией о путях. |
