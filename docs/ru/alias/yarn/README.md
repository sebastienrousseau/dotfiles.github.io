# Алиасы Yarn

Ярлыки для пакетного менеджера JavaScript Yarn.

## Обзор

Эти алиасы определены в `yarn.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

| Алиас | Описание |
|-------|----------|
| `y` | Сокращение Yarn |
| `yi` | Установить зависимости |
| `ya` | Добавить зависимость |
| `yad` | Добавить dev зависимость |
| `yrun` | Запустить скрипт |
| `ystem` | Запустить проект |
