# Алиасы PS

Мониторинг и управление запущенными процессами с помощью простых ярлыков.

## Обзор

Эти алиасы определены в `ps.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

| Алиас | Описание |
|-------|----------|
| `pid` | Показать UID, PID, родительский PID, использование CPU, время запуска, TTY и команду |
| `ps` | Показать все процессы |
| `psa` | Список всех процессов |
