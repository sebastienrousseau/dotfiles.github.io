# Алиасы Sudo

Выполняйте команды с повышенными привилегиями.

## Обзор

Эти алиасы определены в `sudo.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

| Алиас | Описание |
|-------|----------|
| `root` | Выполнить команду от суперпользователя |
| `s` | Выполнить команду от суперпользователя |
| `su` | Выполнить команду от суперпользователя |
| `sudo` | Выполнить команду от суперпользователя |
