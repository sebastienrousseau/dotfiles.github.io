# Псевдонимы Lua

Быстрый доступ к интерпретаторам Lua REPL с помощью интуитивных сокращений оболочки.

![Dotfiles banner](https://kura.pro/dotfiles/v2/images/titles/title-dotfiles.svg "Dotfiles banner")

## Обзор

Эти псевдонимы определены в `lua.aliases.sh` и автоматически загружаются chezmoi. Они обеспечивают быстрый доступ к различным интерпретаторам Lua, доступным в вашей системе.

## Справочник

| Псевдоним    | Команда    | Описание                                 |
|--------------|------------|------------------------------------------|
| `lua-repl`   | `lua`      | Запуск стандартного Lua REPL             |
| `luajit-repl`| `luajit`   | Запуск LuaJIT REPL (быстрый JIT)         |
| `lua51`      | `lua5.1`   | Запуск интерпретатора Lua 5.1            |

## Примечания

- Каждый псевдоним доступен только если соответствующий интерпретатор установлен в вашей системе.
- LuaJIT предоставляет более быструю JIT-компилируемую среду Lua, совместимую с Lua 5.1.
- Используйте `lua51`, когда вам специально нужна совместимость с Lua 5.1.
