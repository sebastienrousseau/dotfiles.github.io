# CD-псевдонимы

Быстрее перемещайтесь по файловой системе. Закладки, история и умные ярлыки.

## Обзор

Эти псевдонимы определены в `cd.aliases.sh` и автоматически загружаются Chezmoi.

## Справочник

### Ярлыки навигации

| Псевдоним | Описание |
|:---|:---|
| `-` | Переключиться на предыдущую директорию |
| `..` | Подняться на один уровень |
| `...` | Подняться на два уровня |
| `....` | Подняться на три уровня |
| `.....` | Подняться на четыре уровня |
| `hom` | Перейти в домашнюю директорию |

### Ярлыки директорий

| Псевдоним | Путь | Описание |
|:---|:---|:---|
| `app` | `~/Applications` | Приложения |
| `cod` | `~/Code` | Директория кода |
| `dsk` | `~/Desktop` | Рабочий стол |
| `doc` | `~/Documents` | Документы |
| `dot` | `~/.dotfiles` | Dotfiles |
| `dwn` | `~/Downloads` | Загрузки |
| `mus` | `~/Music` | Музыка |
| `pic` | `~/Pictures` | Изображения |
| `vid` | `~/Videos` | Видео |

### Системные директории

| Псевдоним | Путь | Описание |
|:---|:---|:---|
| `etc` | `/etc` | Системная конфигурация |
| `var` | `/var` | Переменные файлы |
| `tmp` | `/tmp` | Временные файлы |
| `usr` | `/usr` | Пользовательские программы |

### Закладки

| Команда | Псевдоним | Описание |
|:---|:---|:---|
| `bookmark [name]` | `bm` | Создать закладку |
| `bookmark_list` | `bml` | Показать закладки |
| `bookmark_update` | `bmu` | Обновить закладку |
| `bookmark_remove` | `bmr` | Удалить закладку |
| `goto <name>` | `bmg` | Перейти к закладке |

```bash
bm work-project      # Создать закладку
bmg work-project     # Перейти к закладке
bml                  # Показать все закладки
```

### История

| Команда | Псевдоним | Описание |
|:---|:---|:---|
| `dirhistory` | `dh` | Показать историю директорий |
| `lwd` | `ld` | Вернуться в последнюю директорию |

### Расширенные функции

| Команда | Псевдоним | Описание |
|:---|:---|:---|
| `mkcd <dir>` | `mcd` | Создать и войти в директорию |
| `proj` | `pr` | Перейти к корню проекта |
| `pushd` | `pd` | Добавить директорию в стек |
| `popd` | — | Извлечь из стека |
| `dirs` | — | Показать стек директорий |

## Конфигурация

Настройте поведение с помощью переменных окружения:

```bash
export SHOW_HIDDEN_FILES=true
export ENABLE_COLOR_OUTPUT=true
export AUTO_LIST_AFTER_CD=true
export MAX_RECENT_DIRS=15
export RESTORE_LAST_DIR=true
```

## Помощь

Выполните `cdhelp` для просмотра всех команд или `cdversion` для информации о версии.
