# Алиасы Wget

Загружайте файлы из интернета с поддержкой возобновления.

## Обзор

Эти алиасы определены в `wget.aliases.sh` и загружаются автоматически Chezmoi.

## Справочник

| Алиас | Описание |
|-------|----------|
| `wg` | Сокращение Wget |
| `wgc` | Wget с возобновлением |
| `wge` | Wget с robots=off |
| `wget` | Wget с возобновлением |
