# Clear एलियास

त्वरित स्क्रीन-क्लियरिंग शॉर्टकट के साथ अपने टर्मिनल वर्कफ़्लो को सुव्यवस्थित करें।

## अवलोकन

clear एलियास आपके टर्मिनल डिस्प्ले को रीसेट करने के कुशल तरीके प्रदान करते हैं जबकि वैकल्पिक रूप से डायरेक्टरी सामग्री, कमांड हिस्ट्री, या फाइल ट्री जैसी उपयोगी संदर्भ दिखाते हैं। ये एलियास `clear.aliases.sh` में परिभाषित हैं और chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

| एलियास | विवरण |
|-------|-------------|
| `c` | टर्मिनल स्क्रीन साफ करें |
| `cdw` | वर्कस्पेस डायरेक्टरी में बदलें |
| `cl` | टर्मिनल स्क्रीन साफ करें (`c` के लिए एलियास) |
| `clc` | स्क्रीन साफ करें और डायरेक्टरी सामग्री सूची दिखाएं |
| `clh` | स्क्रीन साफ करें और कमांड हिस्ट्री दिखाएं |
| `clp` | स्क्रीन साफ करें और वर्किंग डायरेक्टरी प्रिंट करें |
| `clt` | स्क्रीन साफ करें और डायरेक्टरी ट्री दिखाएं |
| `cpl` | स्क्रीन साफ करें और वर्किंग डायरेक्टरी सामग्री के साथ दिखाएं |
| `cplt` | स्क्रीन साफ करें और वर्किंग डायरेक्टरी ट्री के साथ दिखाएं |
