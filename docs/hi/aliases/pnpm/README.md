# Pnpm एलियास

तेज़, डिस्क-कुशल पैकेज मैनेजर के लिए शॉर्टकट।

## अवलोकन

ये एलियास `pnpm.aliases.sh` में परिभाषित हैं और chezmoi द्वारा स्वचालित रूप से लोड होते हैं। वे Node.js पैकेज मैनेजमेंट के लिए सामान्य [pnpm](https://pnpm.io/) कमांड तक त्वरित पहुंच प्रदान करते हैं।

## संदर्भ

### प्रोजेक्ट इनिशियलाइज़ेशन

| एलियास | विवरण |
|-------|-------------|
| `pni` | नया प्रोजेक्ट इनिशियलाइज़ करें |
| `pnc` | नया प्रोजेक्ट बनाएं |
| `pnin` | प्रोजेक्ट डिपेंडेंसी इंस्टॉल करें |
| `pnun` | प्रोजेक्ट डिपेंडेंसी अनइंस्टॉल करें |

### डेवलपमेंट

| एलियास | विवरण |
|-------|-------------|
| `pnb` | प्रोजेक्ट बिल्ड करें |
| `pnrun` | प्रोजेक्ट से स्क्रिप्ट चलाएं |
| `pnd` | प्रोजेक्ट को dev मोड में चलाएं |
| `pns` | प्रोजेक्ट को serve मोड में चलाएं |
| `pnst` | प्रोजेक्ट स्टार्ट करें |

### टेस्टिंग और क्वालिटी

| एलियास | विवरण |
|-------|-------------|
| `pnt` | प्रोजेक्ट टेस्ट करें |
| `pntc` | कवरेज के साथ प्रोजेक्ट टेस्ट करें |
| `pnln` | प्रोजेक्ट lint करें |
| `pndoc` | प्रोजेक्ट डॉक्यूमेंटेशन जनरेट करें |

### डिपेंडेंसी मैनेजमेंट

| एलियास | विवरण |
|-------|-------------|
| `pnls` | सभी डिपेंडेंसी सूचीबद्ध करें |
| `pnout` | पुरानी डिपेंडेंसी जांचें |
| `pnwhy` | जांचें कि डिपेंडेंसी क्यों इंस्टॉल है |
| `pnad` | dev डिपेंडेंसी जोड़ें |
| `pnrm` | डिपेंडेंसी हटाएं |
| `pnup` | डिपेंडेंसी अपडेट करें |

### ग्लोबल पैकेज

| एलियास | विवरण |
|-------|-------------|
| `pnga` | ग्लोबल डिपेंडेंसी जोड़ें |
| `pngls` | सभी ग्लोबल डिपेंडेंसी सूचीबद्ध करें |
| `pngrm` | ग्लोबल डिपेंडेंसी हटाएं |
| `pngu` | ग्लोबल डिपेंडेंसी अपडेट करें |

### पब्लिशिंग और यूटिलिटीज

| एलियास | विवरण |
|-------|-------------|
| `pnpub` | प्रोजेक्ट पब्लिश करें |
| `pnx` | pnpx के लिए शॉर्टकट |
| `pnh` | सहायता दिखाएं |
