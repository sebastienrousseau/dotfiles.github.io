# Tmux एलियास

tmux के साथ टर्मिनल सेशन, विंडो और पेन प्रबंधित करें।

## अवलोकन

ये एलियास `tmux.aliases.sh` में परिभाषित हैं और Chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

| एलियास | विवरण |
|:---|:---|
| `tm` | tmux शुरू करें |
| `tma` | अंतिम सेशन से जुड़ें |
| `tmat` | विशिष्ट सेशन से जुड़ें |
| `tmks` | वर्तमान के अलावा सभी सेशन बंद करें |
| `tmka` | सभी सेशन बंद करें |
| `tml` | सभी सेशन सूचीबद्ध करें |
| `tmn` | नया अनाम सेशन |
| `tms` | नया नामित सेशन |
| `tmr` | tmux कॉन्फ़िगरेशन रीलोड करें |
| `tmls` | विंडो सूचीबद्ध करें |
| `tmlp` | पेन सूचीबद्ध करें |
| `tmi` | tmux जानकारी दिखाएं |
