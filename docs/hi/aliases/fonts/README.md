# Font एलियास

सिस्टम फ़ॉन्ट्स और कैश प्रबंधित करने के लिए शेल एलियास।

## अवलोकन

Font एलियास सामान्य फ़ॉन्ट मैनेजमेंट कार्यों के लिए सुविधाजनक शॉर्टकट प्रदान करते हैं। ये कमांड नए फ़ॉन्ट्स इंस्टॉल करने के बाद फ़ॉन्ट कैश अपडेट करने और आपके सिस्टम पर सभी उपलब्ध फ़ॉन्ट फैमिली सूचीबद्ध करने को सरल करते हैं।

## संदर्भ

### कमांड

| एलियास | कमांड | विवरण |
|-------|---------|-------------|
| `update-fonts` | `fc-cache -fv` | सिस्टम फ़ॉन्ट कैश अपडेट करता है |
| `list-fonts` | — | सभी इंस्टॉल फ़ॉन्ट फैमिली सूचीबद्ध करता है |

### इंस्टॉल फ़ॉन्ट्स

निम्नलिखित फ़ॉन्ट्स डिफ़ॉल्ट रूप से कॉन्फ़िगर हैं:

| फ़ॉन्ट | उद्देश्य |
|------|---------|
| JetBrainsMono Nerd Font | प्राइमरी टर्मिनल फ़ॉन्ट |
| Symbols Nerd Font | आइकन फॉलबैक |
