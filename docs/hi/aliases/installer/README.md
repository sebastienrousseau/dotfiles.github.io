# Installer एलियास

बूटस्ट्रैपिंग और रिमोट डिप्लॉयमेंट के लिए टूल्स।

## अवलोकन

installer एलियास आपके dotfiles एनवायरनमेंट को लोकल में सेटअप करने और आपकी कॉन्फ़िगरेशन को रिमोट सर्वर पर डिप्लॉय करने के लिए सुव्यवस्थित कमांड प्रदान करते हैं।

## संदर्भ

| एलियास | विवरण |
|-------|-------------|
| `dot-install` | लोकल में `install.sh` बूटस्ट्रैप स्क्रिप्ट चलाएं। |
| `dot-teleport` | अपनी कॉन्फ़िग को रिमोट सर्वर पर टेलीपोर्ट करें (`dot-teleport user@host`)। |
