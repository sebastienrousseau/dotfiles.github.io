# Subversion एलियास

Subversion वर्जन कंट्रोल ऑपरेशन के लिए शॉर्टकट।

## अवलोकन

ये एलियास `subversion.aliases.sh` में परिभाषित हैं और Chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

| एलियास | विवरण |
|:---|:---|
| `sad` | वर्जन कंट्रोल में फाइलें और डायरेक्टरी जोड़ें |
| `sau` | कैश ऑथेंटिकेशन क्रेडेंशियल प्रबंधित करें |
| `sbl` | प्रत्येक लाइन कब अंतिम बार बदली दिखाएं |
| `sci` | रिपॉजिटरी में परिवर्तन कमिट करें |
| `scl` | वर्किंग कॉपी साफ करें |
| `sco` | वर्किंग कॉपी चेकआउट करें |
| `scp` | फाइलें और डायरेक्टरी कॉपी करें |
| `sdi` | लोकल परिवर्तन या अंतर प्रदर्शित करें |
| `sdl` | वर्जन कंट्रोल से फाइलें हटाएं |
| `sin` | आइटम के बारे में जानकारी प्रदर्शित करें |
| `slg` | लॉग मैसेज दिखाएं |
| `slk` | वर्किंग कॉपी पाथ लॉक करें |
| `sls` | डायरेक्टरी एंट्री सूचीबद्ध करें |
| `smd` | वर्जन कंट्रोल के तहत नई डायरेक्टरी बनाएं |
| `smg` | वर्किंग कॉपी में परिवर्तन मर्ज करें |
| `smv` | आइटम मूव या रीनेम करें |
| `srs` | कॉन्फ्लिक्ट हल करें |
| `srv` | प्रिस्टीन वर्किंग कॉपी स्थिति रिस्टोर करें |
| `sst` | वर्किंग कॉपी की स्थिति प्रिंट करें |
| `sulk` | वर्किंग कॉपी पाथ अनलॉक करें |
| `sup` | रिपॉजिटरी से वर्किंग कॉपी अपडेट करें |
