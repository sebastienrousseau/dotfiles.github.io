# Python एलियास

Python डेवलपमेंट के लिए शॉर्टकट। इंटरप्रेटर, पैकेज, और वर्चुअल एनवायरनमेंट।

## अवलोकन

ये एलियास `python.aliases.sh` में परिभाषित हैं और Chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

| एलियास | कमांड | विवरण |
|:---|:---|:---|
| `py` | `python` | Python इंटरप्रेटर |
| `pip` | `pip3` | पैकेज इंस्टॉलर |
| `v` | `virtualenv` | वर्चुअल एनवायरनमेंट |
| `uv` | `uv` | तेज़ पैकेज मैनेजर |
| `uvpi` | `uv pip install` | uv के साथ इंस्टॉल करें |
