# Update एलियास

अपने सिस्टम और पैकेज को अपडेट रखें।

## अवलोकन

ये एलियास `update.aliases.sh` में परिभाषित हैं और Chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

### macOS

| एलियास | विवरण |
|:---|:---|
| `upd` | macOS सॉफ़्टवेयर, Homebrew, Casks, App Store, Rust, Ruby, Python, और Node.js अपडेट करें |

### Linux

| एलियास | विवरण |
|:---|:---|
| `open` | फ़ाइल या URL को पसंदीदा एप्लिकेशन में खोलें |
| `pbcopy` | क्लिपबोर्ड में कॉपी करें |
| `pbpaste` | क्लिपबोर्ड से पेस्ट करें |
| `upd` | Linux पैकेज, pnpm, Rust, Ruby, Python, और Node.js अपडेट करें |
