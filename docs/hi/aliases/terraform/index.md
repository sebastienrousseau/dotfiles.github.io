# Terraform एलियास

Terraform और Ansible के साथ इन्फ्रास्ट्रक्चर एज़ कोड के लिए शॉर्टकट।

## अवलोकन

ये एलियास `terraform.aliases.sh` में परिभाषित हैं और Chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

### Terraform

| एलियास | विवरण |
|:---|:---|
| `tf` | Terraform शॉर्टकट |
| `tfi` | कॉन्फ़िगरेशन इनिशियलाइज़ करें |
| `tfp` | परिवर्तन प्लान करें |
| `tfa` | परिवर्तन लागू करें |
| `tfaa` | ऑटो-अप्रूव के साथ लागू करें |
| `tfd` | इन्फ्रास्ट्रक्चर नष्ट करें |
| `tff` | कॉन्फ़िगरेशन फ़ॉर्मेट करें |
| `tfv` | कॉन्फ़िगरेशन वैलिडेट करें |

### Ansible

| एलियास | विवरण |
|:---|:---|
| `ans` | Ansible शॉर्टकट |
| `ansp` | Ansible Playbook चलाएं |
| `anslint` | Ansible फ़ाइलें लिंट करें |
