# Default एलियास

रोज़मर्रा के टर्मिनल कार्यों के लिए आवश्यक शेल शॉर्टकट।

## अवलोकन

default एलियास यूनिवर्सल शेल शॉर्टकट का संग्रह प्रदान करते हैं जो सामान्य टर्मिनल ऑपरेशन को सुव्यवस्थित करते हैं। ये एलियास `default.aliases.sh` में परिभाषित हैं और सभी समर्थित शेल में chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

ये शॉर्टकट सामान्य यूटिलिटीज, एग्जिट कमांड, नेटवर्क डायग्नोस्टिक्स, सिस्टम मॉनिटरिंग, और फाइल सिस्टम नेविगेशन को कवर करते हैं।

## संदर्भ

### सामान्य

| एलियास | कमांड | विवरण |
|-------|---------|-------------|
| `da` | `date "+%Y-%m-%d %A %T %Z"` | वर्तमान तिथि और समय प्रदर्शित करें |
| `p` | `pwd` | वर्किंग डायरेक्टरी प्रिंट करें |
| `path` | `echo ${PATH//:/\\n}` | PATH वेरिएबल को नई लाइनों पर प्रदर्शित करें |
| `r` | `reload` | शेल रीलोड करें |
| `t` | `tail -f` | रियल टाइम में फाइल आउटपुट फॉलो करें |
| `wk` | `date +%V` | वर्तमान सप्ताह संख्या दिखाएं |

### एग्जिट और शटडाउन

| एलियास | कमांड | विवरण |
|-------|---------|-------------|
| `q` | `exit` | शेल से बाहर निकलें |
| `quit` | `exit` | शेल से बाहर निकलें |
| `bye` | `exit` | शेल से बाहर निकलें |
| `:q` | `exit` | शेल से बाहर निकलें (vim स्टाइल) |
| `halt` | `sudo /sbin/halt` | सिस्टम हॉल्ट करें |
| `poweroff` | `sudo /sbin/shutdown` | सिस्टम पावर ऑफ करें |
| `reboot` | `sudo /sbin/reboot` | सिस्टम रीबूट करें |

### हिस्ट्री

| एलियास | कमांड | विवरण |
|-------|---------|-------------|
| `h` | `history` | कमांड हिस्ट्री देखें |
| `history` | `fc -il 1` | ISO 8601 टाइमस्टैम्प के साथ हिस्ट्री दिखाएं |

### नेटवर्क

| एलियास | कमांड | विवरण |
|-------|---------|-------------|
| `ifconfig` | `sudo ifconfig` | नेटवर्क इंटरफेस कॉन्फ़िगर करें |
| `ipinfo` | `ipconfig getpacket en0` | en0 के लिए नेटवर्क पैरामीटर प्राप्त करें |
| `nls` | `sudo lsof -i -P \| grep LISTEN` | सक्रिय नेटवर्क लिसनर दिखाएं |
| `op` | `sudo lsof -i -P` | खुले पोर्ट्स सूची |
| `ping` | `ping -c 5` | 5 पैकेट सीमा के साथ पिंग |
| `ports` | `netstat -tulan` | सभी लिसनिंग पोर्ट्स सूची |

### सिस्टम मॉनिटरिंग

| एलियास | कमांड | विवरण |
|-------|---------|-------------|
| `top` | `sudo btop` | इंटरैक्टिव प्रोसेस मॉनिटर |
| `spd` | `sudo rm -rf /private/var/log/asl/*` | ASL लॉग फाइलें साफ करें |

### यूटिलिटीज

| एलियास | कमांड | विवरण |
|-------|---------|-------------|
| `ctf` | `ls -1 \| wc -l` | वर्तमान डायरेक्टरी में फाइलें गिनें |
| `qfind` | `find . -name` | त्वरित फाइल खोज |
| `reload` | `exec $SHELL -l` | शेल रीलोड करें |
| `wth` | `curl -s "wttr.in/?format=3"` | वर्तमान मौसम प्राप्त करें |
