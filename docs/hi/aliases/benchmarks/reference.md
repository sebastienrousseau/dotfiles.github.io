# Benchmarks संदर्भ परिशिष्ट

## स्रोत फ़ाइलें

- `.chezmoitemplates/aliases/benchmarks/benchmarks.aliases.sh`

## उपनाम संदर्भ

| उपनाम | कमांड |
| :--- | :--- |
| `bench` | `hyperfine` |
| `benchw` | `hyperfine --warmup 3` |

## सत्यापन

- उपनाम अपडेट के बाद `chezmoi apply` चलाएं।
- Shell स्थिति सत्यापित करने के लिए `dot doctor` चलाएं।
