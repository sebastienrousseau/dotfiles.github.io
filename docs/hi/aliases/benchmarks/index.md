# Benchmarks उपनाम

संक्षिप्त benchmarks कमांड।

यह पेज एक उपनाम समूह और उसके सामान्य कमांड शॉर्टकट प्रस्तुत करता है। नामकरण परंपराओं, अपेक्षित इनपुट और सुरक्षित निष्पादन पैटर्न को समझने के लिए पहले इस पेज को पढ़ें। फिर संपूर्ण कमांड सूची और सटीक सिंटैक्स के लिए संदर्भ परिशिष्ट खोलें। ये उपनाम दोहराव टाइपिंग को कम करते हैं और macOS, Linux और WSL पर सुसंगत वर्कफ़्लो व्यवहार बनाए रखते हैं।

## कब उपयोग करें

जब आपको पूर्वानुमानित कमांड पैटर्न की आवश्यकता हो, तो दैनिक Shell कार्य से पहले इस पेज का उपयोग करें।

## क्या शामिल है

- उपनाम: 2
- सहायक फ़ंक्शन: 0
- स्रोत फ़ाइलें: 1

## त्वरित उदाहरण

- `bench` → `hyperfine`
- `benchw` → `hyperfine --warmup 3`
