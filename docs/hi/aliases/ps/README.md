# PS एलियास

सरल शॉर्टकट के साथ चल रही प्रोसेस को मॉनिटर और प्रबंधित करें।

## अवलोकन

ये एलियास `ps.aliases.sh` में परिभाषित हैं और Chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

| एलियास | विवरण |
|:---|:---|
| `pid` | UID, PID, पैरेंट PID, CPU उपयोग, स्टार्ट टाइम, TTY, और कमांड प्रदर्शित करें |
| `ps` | सभी प्रोसेस प्रदर्शित करें |
| `psa` | सभी प्रोसेस सूचीबद्ध करें |
