# Interactive एलियास

विनाशकारी ऑपरेशन से पहले पुष्टि प्रॉम्प्ट के साथ सुरक्षित शेल कमांड।

## अवलोकन

Interactive एलियास सामान्य शेल कमांड में पुष्टि प्रॉम्प्ट जोड़ते हैं जो स्थायी रूप से फाइलें संशोधित या हटा सकते हैं। `interactive.aliases.sh` में परिभाषित और chezmoi द्वारा स्वचालित रूप से लोड होते हैं, ये एलियास संभावित विनाशकारी ऑपरेशन निष्पादित करने से पहले यूजर पुष्टि की आवश्यकता द्वारा आकस्मिक डेटा हानि रोकने में मदद करते हैं।

## संदर्भ

### फाइल ऑपरेशन

| एलियास | विवरण |
|-------|-------------|
| `cp` | वर्बोस आउटपुट के साथ फाइलें और डायरेक्टरी इंटरैक्टिवली कॉपी करें। मौजूदा फाइलें ओवरराइट करने से पहले प्रॉम्प्ट करता है। |
| `mv` | वर्बोस आउटपुट के साथ फाइलें इंटरैक्टिवली मूव या रीनेम करें। मौजूदा फाइलें ओवरराइट करने से पहले प्रॉम्प्ट करता है। |
| `ln` | वर्बोस आउटपुट के साथ symbolic links इंटरैक्टिवली बनाएं। मौजूदा लिंक ओवरराइट करने से पहले प्रॉम्प्ट करता है। |

### फाइल रिमूवल

| एलियास | विवरण |
|-------|-------------|
| `rm` | वर्बोस आउटपुट के साथ फाइलें या डायरेक्टरी इंटरैक्टिवली हटाएं। प्रत्येक रिमूवल से पहले प्रॉम्प्ट करता है। |
| `del` | वर्बोस आउटपुट के साथ फाइलें या डायरेक्टरी इंटरैक्टिवली और रिकर्सिवली हटाएं। |
| `zap` | `rm` के लिए एलियास। वर्बोस आउटपुट के साथ फाइलें या डायरेक्टरी इंटरैक्टिवली हटाता है। |

### ट्रैश मैनेजमेंट

| एलियास | विवरण |
|-------|-------------|
| `bin` | ट्रैश डायरेक्टरी (`~/.Trash`) में सभी फाइलें जबरन और रिकर्सिवली हटाएं। |

### अनुमतियां और यूटिलिटीज

| एलियास | विवरण |
|-------|-------------|
| `chmod` | वर्बोस आउटपुट के साथ फाइल या डायरेक्टरी अनुमतियां बदलें। |
| `chown` | वर्बोस आउटपुट के साथ फाइल या डायरेक्टरी ओनर और ग्रुप बदलें। |
| `diff` | दो फाइलों के बीच अंतर यूनिफाइड फॉर्मेट में तुलना करें और दिखाएं। |
| `grep` | फाइलों या आउटपुट में पैटर्न खोजें, लाइन नंबर दिखाते हुए और केस-इनसेंसिटिवली। |
| `mkdir` | आवश्यक पैरेंट डायरेक्टरी और वर्बोस आउटपुट के साथ नई डायरेक्टरी बनाएं। |
