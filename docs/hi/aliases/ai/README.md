# AI एलियास

AI और LLM टूल्स के लिए शॉर्टकट। अपने AI-असिस्टेड वर्कफ़्लो को तेज करें।

## अवलोकन

ये एलियास `ai.aliases.sh` में परिभाषित हैं और Chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

### GitHub Copilot

| एलियास | विवरण |
|:---|:---|
| `ghcp` | Copilot शॉर्टकट |
| `ghs` | कोड सुझाव |
| `ghe` | कोड समझाएं |

### Ollama

| एलियास | विवरण |
|:---|:---|
| `ol` | Ollama शॉर्टकट |
| `olr` | मॉडल चलाएं |
