# GNU Coreutils एलियास

तेज़ कमांड-लाइन वर्कफ़्लो के लिए आवश्यक Unix यूटिलिटीज के शॉर्टकट।

## अवलोकन

GNU Coreutils एलियास स्टैंडर्ड Unix यूटिलिटीज के लिए शॉर्टहैंड कमांड प्रदान करते हैं। `gnu.aliases.sh` में परिभाषित और chezmoi द्वारा स्वचालित रूप से लोड होते हैं, ये एलियास फाइल ऑपरेशन, टेक्स्ट प्रोसेसिंग, और डेटा वेरिफिकेशन कार्यों को सरल करते हैं।

## संदर्भ

### फाइल मैनेजमेंट

| एलियास | विवरण |
|-------|-------------|
| `basename` | फाइलनेम से डायरेक्टरी और सफिक्स स्ट्रिप करें |
| `cp` | फाइलें और डायरेक्टरी कॉपी करें |
| `dirname` | फाइलनेम से नॉन-डायरेक्टरी सफिक्स स्ट्रिप करें |
| `ln` | फाइलों के बीच लिंक बनाएं |
| `ls` | डायरेक्टरी सामग्री सूचीबद्ध करें |
| `mkdir` | डायरेक्टरी बनाएं |
| `mv` | फाइलें या डायरेक्टरी मूव या रीनेम करें |
| `pwd` | वर्किंग डायरेक्टरी नाम प्रिंट करें |
| `rm` | फाइलें या डायरेक्टरी हटाएं |
| `rmdir` | खाली डायरेक्टरी हटाएं |

### टेक्स्ट प्रोसेसिंग

| एलियास | विवरण |
|-------|-------------|
| `awk` | पैटर्न स्कैनिंग और प्रोसेसिंग लैंग्वेज |
| `cat` | फाइलें कॉन्कैटेनेट और डिस्प्ले करें |
| `cut` | फाइलों की प्रत्येक लाइन से सेक्शन हटाएं |
| `grep` | पैटर्न मैच करने वाली लाइनें प्रिंट करें |
| `head` | फाइलों का पहला भाग आउटपुट करें |
| `sed` | टेक्स्ट फ़िल्टर और ट्रांसफॉर्म करने के लिए स्ट्रीम एडिटर |
| `sort` | टेक्स्ट फाइलों की लाइनें सॉर्ट करें |
| `tail` | फाइलों का अंतिम भाग आउटपुट करें |
| `tr` | कैरेक्टर ट्रांसलेट या डिलीट करें |

### चेकसम यूटिलिटीज

| एलियास | विवरण |
|-------|-------------|
| `b2sum` | BLAKE2 मैसेज डाइजेस्ट प्रिंट या चेक करें |
| `cksum` | CRC चेकसम और बाइट काउंट प्रिंट करें |
| `sha1sum` | SHA1 मैसेज डाइजेस्ट प्रिंट या चेक करें |
| `sha256sum` | SHA256 मैसेज डाइजेस्ट प्रिंट या चेक करें |
| `sha512sum` | SHA512 मैसेज डाइजेस्ट प्रिंट या चेक करें |

### एनकोडिंग यूटिलिटीज

| एलियास | विवरण |
|-------|-------------|
| `base32` | base32 डेटा प्रिंट या कन्वर्ट करें |
| `base64` | base64 डेटा एनकोड या डिकोड करें |
| `basenc` | विभिन्न फॉर्मेट में डेटा एनकोड या डिकोड करें |
