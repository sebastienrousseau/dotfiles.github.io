# Permission एलियास

आपके शेल एनवायरनमेंट में फाइल और डायरेक्टरी अनुमतियां प्रबंधित करने के लिए शॉर्टकट।

## अवलोकन

ये एलियास `permission.aliases.sh` में परिभाषित हैं और chezmoi द्वारा स्वचालित रूप से लोड होते हैं। वे सामान्य अनुमति पैटर्न और ओनरशिप कमांड तक त्वरित पहुंच प्रदान करते हैं।

## संदर्भ

### न्यूमेरिक अनुमति शॉर्टकट

| एलियास | विवरण |
|-------|-------------|
| `000` | यूजर, ग्रुप, और अन्य के लिए कोई रीड, राइट, या एक्ज़ीक्यूट नहीं |
| `400` | केवल यूजर के लिए रीड-ओनली |
| `444` | सभी के लिए रीड-ओनली |
| `600` | केवल यूजर के लिए रीड और राइट |
| `644` | सभी के लिए रीड, लेकिन केवल यूजर के लिए राइट |
| `666` | सभी के लिए रीड और राइट |
| `755` | यूजर के लिए रीड, राइट, और एक्ज़ीक्यूट, लेकिन ग्रुप और अन्य के लिए केवल रीड और एक्ज़ीक्यूट |
| `764` | यूजर और ग्रुप के लिए रीड और राइट, लेकिन अन्य के लिए केवल रीड |
| `777` | सभी के लिए रीड, राइट, और एक्ज़ीक्यूट |

### फाइल मोड कमांड

| एलियास | विवरण |
|-------|-------------|
| `chmod` | फाइल मोड बिट्स बदलें |
| `chmodr` | फाइल मोड बिट्स रिकर्सिवली बदलें |
| `chmodu` | फाइल मोड बिट्स रिकर्सिवली वर्तमान यूजर पर बदलें |
| `chmox` | फाइल को एक्ज़ीक्यूटेबल बनाएं |

### ओनरशिप कमांड

| एलियास | विवरण |
|-------|-------------|
| `chown` | फाइल ओनर और ग्रुप बदलें |
| `chownr` | फाइल ओनर और ग्रुप रिकर्सिवली बदलें |
| `chownu` | फाइल ओनर और ग्रुप रिकर्सिवली वर्तमान यूजर पर बदलें |
| `chgrp` | फाइलों या डायरेक्टरी का ग्रुप ओनरशिप बदलें |
| `chgrpr` | ग्रुप ओनरशिप रिकर्सिवली बदलें |
| `chgrpu` | ग्रुप ओनरशिप रिकर्सिवली वर्तमान यूजर पर बदलें |
