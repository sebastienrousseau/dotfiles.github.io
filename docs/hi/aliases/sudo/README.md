# Sudo एलियास

एलिवेटेड प्रिविलेज के साथ कमांड निष्पादित करें।

## अवलोकन

ये एलियास `sudo.aliases.sh` में परिभाषित हैं और Chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

| एलियास | विवरण |
|:---|:---|
| `root` | सुपरयूजर के रूप में कमांड निष्पादित करें |
| `s` | सुपरयूजर के रूप में कमांड निष्पादित करें |
| `su` | सुपरयूजर के रूप में कमांड निष्पादित करें |
| `sudo` | सुपरयूजर के रूप में कमांड निष्पादित करें |
