# Mkdir एलियास

इंटेलिजेंट नेमिंग कन्वेंशन के साथ डायरेक्टरी क्रिएशन को सुव्यवस्थित करें।

## अवलोकन

mkdir एलियास स्वचालित तिथि और समय स्टैम्प के साथ डायरेक्टरी बनाने के लिए शॉर्टकट प्रदान करते हैं। ये एलियास `mkdir.aliases.sh` में परिभाषित हैं और chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

| एलियास | विवरण |
|-------|-------------|
| `md` | डायरेक्टरी बनाएं |
| `mdd` | तिथि के साथ डायरेक्टरी बनाएं |
| `mdt` | समय के साथ डायरेक्टरी बनाएं |
| `mde` | वर्तमान तिथि के साथ example डायरेक्टरी बनाएं |
| `mdn` | वर्तमान तिथि के साथ notes डायरेक्टरी बनाएं |
| `mdw` | वर्तमान तिथि के साथ work डायरेक्टरी बनाएं |
