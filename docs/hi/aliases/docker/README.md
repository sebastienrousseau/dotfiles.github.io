# Docker एलियास

Docker, Compose, और Swarm के लिए सहज शॉर्टकट के साथ अपने कंटेनर वर्कफ़्लो को सुव्यवस्थित करें।

## अवलोकन

ये एलियास `docker.aliases.sh` में परिभाषित हैं और chezmoi द्वारा स्वचालित रूप से लोड होते हैं। वे Docker डेवलपमेंट के लिए शॉर्टकट का व्यापक सेट प्रदान करते हैं, जिसमें कंटेनर, इमेज, वॉल्यूम, नेटवर्क, Docker Compose, और Docker Swarm ऑपरेशन शामिल हैं।

## संदर्भ

### बेसिक कमांड

| एलियास | विवरण |
|-------|-------------|
| `dk` | Docker शॉर्टकट |
| `dkv` | Docker वर्जन दिखाएं |
| `dki` | सिस्टम-वाइड जानकारी प्रदर्शित करें |
| `dkl` | Docker रजिस्ट्री में लॉगइन |
| `dklo` | Docker रजिस्ट्री से लॉगआउट |

### कंटेनर ऑपरेशन

| एलियास | विवरण |
|-------|-------------|
| `dkps` | चल रहे कंटेनर सूचीबद्ध करें |
| `dkpsa` | सभी कंटेनर सूचीबद्ध करें |
| `dkr` | नए कंटेनर में कमांड चलाएं |
| `dkri` | इंटरैक्टिव कंटेनर चलाएं |
| `dkrd` | बैकग्राउंड में कंटेनर चलाएं |
| `dks` | कंटेनर स्टार्ट करें |
| `dkst` | कंटेनर स्टॉप करें |
| `dkrs` | कंटेनर रीस्टार्ट करें |
| `dkrm` | कंटेनर हटाएं |
| `dkrma` | सभी कंटेनर हटाएं |

### Docker Compose

| एलियास | विवरण |
|-------|-------------|
| `dc` | Docker Compose शॉर्टकट |
| `dcu` | कंटेनर बनाएं और स्टार्ट करें |
| `dcud` | बैकग्राउंड में कंटेनर बनाएं और स्टार्ट करें |
| `dcd` | कंटेनर स्टॉप करें और हटाएं |
| `dcr` | सर्विसेज रीस्टार्ट करें |
| `dcs` | सर्विसेज स्टॉप करें |
| `dcps` | कंटेनर सूचीबद्ध करें |
| `dcl` | लॉग देखें |
| `dclf` | लॉग फॉलो करें |
| `dcb` | सर्विसेज बिल्ड करें |

### इमेज

| एलियास | विवरण |
|-------|-------------|
| `dki` | इमेज सूचीबद्ध करें |
| `dkb` | इमेज बिल्ड करें |
| `dkbt` | इमेज बिल्ड करें और टैग करें |
| `dkpu` | रजिस्ट्री में इमेज पुश करें |
| `dkrmi` | इमेज हटाएं |
| `dkprune` | अप्रयुक्त डेटा हटाएं |

### वॉल्यूम

| एलियास | विवरण |
|-------|-------------|
| `dkv` | वॉल्यूम शॉर्टकट |
| `dkvls` | वॉल्यूम सूचीबद्ध करें |
| `dkvc` | वॉल्यूम बनाएं |
| `dkvrm` | वॉल्यूम हटाएं |

### नेटवर्क

| `dknrm` | नेटवर्क हटाएं |
