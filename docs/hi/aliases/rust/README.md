# Rust एलियास

Cargo और Rustup का उपयोग करके Rust डेवलपमेंट के लिए शॉर्टकट।

## अवलोकन

ये एलियास `rust.aliases.sh` में परिभाषित हैं और Chezmoi द्वारा स्वचालित रूप से लोड होते हैं।

## संदर्भ

### बेसिक कमांड

| एलियास | विवरण |
|:---|:---|
| `cg` | Cargo शॉर्टकट |
| `cgn` | नया बाइनरी प्रोजेक्ट बनाएं |
| `cgni` | नया लाइब्रेरी प्रोजेक्ट बनाएं |
| `cginit` | वर्तमान डायरेक्टरी में प्रोजेक्ट इनिशियलाइज़ करें |

### बिल्ड और रन

| एलियास | विवरण |
|:---|:---|
| `cgb` | डीबग बिल्ड |
| `cgbr` | रिलीज़ बिल्ड |
| `cgr` | डीबग चलाएं |
| `cgrr` | रिलीज़ चलाएं |
| `cgw` | देखें और रीबिल्ड करें |

### टेस्टिंग और बेंचमार्किंग

| एलियास | विवरण |
|:---|:---|
| `cgt` | टेस्ट चलाएं |
| `cgtr` | रिलीज़ मोड में टेस्ट चलाएं |
| `cgbh` | बेंचमार्क चलाएं |

### कोड क्वालिटी

| एलियास | विवरण |
|:---|:---|
| `cgc` | कम्पाइलेशन जांचें |
| `cgcl` | बिल्ड आर्टिफैक्ट्स साफ करें |
| `cgcy` | clippy लिंट चलाएं |
| `cgf` | कोड फॉर्मेट करें |
| `cgfx` | कोड समस्याएं ऑटो-फिक्स करें |
| `cgaud` | सुरक्षा कमज़ोरियां जांचें |

### डिपेंडेंसी

| एलियास | विवरण |
|:---|:---|
| `cga` | डिपेंडेंसी जोड़ें |
| `cgad` | dev डिपेंडेंसी जोड़ें |
| `cgu` | डिपेंडेंसी अपडेट करें |
| `cgo` | पुरानी डिपेंडेंसी जांचें |
| `cgtree` | डिपेंडेंसी ट्री प्रदर्शित करें |

### पैकेज मैनेजमेंट

| एलियास | विवरण |
|:---|:---|
| `cgi` | बाइनरी इंस्टॉल करें |
| `cgun` | बाइनरी अनइंस्टॉल करें |
| `cgp` | crates.io पर पब्लिश करें |
| `cgs` | crates.io खोजें |

### Rustup अपडेट

| एलियास | विवरण |
|:---|:---|
| `ru` | सभी टूलचेन अपडेट करें |
| `rus` | stable टूलचेन अपडेट करें |
| `run` | nightly टूलचेन अपडेट करें |
| `rti` | विशिष्ट टूलचेन इंस्टॉल करें |

### टूलचेन मैनेजमेंट

| एलियास | विवरण |
|:---|:---|
| `rtl` | इंस्टॉल टूलचेन सूचीबद्ध करें |
| `rtu` | टूलचेन अनइंस्टॉल करें |
| `rde` | डिफ़ॉल्ट टूलचेन सेट करें |

### डॉक्यूमेंटेशन

| एलियास | विवरण |
|:---|:---|
| `rdo` | Rust डॉक्यूमेंटेशन खोलें |
