# اختصارات Installer

أدوات للتمهيد والنشر البعيد.

## نظرة عامة

توفر اختصارات installer أوامر مبسطة لإعداد بيئة dotfiles الخاصة بك محليًا ونشر إعداداتك إلى خوادم بعيدة.

## المرجع

| الاختصار | الوصف |
|-------|-------------|
| `dot-install` | تشغيل سكريبت التمهيد `install.sh` محليًا. |
| `dot-teleport` | نقل إعداداتك إلى خادم بعيد (`dot-teleport user@host`). |
