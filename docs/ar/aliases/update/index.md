# اختصارات Update

حافظ على تحديث نظامك وحزمك.

## نظرة عامة

هذه الاختصارات معرّفة في `update.aliases.sh` ويتم تحميلها تلقائيًا بواسطة Chezmoi.

## المرجع

### macOS

| الاختصار | الوصف |
|:---|:---|
| `upd` | تحديث برامج macOS وHomebrew وCasks وApp Store وRust وRuby وPython وNode.js |

### Linux

| الاختصار | الوصف |
|:---|:---|
| `open` | فتح ملف أو رابط في التطبيق المفضل |
| `pbcopy` | نسخ إلى الحافظة |
| `pbpaste` | لصق من الحافظة |
| `upd` | تحديث حزم Linux وpnpm وRust وRuby وPython وNode.js |
