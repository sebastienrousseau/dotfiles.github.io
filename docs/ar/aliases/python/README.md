# اختصارات Python

مختصرات لتطوير Python. المفسر والحزم والبيئات الافتراضية.

## نظرة عامة

هذه الاختصارات معرّفة في `python.aliases.sh` ويتم تحميلها تلقائيًا بواسطة Chezmoi.

## المرجع

| الاختصار | الأمر | الوصف |
|:---|:---|:---|
| `py` | `python` | مفسر Python |
| `pip` | `pip3` | مثبت الحزم |
| `v` | `virtualenv` | بيئة افتراضية |
| `uv` | `uv` | مدير حزم سريع |
| `uvpi` | `uv pip install` | التثبيت مع uv |
