# اختصارات PS

راقب وأدر العمليات الجارية بمختصرات بسيطة.

## نظرة عامة

هذه الاختصارات معرّفة في `ps.aliases.sh` ويتم تحميلها تلقائيًا بواسطة Chezmoi.

## المرجع

| الاختصار | الوصف |
|:---|:---|
| `pid` | عرض UID وPID ومعرف العملية الأب واستخدام CPU ووقت البدء وTTY والأمر |
| `ps` | عرض جميع العمليات |
| `psa` | سرد جميع العمليات |
