# اختصارات Subversion

مختصرات لعمليات التحكم في الإصدارات Subversion.

## نظرة عامة

هذه الاختصارات معرّفة في `subversion.aliases.sh` ويتم تحميلها تلقائيًا بواسطة Chezmoi.

## المرجع

| الاختصار | الوصف |
|:---|:---|
| `sad` | إضافة ملفات ومجلدات للتحكم في الإصدارات |
| `sau` | إدارة بيانات اعتماد المصادقة المخزنة |
| `sbl` | عرض متى تم تغيير كل سطر آخر مرة |
| `sci` | التزام التغييرات في المستودع |
| `scl` | تنظيف نسخة العمل |
| `sco` | استخراج نسخة عمل |
| `scp` | نسخ الملفات والمجلدات |
| `sdi` | عرض التغييرات المحلية أو الفروقات |
| `sdl` | إزالة الملفات من التحكم في الإصدارات |
| `shp` | عرض المساعدة للأوامر |
| `sin` | عرض معلومات عن عنصر |
| `slg` | عرض رسائل السجل |
| `slk` | قفل مسارات نسخة العمل |
| `sls` | سرد إدخالات المجلد |
| `smd` | إنشاء مجلد جديد تحت التحكم في الإصدارات |
| `smg` | دمج التغييرات في نسخة العمل |
| `smv` | نقل أو إعادة تسمية عنصر |
| `sst` | طباعة حالة نسخة العمل |
| `sup` | تحديث نسخة العمل من المستودع |
