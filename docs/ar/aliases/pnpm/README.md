# اختصارات Pnpm

مختصرات لمدير الحزم السريع والفعال في استخدام القرص.

## نظرة عامة

هذه الاختصارات معرّفة في `pnpm.aliases.sh` ويتم تحميلها تلقائيًا بواسطة chezmoi. توفر وصولًا سريعًا لأوامر [pnpm](https://pnpm.io/) الشائعة لإدارة حزم Node.js.

## المرجع

### تهيئة المشروع

| الاختصار | الوصف |
|-------|-------------|
| `pni` | تهيئة مشروع جديد |
| `pnc` | إنشاء مشروع جديد |
| `pnin` | تثبيت تبعيات المشروع |
| `pnun` | إلغاء تثبيت تبعيات المشروع |

### التطوير

| الاختصار | الوصف |
|-------|-------------|
| `pnb` | بناء المشروع |
| `pnrun` | تشغيل سكريبت من المشروع |
| `pnd` | تشغيل المشروع في وضع التطوير |
| `pns` | تشغيل المشروع في وضع الخدمة |
| `pnst` | بدء المشروع |

### الاختبار والجودة

| الاختصار | الوصف |
|-------|-------------|
| `pnt` | اختبار المشروع |
| `pntc` | اختبار المشروع مع التغطية |
| `pnln` | فحص المشروع |
| `pndoc` | إنشاء توثيق المشروع |

### إدارة التبعيات

| الاختصار | الوصف |
|-------|-------------|
| `pnls` | سرد جميع التبعيات |
| `pnout` | فحص التبعيات القديمة |
| `pnwhy` | فحص سبب تثبيت تبعية |
| `pnad` | إضافة تبعية تطوير |
| `pnrm` | إزالة تبعية |
| `pnup` | تحديث تبعية |

### الحزم العامة

| الاختصار | الوصف |
|-------|-------------|
| `pnga` | إضافة تبعية عامة |
| `pngls` | سرد جميع التبعيات العامة |
| `pngrm` | إزالة تبعية عامة |
| `pngu` | تحديث تبعية عامة |

### النشر والأدوات

| الاختصار | الوصف |
|-------|-------------|
| `pnpub` | نشر المشروع |
| `pnx` | اختصار لـ pnpx |
| `pnh` | عرض المساعدة |
