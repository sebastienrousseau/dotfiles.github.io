# اختصارات Wget

تنزيل الملفات من الويب مع دعم الاستئناف.

## نظرة عامة

هذه الاختصارات معرّفة في `wget.aliases.sh` ويتم تحميلها تلقائيًا بواسطة Chezmoi.

## المرجع

| الاختصار | الوصف |
|:---|:---|
| `wg` | اختصار Wget |
| `wgc` | Wget مع الاستمرار |
| `wge` | Wget مع robots=off |
| `wget` | Wget مع الاستمرار |
