# اختصارات Go

بسّط سير عمل تطوير Go بمختصرات شل بديهية.

## نظرة عامة

هذه الاختصارات معرّفة في `go.aliases.sh` ويتم تحميلها تلقائيًا بواسطة chezmoi. توفر وصولًا سريعًا لأوامر Go الشائعة للتشغيل والبناء والاختبار وإدارة الوحدات.

## المرجع

| الاختصار | الأمر | الوصف |
|----------|---------------|------------------------------------|
| `go` | `go` | اختصار Go |
| `gor` | `go run` | تشغيل برنامج Go |
| `gob` | `go build` | بناء حزمة Go |
| `got` | `go test` | تشغيل اختبارات Go |
| `gomt` | `go mod tidy` | ترتيب تبعيات وحدة Go |
| `goget` | `go get` | تنزيل وتثبيت حزم Go |
