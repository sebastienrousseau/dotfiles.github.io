# اختصارات Interactive

أوامر شل آمنة مع مطالبات تأكيد قبل العمليات المدمرة.

## نظرة عامة

تضيف اختصارات Interactive مطالبات تأكيد لأوامر الشل الشائعة التي يمكنها تعديل أو حذف الملفات بشكل دائم. معرّفة في `interactive.aliases.sh` ويتم تحميلها تلقائيًا بواسطة chezmoi، تساعد هذه الاختصارات في منع فقدان البيانات عن طريق الخطأ من خلال طلب تأكيد المستخدم قبل تنفيذ العمليات الخطيرة المحتملة.

## المرجع

### عمليات الملفات

| الاختصار | الوصف |
|-------|-------------|
| `cp` | نسخ الملفات والمجلدات تفاعليًا مع مخرجات مفصلة. يطالب قبل الكتابة فوق الملفات الموجودة. |
| `mv` | نقل أو إعادة تسمية الملفات تفاعليًا مع مخرجات مفصلة. يطالب قبل الكتابة فوق الملفات الموجودة. |
| `ln` | إنشاء روابط رمزية تفاعليًا مع مخرجات مفصلة. يطالب قبل الكتابة فوق الروابط الموجودة. |

### إزالة الملفات

| الاختصار | الوصف |
|-------|-------------|
| `rm` | إزالة الملفات أو المجلدات تفاعليًا مع مخرجات مفصلة. يطالب قبل كل إزالة. |
| `del` | إزالة الملفات أو المجلدات تفاعليًا وتكراريًا مع مخرجات مفصلة. يطالب قبل كل إزالة. |
| `zap` | اختصار لـ `rm`. يزيل الملفات أو المجلدات تفاعليًا مع مخرجات مفصلة. |

### إدارة سلة المهملات

| الاختصار | الوصف |
|-------|-------------|
| `bin` | إزالة جميع الملفات في مجلد سلة المهملات (`~/.Trash`) بقوة وتكراريًا. |

### الصلاحيات والأدوات

| الاختصار | الوصف |
|-------|-------------|
| `chmod` | تغيير صلاحيات الملف أو المجلد مع مخرجات مفصلة. |
| `chown` | تغيير مالك ومجموعة الملف أو المجلد مع مخرجات مفصلة. |
| `diff` | مقارنة وعرض الفروقات بين ملفين بتنسيق موحد. |
| `grep` | البحث عن نمط في الملفات أو المخرجات، مع عرض أرقام الأسطر وبدون حساسية لحالة الأحرف. |
| `mkdir` | إنشاء مجلد جديد مع المجلدات الأب حسب الحاجة ومخرجات مفصلة. |
