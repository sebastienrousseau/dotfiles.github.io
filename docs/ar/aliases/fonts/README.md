# اختصارات Font

اختصارات شل لإدارة خطوط النظام وذاكرة التخزين المؤقت.

## نظرة عامة

توفر اختصارات Font مختصرات مريحة لمهام إدارة الخطوط الشائعة. تبسط هذه الأوامر تحديث ذاكرة تخزين الخطوط بعد تثبيت خطوط جديدة وسرد جميع عائلات الخطوط المتاحة على نظامك.

## المرجع

### الأوامر

| الاختصار | الأمر | الوصف |
|-------|---------|-------------|
| `update-fonts` | `fc-cache -fv` | تحديث ذاكرة تخزين خطوط النظام |
| `list-fonts` | — | سرد جميع عائلات الخطوط المثبتة |

### الخطوط المثبتة

الخطوط التالية مُعدّة افتراضيًا:

| الخط | الغرض |
|------|---------|
| JetBrainsMono Nerd Font | الخط الأساسي للطرفية |
| Symbols Nerd Font | احتياطي للأيقونات |
