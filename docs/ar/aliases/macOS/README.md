# اختصارات macOS

مختصرات شل لإدارة نظام macOS والصيانة.

## نظرة عامة

هذه الاختصارات معرّفة في `macOS.aliases.sh` ويتم تحميلها تلقائيًا بواسطة chezmoi. توفر وصولًا سريعًا لعمليات macOS الشائعة بما في ذلك التحكم في Finder وصيانة النظام وأدوات المطورين.

## المرجع

### إدارة النظام

| الاختصار | الوصف |
|-------|-------------|
| `lockScreen` | قفل الشاشة |
| `wifiOn` | تشغيل Wi-Fi |
| `wifiOff` | إيقاف Wi-Fi |
| `vp` | التحقق من صلاحيات macOS |
| `vv` | التحقق من وحدة تخزين macOS |

### Finder

| الاختصار | الوصف |
|-------|-------------|
| `ofd` | فتح المجلد الحالي في Finder |
| `finderShowHidden` | إظهار الملفات المخفية في Finder |
| `finderHideHidden` | إخفاء الملفات المخفية في Finder |
| `showDesktopIcons` | إظهار جميع الأيقونات على سطح المكتب |
| `hideDesktopIcons` | إخفاء جميع الأيقونات على سطح المكتب |

### التنظيف

| الاختصار | الوصف |
|-------|-------------|
| `clds` | حذف ملفات .DS_Store تكراريًا |
| `clls` | تنظيف LaunchServices لإزالة التكرارات في قائمة "فتح باستخدام" |
| `trash` | إفراغ سلة المهملات على جميع وحدات التخزين المركبة والقرص الرئيسي |

### أدوات المطورين

| الاختصار | الوصف |
|-------|-------------|
| `xcode` | تشغيل Xcode |
| `purge` | تطهير Xcode DerivedData |
| `iphone` | فتح محاكيات الأجهزة |
| `safariSafeMode` | تشغيل Safari في الوضع الآمن |

### العرض

| الاختصار | الوصف |
|-------|-------------|
| `screensaverDesktop` | تشغيل شاشة التوقف على سطح المكتب |
