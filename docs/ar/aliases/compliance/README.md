# اختصارات Compliance

أدوات شل للامتثال التنظيمي وتطبيق الخصوصية.

## نظرة عامة

توفر اختصارات Compliance ضوابط أساسية لإدارة الخصوصية والحفاظ على مسارات التدقيق في بيئة الشل الخاصة بك. تساعد هذه الاختصارات في تعطيل التتبع عبر أدوات CLI الشائعة وتتبع تغييرات الإعدادات لمتطلبات الامتثال.

## المرجع

### الضوابط

| الاختصار | الوصف |
|-------|-------------|
| `privacy-mode` | تعطيل التتبع لأدوات CLI (Homebrew وDotnet وAzure وغيرها) للجلسة الحالية. |
| `dot-audit` | عرض مسار تدقيق الإعدادات الأخير (سجل Git أو سجل التدقيق). |
