# اختصارات AI

مختصرات لأدوات الذكاء الاصطناعي ونماذج اللغة الكبيرة. سرّع سير عملك المدعوم بالذكاء الاصطناعي.

## نظرة عامة

هذه الاختصارات معرّفة في `ai.aliases.sh` ويتم تحميلها تلقائيًا بواسطة Chezmoi.

## المرجع

### GitHub Copilot

| الاختصار | الوصف |
|:---|:---|
| `ghcp` | اختصار Copilot |
| `ghs` | اقتراح الكود |
| `ghe` | شرح الكود |

### Ollama

| الاختصار | الوصف |
|:---|:---|
| `ol` | اختصار Ollama |
| `olr` | تشغيل النموذج |
