# اختصارات Modern Tooling

بدائل أسرع وأذكى لأوامر Unix القديمة.

## نظرة عامة

اختصارات modern المعرّفة في `modern.aliases.sh` توفر بدائل مبنية بـ Rust لأدوات Unix التقليدية. يتم تحميل هذه الاختصارات تلقائيًا بواسطة `chezmoi` وتعود بشكل سلس للأوامر القياسية عندما لا تكون الأدوات الحديثة متاحة.

## المرجع

### سرد الملفات مع eza

يستبدل `ls` عندما يكون `eza` مثبتًا.

| الاختصار | الأمر | الوصف |
|-------|---------|-------------|
| `ls` | `eza --icons` | سرد الملفات مع أيقونات |
| `ll` | `eza -alF` | تنسيق قائمة طويلة |
| `la` | `eza -a` | سرد جميع الملفات بما في ذلك المخفية |
| `lt` | `eza --tree` | عرض شجرة المجلد |

يعود إلى `ls` القياسي إذا لم يكن `eza` مثبتًا.

### محتوى الملف مع bat

يستبدل `cat` عندما يكون `bat` مثبتًا.

| الاختصار | الأمر | الوصف |
|-------|---------|-------------|
| `cat` | `bat` | عرض محتوى الملف مع تمييز الصياغة |

### البحث مع ripgrep

يستبدل `grep` عندما يكون `rg` مثبتًا.

| الاختصار | الأمر | الوصف |
|-------|---------|-------------|
| `grep` | `rg` | البحث في محتويات الملفات باستخدام Ripgrep |
