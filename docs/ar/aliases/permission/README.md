# اختصارات Permission

مختصرات لإدارة صلاحيات الملفات والمجلدات في بيئة الشل الخاصة بك.

## نظرة عامة

هذه الاختصارات معرّفة في `permission.aliases.sh` ويتم تحميلها تلقائيًا بواسطة chezmoi. توفر وصولًا سريعًا لأنماط الصلاحيات الشائعة وأوامر الملكية.

## المرجع

### مختصرات الصلاحيات الرقمية

| الاختصار | الوصف |
|-------|-------------|
| `000` | تعيين الصلاحيات بدون قراءة أو كتابة أو تنفيذ للمستخدم والمجموعة والآخرين |
| `400` | تعيين الصلاحيات بدون قراءة أو كتابة، لكن السماح بالتنفيذ للمستخدم فقط |
| `444` | تعيين الصلاحيات بدون كتابة أو تنفيذ، لكن السماح بالقراءة للجميع |
| `600` | تعيين الصلاحيات للقراءة والكتابة للمستخدم فقط |
| `644` | تعيين الصلاحيات للقراءة للجميع، لكن الكتابة للمستخدم فقط |
| `666` | تعيين الصلاحيات للقراءة والكتابة للجميع |
| `755` | تعيين الصلاحيات للقراءة والكتابة والتنفيذ للمستخدم، لكن القراءة والتنفيذ فقط للمجموعة والآخرين |
| `764` | تعيين الصلاحيات للقراءة والكتابة للمستخدم والمجموعة، لكن القراءة فقط للآخرين |
| `777` | تعيين الصلاحيات للقراءة والكتابة والتنفيذ للجميع |

### أوامر وضع الملف

| الاختصار | الوصف |
|-------|-------------|
| `chmod` | تغيير بتات وضع الملف |
| `chmodr` | تغيير بتات وضع الملف تكراريًا |
| `chmodu` | تغيير بتات وضع الملف تكراريًا للمستخدم الحالي |
| `chmox` | جعل الملف قابلًا للتنفيذ |

### أوامر الملكية

| الاختصار | الوصف |
|-------|-------------|
| `chown` | تغيير مالك ومجموعة الملف |
| `chownr` | تغيير مالك ومجموعة الملف تكراريًا |
| `chownu` | تغيير مالك ومجموعة الملف تكراريًا للمستخدم الحالي |
| `chgrp` | تغيير ملكية المجموعة للملفات أو المجلدات |
| `chgrpr` | تغيير ملكية المجموعة للملفات أو المجلدات تكراريًا |
| `chgrpu` | تغيير ملكية المجموعة للملفات أو المجلدات تكراريًا للمستخدم الحالي |
