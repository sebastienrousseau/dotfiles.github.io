# اختصارات Mkdir

بسّط إنشاء المجلدات باتفاقيات تسمية ذكية.

## نظرة عامة

توفر اختصارات mkdir مختصرات لإنشاء مجلدات بطوابع تاريخ ووقت تلقائية. هذه الاختصارات معرّفة في `mkdir.aliases.sh` ويتم تحميلها تلقائيًا بواسطة chezmoi.

## المرجع

| الاختصار | الوصف |
|-------|-------------|
| `md` | إنشاء مجلد |
| `mdd` | إنشاء مجلد بالتاريخ |
| `mdt` | إنشاء مجلد بالوقت |
| `mde` | إنشاء مجلد example بالتاريخ الحالي |
| `mdn` | إنشاء مجلد notes بالتاريخ الحالي |
| `mdw` | إنشاء مجلد work بالتاريخ الحالي |
