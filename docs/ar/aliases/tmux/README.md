# اختصارات Tmux

إدارة جلسات الطرفية والنوافذ والأجزاء مع tmux.

## نظرة عامة

هذه الاختصارات معرّفة في `tmux.aliases.sh` ويتم تحميلها تلقائيًا بواسطة Chezmoi.

## المرجع

| الاختصار | الوصف |
|:---|:---|
| `tm` | بدء tmux |
| `tma` | الربط بآخر جلسة |
| `tmat` | الربط بجلسة محددة |
| `tmks` | إنهاء جميع الجلسات باستثناء الحالية |
| `tmka` | إنهاء جميع الجلسات |
| `tml` | سرد جميع الجلسات |
| `tmn` | جلسة جديدة بدون اسم |
| `tms` | جلسة جديدة باسم |
| `tmr` | إعادة تحميل إعدادات tmux |
| `tmls` | سرد النوافذ |
| `tmlp` | سرد الأجزاء |
| `tmi` | عرض معلومات tmux |
