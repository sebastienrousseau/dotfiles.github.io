# اختصارات Docker

بسّط سير عمل الحاويات بمختصرات بديهية لـ Docker وCompose وSwarm.

## نظرة عامة

هذه الاختصارات معرّفة في `docker.aliases.sh` ويتم تحميلها تلقائيًا بواسطة chezmoi. توفر مجموعة شاملة من المختصرات لتطوير Docker، تغطي الحاويات والصور والأحجام والشبكات وعمليات Docker Compose وDocker Swarm.

## المرجع

### الأوامر الأساسية

| الاختصار | الوصف |
|-------|-------------|
| `dk` | اختصار Docker |
| `dkv` | عرض إصدار Docker |
| `dki` | عرض معلومات على مستوى النظام |
| `dkl` | تسجيل الدخول إلى سجل Docker |
| `dklo` | تسجيل الخروج من سجل Docker |
