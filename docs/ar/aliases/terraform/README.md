# اختصارات Terraform

مختصرات للبنية التحتية كرمز مع Terraform وAnsible.

## نظرة عامة

هذه الاختصارات معرّفة في `terraform.aliases.sh` ويتم تحميلها تلقائيًا بواسطة Chezmoi.

## المرجع

### Terraform

| الاختصار | الوصف |
|:---|:---|
| `tf` | اختصار Terraform |
| `tfi` | تهيئة الإعدادات |
| `tfp` | تخطيط التغييرات |
| `tfa` | تطبيق التغييرات |
| `tfaa` | التطبيق مع الموافقة التلقائية |
| `tfd` | تدمير البنية التحتية |
| `tff` | تنسيق الإعدادات |
| `tfv` | التحقق من الإعدادات |

### Ansible

| الاختصار | الوصف |
|:---|:---|
| `ans` | اختصار Ansible |
| `ansp` | تشغيل Ansible Playbook |
| `anslint` | فحص ملفات Ansible |
