# اختصارات Clear

بسّط سير عملك في الطرفية مع مختصرات سريعة لمسح الشاشة.

## نظرة عامة

توفر اختصارات clear طرقًا فعالة لإعادة تعيين عرض طرفيتك مع عرض اختياري لسياق مفيد مثل محتويات المجلد أو سجل الأوامر أو أشجار الملفات. هذه الاختصارات معرّفة في `clear.aliases.sh` ويتم تحميلها تلقائيًا بواسطة chezmoi.

## المرجع

| الاختصار | الوصف |
|-------|-------------|
| `c` | مسح شاشة الطرفية |
| `cdw` | التغيير إلى مجلد العمل |
| `cl` | مسح شاشة الطرفية (اختصار لـ `c`) |
| `clc` | مسح الشاشة وسرد محتويات المجلد |
| `clh` | مسح الشاشة وعرض سجل الأوامر |
| `clp` | مسح الشاشة وطباعة مجلد العمل |
| `clt` | مسح الشاشة وعرض شجرة المجلد |
| `cpl` | مسح الشاشة وعرض مجلد العمل مع المحتويات |
| `cplt` | مسح الشاشة وعرض مجلد العمل مع الشجرة |
