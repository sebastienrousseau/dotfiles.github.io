# اختصارات Yarn

مختصرات لمدير حزم JavaScript Yarn.

## نظرة عامة

هذه الاختصارات معرّفة في `yarn.aliases.sh` ويتم تحميلها تلقائيًا بواسطة Chezmoi.

## المرجع

| الاختصار | الوصف |
|:---|:---|
| `y` | اختصار Yarn |
| `yi` | تثبيت التبعيات |
| `ya` | إضافة تبعية |
| `yad` | إضافة تبعية تطوير |
| `yrun` | تشغيل سكريبت |
| `ystem` | بدء المشروع |
