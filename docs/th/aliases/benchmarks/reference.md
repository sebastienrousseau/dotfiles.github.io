# ภาคผนวกอ้างอิง Benchmarks

## ไฟล์ซอร์ส

- `.chezmoitemplates/aliases/benchmarks/benchmarks.aliases.sh`

## อ้างอิงนามแฝง

| นามแฝง | คำสั่ง |
| :--- | :--- |
| `bench` | `hyperfine` |
| `benchw` | `hyperfine --warmup 3` |

## การตรวจสอบ

- รัน `chezmoi apply` หลังอัปเดตนามแฝง
- รัน `dot doctor` เพื่อตรวจสอบสถานะ Shell
