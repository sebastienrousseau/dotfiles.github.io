# Aliases de Compliance

Ferramentas de shell para conformidade regulatoria e aplicacao de privacidade.

## Visao Geral

Os aliases de compliance fornecem controles essenciais para gerenciar privacidade e manter trilhas de auditoria no seu ambiente de shell. Estes aliases ajudam a desativar telemetria em ferramentas CLI comuns e rastrear alteracoes de configuracao para requisitos de conformidade.

## Referencia

### Controles

| Alias | Descricao |
|-------|-------------|
| `privacy-mode` | Desativa telemetria para ferramentas CLI (Homebrew, Dotnet, Azure, etc.) para a sessao atual. |
| `dot-audit` | Exibe a trilha de auditoria de configuracao recente (historico Git ou log de auditoria). |
