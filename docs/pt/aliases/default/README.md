# Alias Default

Gerenciar alias Default. Parte da configuracao **Universal Dotfiles**.

![Dotfiles banner][banner]

Esses alias sao definidos em `default.aliases.sh` e sao carregados automaticamente pelo `chezmoi`.

## âš¡ Alias

Alias para tarefas comuns e monitoramento:
- `c`: atalho para `clear`.
- `p`: atalho para `pwd`.
- `h`: lista comandos recentes.
- `path`: mostra `$PATH` em novas linhas.
- `quit`: atalho para `exit`.
- `halt`, `poweroff`, `reboot`: desligar/reiniciar.
- `srv`: iniciar servidor HTTP simples.
- `top`: monitoramento interativo.
- `qfind`: busca rapida por arquivos.
- `clc`: limpar tela e listar conteudo.
