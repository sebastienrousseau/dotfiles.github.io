# Aliases de UUID

Gere identificadores unicos universais (UUIDs).

## Visao Geral

Estes aliases sao definidos em `uuid.aliases.sh` e carregados automaticamente pelo Chezmoi.

## Referencia

| Alias | Descricao |
|-------|-------------|
| `uuid` | Gerar um UUID e copiar para a area de transferencia |
