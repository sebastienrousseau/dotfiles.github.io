# Aliases de Sudo

Atalhos para operações de superusuario.

## Visão Geral

Estes aliases são definidos em `sudo.aliases.sh` e carregados automaticamente pelo Chezmoi.

## Referência

| Alias | Descrição |
|-------|-------------|
| `root` | Executar comando como superusuario |
| `s` | Executar comando como superusuario |
| `su` | Executar comando como superusuario |
| `sudo` | Executar comando como superusuario |
