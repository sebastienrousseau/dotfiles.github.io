# Aliases de PNPM

Atalhos para o gerenciador de pacotes rapido e eficiente PNPM.

## Visão Geral

Estes aliases são definidos em `pnpm.aliases.sh` e carregados automaticamente pelo Chezmoi.

## Referência

### Projeto

| Alias | Descrição |
|-------|-------------|
| `pni` | Inicializar novo projeto |
| `pnin` | Instalar dependencias do projeto |
| `pnb` | Build do projeto |
| `pnrun` | Executar script do projeto |
| `pnd` | Rodar projeto em modo dev |
| `pns` | Rodar projeto em modo serve |
| `pnst` | Iniciar projeto |
| `pnln` | Lint do projeto |
| `pnls` | Listar dependencias |
| `pnout` | Ver dependencias desatualizadas |
| `pnt` | Testar projeto |
| `pntc` | Testar com coverage |
| `pndoc` | Gerar documentacao |
| `pnpub` | Publicar projeto |

### Dependencias

| Alias | Descrição |
|-------|-------------|
| `pnwhy` | Ver porque a dependencia esta instalada |
| `pnui` | Atualizar dependencia interativamente |
| `pnuil` | Atualizar interativamente para ultima versão |
| `pnup` | Atualizar dependencia |
| `pnad` | Adicionar dependencia de desenvolvimento |
| `pnap` | Adicionar dependencia peer |
| `pnrm` | Remover dependencia |
| `pnun` | Desinstalar dependencias do projeto |

### Global

| Alias | Descrição |
|-------|-------------|
| `pnga` | Adicionar dependencia global |
| `pngls` | Listar dependencias globais |
| `pngrm` | Remover dependencia global |
| `pngu` | Atualizar dependencia global |

### Utilitarios

| Alias | Descrição |
|-------|-------------|
| `pnh` | Mostrar ajuda |
| `pnc` | Criar novo projeto |
| `pnsv` | Rodar projeto em modo servidor |
| `pnx` | Atalho para pnpx |
