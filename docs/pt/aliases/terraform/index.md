# Aliases de Terraform

Atalhos para infraestrutura como codigo com Terraform e Ansible.

## Visão Geral

Estes aliases são definidos em `terraform.aliases.sh` e carregados automaticamente pelo Chezmoi.

## Referência

### Terraform

| Alias | Descrição |
|-------|-------------|
| `tf` | Atalho do Terraform |
| `tfi` | Init |
| `tfp` | Plan |
| `tfa` | Apply |
| `tfaa` | Apply (auto-approve) |
| `tfd` | Destroy |
| `tff` | Format |
| `tfv` | Validate |

### Ansible

| Alias | Descrição |
|-------|-------------|
| `ans` | Atalho do Ansible |
| `ansp` | Ansible Playbook |
| `anslint` | Ansible Lint |
