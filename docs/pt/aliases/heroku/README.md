# Alias de Heroku

Gerenciar alias de Heroku. Parte da configuracao **Universal Dotfiles**.

![Dotfiles banner][banner]

Esses alias sao definidos em `heroku.aliases.sh` e sao carregados automaticamente pelo `chezmoi`.

## âš¡ Alias

Esses alias ajudam a gerenciar apps Heroku e recursos associados.
- `hk` atalho geral de Heroku.
- `hkb` abrir navegador na app atual.
- `hkba` mostrar addons da app.
- `hkconf` mostrar configuracao da app.
- `hkcs` escalar dynos da app.
- `hkinfo` mostrar informacoes da app.
- `hkl` mostrar logs recentes.
- `hklf` seguir logs em tempo real.
- `hklt` mostrar logs com tail.
- `hkop` abrir painel Heroku.
- `hkps` listar processos.
- `hkpsa` listar apps.
- `hkpsoff` parar todos os dynos.
- `hkpso` iniciar dynos.
- `hkrestart` reiniciar app.
- `hkstack` mostrar stack da app.
- `hkstackset` definir stack.
- `hkt` executar testes.
- `hktl` listar pipelines.
- `hktla` listar apps em um pipeline.
- `hktop` mostrar uso de processos.
- `hkreleases` listar releases.
- `hkrl` promover release atual.
- `hkrb` reverter para release anterior.
- `hkrf` forcar release (rollback se falhar).
- `hkrfi` forcar release (rollback, interativo).
