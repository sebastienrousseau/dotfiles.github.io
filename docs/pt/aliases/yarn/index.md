# Aliases de Yarn

Atalhos para o gerenciador de pacotes Yarn.

## Visão Geral

Estes aliases são definidos em `yarn.aliases.sh` e carregados automaticamente pelo Chezmoi.

## Referência

| Alias | Descrição |
|-------|-------------|
| `y` | Atalho do Yarn |
| `yi` | Instalar |
| `ya` | Adicionar dependencia |
| `yad` | Adicionar dependencia de desenvolvimento |
| `yrun` | Executar script |
| `ystem` | Iniciar |
