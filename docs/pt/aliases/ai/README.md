# Aliases de IA

Atalhos para ferramentas de IA e LLM. Acelere seu fluxo de trabalho assistido por IA.

## Visao Geral

Estes aliases sao definidos em `ai.aliases.sh` e carregados automaticamente pelo Chezmoi.

## Referencia

### GitHub Copilot

| Alias | Descricao |
|:---|:---|
| `ghcp` | Atalho do Copilot |
| `ghs` | Sugerir codigo |
| `ghe` | Explicar codigo |

### Ollama

| Alias | Descricao |
|:---|:---|
| `ol` | Atalho do Ollama |
| `olr` | Executar modelo |
