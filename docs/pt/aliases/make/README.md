# Aliases de Make

Atalhos para utilitarios GNU Make.

## Visao Geral

Estes aliases sao definidos em `make.aliases.sh` e carregados automaticamente pelo Chezmoi.

## Referencia

| Alias | Descricao |
|-------|-------------|
| `mk` | Make |
| `mkc` | Make clean |
| `mkd` | Make doc |
| `mkf` | Make format |
| `mkh` | Make help |
| `mki` | Make install |
| `mka` | Make all |
| `mkr` | Make run |
| `mkt` | Make test |
