# Aliases de Make

Atalhos para utilitarios GNU Make.

## Visão Geral

Estes aliases são definidos em `make.aliases.sh` e carregados automaticamente pelo Chezmoi.

## Referência

| Alias | Descrição |
|-------|-------------|
| `mk` | Make |
| `mkc` | Make clean |
| `mkd` | Make doc |
| `mkf` | Make format |
| `mkh` | Make help |
| `mki` | Make install |
| `mka` | Make all |
| `mkr` | Make run |
| `mkt` | Make test |
