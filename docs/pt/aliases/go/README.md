# Aliases de Go

Atalhos para desenvolvimento com a linguagem Go.

## Visao Geral

Estes aliases sao definidos em `go.aliases.sh` e carregados automaticamente pelo Chezmoi.

## Referencia

| Alias | Descricao |
|-------|-------------|
| `go` | Atalho do Go |
| `gor` | `go run` |
| `gob` | `go build` |
| `got` | `go test` |
| `gomt` | `go mod tidy` |
| `goget` | `go get` |
