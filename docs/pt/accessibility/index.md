# Acessibilidade

- Navegação por teclado é suportada.
- Indicadores de foco são visíveis.
- Preferências de movimento reduzido são respeitadas.

Reportar problemas de acessibilidade: [info@dotfiles.io](mailto:info@dotfiles.io).
