# Arşiv Takma Adları

Kısa arşiv komut adları.

Bu sayfa bir takma ad grubunu ve yaygın komut kısayollarını tanıtır.
Adlandırma kurallarını, beklenen girdileri ve güvenli çalıştırma kalıplarını anlamak için önce bu sayfayı okuyun.
Ardından tam komut envanteri ve kesin sözdizimi için referans ekini açın.
Bu takma adlar tekrarlanan yazmayı azaltır ve macOS, Linux ve WSL'de tutarlı iş akışı davranışını korur.

## Ne zaman kullanılır

Öngörülebilir komut kalıplarına ihtiyaç duyduğunuzda günlük kabuk çalışmanızdan önce bu sayfayı kullanın.

## Neler dahil

- Takma adlar: 40
- Yardımcı fonksiyonlar: 3
- Kaynak dosyalar: 1

## Hızlı Örnekler

- `ac` → `'compress'        # Genel sıkıştırma (Arşiv Oluştur)`
- `acl` → `'compress_large' # Eski compress_large`
- `bak` → `'backup'         # Hızlı yedekleme fonksiyonu`
- `c7z` → `'7z a' # 7z arşivi oluştur`
- `cbz` → `'bzip2 -zk' # bzip2 ile sıkıştır`
- `cgz` → `'gzip -cv' # gzip ile sıkıştır`
- `clz4` → `'lz4 -zc' # lz4 ile sıkıştır`
- `crar` → `'rar a' # rar arşivi oluştur`
- `ctar` → `'tar -cvf'         # tar arşivi oluştur`
- `ctbz` → `'tar -jcvf'        # tar.bz2 arşivi oluştur`
- `ctgz` → `'tar -zcvf'        # tar.gz arşivi oluştur`
- `ctxz` → `'tar -Jcvf'        # tar.xz arşivi oluştur`
