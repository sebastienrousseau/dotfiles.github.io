# 權限別名

權限與擁有者指令簡寫。

本頁介紹一個別名群組及其常用指令捷徑。
請先閱讀以了解命名慣例、預期輸入和安全執行模式。
然後開啟參考附錄查看完整指令清單和精確語法。
這些別名減少重複輸入，並在 macOS、Linux 和 WSL 上保持一致的工作流程。

## 使用時機

在需要可預測指令模式的日常 Shell 工作前使用本頁。

## 包含內容

- 別名數：17
- 輔助函式：0
- 來源檔案：1

## 快速範例

- `000` → `chmod -R 000`
- `400` → `chmod -R 400`
- `444` → `chmod -R 444`
- `600` → `chmod -R 600`
- `644` → `chmod -R 644`
- `755` → `chmod -R 755`
- `764` → `chmod -R 764`
- `chgrp` → `chgrp -v`
- `chgrpr` → `chgrp -Rv`
- `chgrpu` → `chgrp -Rv ${USER}`
- `chmod` → `chmod -v`
- `chmodr` → `chmod -Rv`
