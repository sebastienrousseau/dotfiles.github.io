# 現代工具別名

現代工具指令簡寫。

本頁介紹一個別名群組及其常用指令捷徑。
請先閱讀以了解命名慣例、預期輸入和安全執行模式。
然後開啟參考附錄查看完整指令清單和精確語法。
這些別名減少重複輸入，並在 macOS、Linux 和 WSL 上保持一致的工作流程。

## 使用時機

在需要可預測指令模式的日常 Shell 工作前使用本頁。

## 包含內容

- 別名數：14
- 輔助函式：0
- 來源檔案：1

## 快速範例

- `cat` → `dot_cat`
- `fz` → `fzf --preview "bat --color=always --style=numbers {} 2>/dev/null | head -500 || cat {} 2>/dev/null | head -500"`
- `l` → `dot_ls`
- `la` → `dot_la`
- `ll` → `dot_ll`
- `llm` → `command ls -ltA`
- `lr` → `dot_lr`
- `lra` → `dot_lra`
- `ls` → `dot_ls`
- `lt` → `dot_lt`
- `lta` → `dot_lta`
- `lx` → `command ls -la`
