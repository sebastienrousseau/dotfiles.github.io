# Subversion 別名

SVN 指令簡寫。

本頁介紹一個別名群組及其常用指令捷徑。
請先閱讀以了解命名慣例、預期輸入和安全執行模式。
然後開啟參考附錄查看完整指令清單和精確語法。
這些別名減少重複輸入，並在 macOS、Linux 和 WSL 上保持一致的工作流程。

## 使用時機

在需要可預測指令模式的日常 Shell 工作前使用本頁。

## 包含內容

- 別名數：74
- 輔助函式：0
- 來源檔案：2

## 快速範例

- `sad` → `'svn add'        # 將新檔案和目錄納入版本控制`
- `sau` → `'svn auth'       # 管理快取的認證憑證`
- `sbl` → `'svn blame'      # 顯示檔案每行最後（或下次）變更的時間`
- `scg` → `'svn changelist' # 關聯（或取消關聯）變更清單 CLNAME 與指定檔案`
- `sci` → `'svn commit'     # 將工作副本的變更傳送到儲存庫`
- `scl` → `'svn cleanup'    # 復原中斷的操作或移除不需要的檔案`
