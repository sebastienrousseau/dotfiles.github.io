# Python 別名

Python 指令簡寫。

本頁介紹一個別名群組及其常用指令捷徑。
請先閱讀以了解命名慣例、預期輸入和安全執行模式。
然後開啟參考附錄查看完整指令清單和精確語法。
這些別名減少重複輸入，並在 macOS、Linux 和 WSL 上保持一致的工作流程。

## 使用時機

在需要可預測指令模式的日常 Shell 工作前使用本頁。

## 包含內容

- 別名數：44
- 輔助函式：2
- 來源檔案：1

## 快速範例

- `black` → `'python -m black' # 使用 black 格式化程式碼`
- `deact` → `'deactivate'                     # 停用虛擬環境`
- `ipy` → `'ipython'           # 互動式 Python Shell`
- `lint` → `'pylint'           # 程式碼靜態檢查`
- `mkvenv` → `'python -m venv .venv'          # 在目前目錄建立 .venv`
- `mkvenvlocal` → `'python -m venv ./venv'    # 在目前目錄建立 ./venv`
- `mypy` → `'python -m mypy'   # 靜態型別檢查`
- `pep8` → `'autopep8'         # 程式碼格式化`
- `pipf` → `'pip freeze'                      # 顯示凍結的套件需求`
- `pipi` → `'pip install'                     # 安裝套件`
- `pipl` → `'pip list'                        # 列出已安裝套件`
- `pipout` → `'pip freeze > requirements.txt' # 儲存套件需求`
