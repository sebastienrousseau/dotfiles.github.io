# 壓縮檔別名

壓縮檔指令簡寫。

本頁介紹一個別名群組及其常用指令捷徑。
請先閱讀以了解命名慣例、預期輸入和安全執行模式。
然後開啟參考附錄查看完整指令清單和精確語法。
這些別名減少重複輸入，並在 macOS、Linux 和 WSL 上保持一致的工作流程。

## 使用時機

在需要可預測指令模式的日常 Shell 工作前使用本頁。

## 包含內容

- 別名數：40
- 輔助函式：3
- 來源檔案：1

## 快速範例

- `ac` → `'compress'        # 通用壓縮（Archive Create）`
- `acl` → `'compress_large' # 舊版 compress_large`
- `bak` → `'backup'         # 快速備份函式`
- `c7z` → `'7z a' # 建立 7z 壓縮檔`
- `cbz` → `'bzip2 -zk' # 使用 bzip2 壓縮`
- `cgz` → `'gzip -cv' # 使用 gzip 壓縮`
- `clz4` → `'lz4 -zc' # 使用 lz4 壓縮`
- `crar` → `'rar a' # 建立 rar 壓縮檔`
- `ctar` → `'tar -cvf'         # 建立 tar 壓縮檔`
- `ctbz` → `'tar -jcvf'        # 建立 tar.bz2 壓縮檔`
- `ctgz` → `'tar -zcvf'        # 建立 tar.gz 壓縮檔`
- `ctxz` → `'tar -Jcvf'        # 建立 tar.xz 壓縮檔`
