# קיצורי Terraform

קיצורי דרך לתשתית כקוד עם Terraform ו-Ansible.

## סקירה

קיצורים אלה מוגדרים ב-`terraform.aliases.sh` ונטענים אוטומטית על ידי Chezmoi.

## הפניה

### Terraform

| קיצור | תיאור |
|:---|:---|
| `tf` | קיצור דרך ל-Terraform |
| `tfi` | אתחל תצורה |
| `tfp` | תכנן שינויים |
| `tfa` | החל שינויים |
| `tfaa` | החל עם אישור אוטומטי |
| `tfd` | הרס תשתית |
| `tff` | עצב תצורה |
| `tfv` | אמת תצורה |

### Ansible

| קיצור | תיאור |
|:---|:---|
| `ans` | קיצור דרך ל-Ansible |
| `ansp` | הרץ Ansible Playbook |
| `anslint` | בדוק קבצי Ansible |
