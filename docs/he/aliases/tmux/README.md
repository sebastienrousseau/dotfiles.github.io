# קיצורי Tmux

נהל סשני טרמינל, חלונות ופאנלים עם tmux.

## סקירה

קיצורים אלה מוגדרים ב-`tmux.aliases.sh` ונטענים אוטומטית על ידי Chezmoi.

## הפניה

| קיצור | תיאור |
|:---|:---|
| `tm` | הפעל tmux |
| `tma` | התחבר לסשן האחרון |
| `tmat` | התחבר לסשן ספציפי |
| `tmks` | סגור את כל הסשנים מלבד הנוכחי |
| `tmka` | סגור את כל הסשנים |
| `tml` | רשום את כל הסשנים |
| `tmn` | סשן חדש ללא שם |
| `tms` | סשן חדש עם שם |
| `tmr` | טען מחדש תצורת tmux |
| `tmls` | רשום חלונות |
| `tmlp` | רשום פאנלים |
| `tmi` | הצג מידע tmux |
