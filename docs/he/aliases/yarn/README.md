# קיצורי Yarn

קיצורי דרך למנהל החבילות Yarn של JavaScript.

## סקירה

קיצורים אלה מוגדרים ב-`yarn.aliases.sh` ונטענים אוטומטית על ידי Chezmoi.

## הפניה

| קיצור | תיאור |
|:---|:---|
| `y` | קיצור דרך ל-Yarn |
| `yi` | התקן תלויות |
| `ya` | הוסף תלות |
| `yad` | הוסף תלות פיתוח |
| `yrun` | הרץ סקריפט |
| `ystem` | הפעל פרויקט |
