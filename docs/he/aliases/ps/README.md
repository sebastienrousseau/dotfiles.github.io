# קיצורי PS

נטר ונהל תהליכים פעילים עם קיצורים פשוטים.

## סקירה

קיצורים אלה מוגדרים ב-`ps.aliases.sh` ונטענים אוטומטית על ידי Chezmoi.

## הפניה

| קיצור | תיאור |
|:---|:---|
| `pid` | הצג UID, PID, PID הורה, שימוש ב-CPU, זמן התחלה, TTY ופקודה |
| `ps` | הצג את כל התהליכים |
| `psa` | הצג רשימת כל התהליכים |
