# קיצורי Go

ייעל את זרימת העבודה בפיתוח Go עם קיצורי מעטפת אינטואיטיביים.

## סקירה

קיצורים אלה מוגדרים ב-`go.aliases.sh` ונטענים אוטומטית על ידי chezmoi. הם מספקים גישה מהירה לפקודות Go נפוצות להרצה, בנייה, בדיקה וניהול מודולים.

## הפניה

| קיצור    | פקודה       | תיאור                        |
|----------|---------------|-----------------------------------|
| `go`     | `go`          | קיצור דרך ל-Go                        |
| `gor`    | `go run`      | הרץ תוכנית Go                     |
| `gob`    | `go build`    | בנה חבילת Go                   |
| `got`    | `go test`     | הרץ בדיקות Go                       |
| `gomt`   | `go mod tidy` | סדר תלויות מודול Go        |
| `goget`  | `go get`      | הורד והתקן חבילות Go   |
