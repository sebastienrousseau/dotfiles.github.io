# קיצורי Compliance

כלי מעטפת לתאימות רגולטורית ואכיפת פרטיות.

## סקירה

קיצורי compliance מספקים בקרות חיוניות לניהול פרטיות ושמירה על נתיבי ביקורת בסביבת המעטפת שלך. כלים אלה עוזרים להשבית טלמטריה בכלי CLI נפוצים ולעקוב אחר שינויי תצורה לדרישות תאימות.

## הפניה

### בקרות

| קיצור | תיאור |
|-------|-------------|
| `privacy-mode` | השבת טלמטריה לכלי CLI (Homebrew, Dotnet, Azure וכו') לסשן הנוכחי. |
| `dot-audit` | הצג נתיב ביקורת תצורה אחרון (היסטוריית Git או יומן ביקורת). |
