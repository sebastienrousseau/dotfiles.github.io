# קיצורי Subversion

קיצורי דרך לפעולות ניהול גרסאות Subversion.

## סקירה

קיצורים אלה מוגדרים ב-`subversion.aliases.sh` ונטענים אוטומטית על ידי Chezmoi.

## הפניה

| קיצור | תיאור |
|:---|:---|
| `sad` | הוסף קבצים ותיקיות לניהול גרסאות |
| `sau` | נהל אישורי אימות מאוחסנים |
| `sbl` | הצג מתי כל שורה שונתה לאחרונה |
| `scg` | שייך רשימת שינויים לקבצים |
| `sci` | בצע commit של שינויים למאגר |
| `scl` | נקה עותק עבודה |
| `sco` | הורד עותק עבודה |
| `scp` | העתק קבצים ותיקיות |
| `sct` | הצג תוכן של קבצים או כתובות URL |
| `sdi` | הצג שינויים מקומיים או הבדלים |
| `sdl` | הסר קבצים מניהול גרסאות |
| `shp` | הצג עזרה לפקודות |
| `sin` | הצג מידע על פריט |
| `sip` | ייבא קובץ לא מנוהל למאגר |
| `slg` | הצג הודעות log |
| `slk` | נעל נתיבי עותק עבודה |
| `sls` | רשום ערכי תיקייה |
| `smd` | צור תיקייה חדשה תחת ניהול גרסאות |
| `smg` | מזג שינויים לעותק עבודה |
| `smgi` | הצג מידע הקשור למיזוג |
| `smv` | הזז או שנה שם של פריט |
| `sp` | הגדר ערך מאפיין |
| `spdl` | הסר מאפיין |
| `spdt` | ערוך מאפיין עם עורך חיצוני |
| `spgt` | הדפס ערך מאפיין |
| `sph` | החל טלאי |
| `spls` | רשום את כל המאפיינים |
| `srl` | העבר עותק עבודה לכתובת URL שונה |
| `srs` | פתור התנגשויות |
| `srsd` | הסר מצב התנגשות |
| `srv` | שחזר עותק עבודה למצב מקורי |
| `sst` | הדפס סטטוס של עותק עבודה |
| `ssw` | החלף עותק עבודה לכתובת URL שונה |
| `sulk` | שחרר נעילת נתיבי עותק עבודה |
| `sup` | עדכן עותק עבודה מהמאגר |
| `supg` | שדרג פורמט אחסון מטא-דאטה |
| `sxp` | צור עותק לא מנוהל של עץ |
