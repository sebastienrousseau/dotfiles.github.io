# System קיצורים

פקודות system קצרות.

דף זה מציג קבוצת קיצורים וקיצורי הפקודות הנפוצים שלה. קראו דף זה תחילה כדי להבין מוסכמות שמות, קלטים צפויים ודפוסי ביצוע בטוחים. לאחר מכן פתחו את נספח ההפניה לרשימת הפקודות המלאה והתחביר המדויק. קיצורים אלו מפחיתים הקלדה חוזרת ושומרים על התנהגות זרימת עבודה עקבית ב-macOS, Linux ו-WSL.

## מתי להשתמש

השתמשו בדף זה לפני עבודת Shell יומיומית כאשר אתם צריכים דפוסי פקודות צפויים.

## מה כלול

- קיצורים: 24
- פונקציות עזר: 0
- קבצי מקור: 1

## דוגמאות מהירות

- `h` → `history`
- `p` → `pwd`
- `path` → `echo -e ${PATH//:/\\n}`
- `ports` → `netstat -tulanp`
- `top` → `sudo btop`
- `now` → `date +"%T"`
- `nowdate` → `date +"%d-%m-%Y"`
- `wget` → `wget -c`
- `speedtest` → `curl -s ... | python3 -`
- `ping` → `ping -c 5`
- `fastping` → `ping -c 100 -s 1`
- `localip` → `ipconfig getifaddr en0`
