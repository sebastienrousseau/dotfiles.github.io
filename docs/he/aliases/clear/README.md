# קיצורי Clear

ייעל את זרימת העבודה בטרמינל עם קיצורי ניקוי מסך מהירים.

## סקירה

קיצורי clear מספקים דרכים יעילות לאפס את תצוגת הטרמינל שלך תוך הצגת הקשר שימושי אופציונלית כמו תוכן תיקיות, היסטוריית פקודות או עצי קבצים. קיצורים אלה מוגדרים ב-`clear.aliases.sh` ונטענים אוטומטית על ידי chezmoi.

## הפניה

| קיצור | תיאור |
|-------|-------------|
| `c` | נקה את מסך הטרמינל |
| `cdw` | עבור לתיקיית העבודה |
| `cl` | נקה את מסך הטרמינל (קיצור ל-`c`) |
| `clc` | נקה מסך והצג תוכן תיקיה |
| `clh` | נקה מסך והצג היסטוריית פקודות |
| `clp` | נקה מסך והדפס תיקיית עבודה |
| `clt` | נקה מסך והצג עץ תיקיות |
| `cpl` | נקה מסך והצג תיקיית עבודה עם תוכן |
| `cplt` | נקה מסך והצג תיקיית עבודה עם עץ |
