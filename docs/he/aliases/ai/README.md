# קיצורי AI

קיצורי דרך לכלי AI ו-LLM. האץ את זרימת העבודה בעזרת AI.

## סקירה

קיצורים אלה מוגדרים ב-`ai.aliases.sh` ונטענים אוטומטית על ידי Chezmoi.

## הפניה

### GitHub Copilot

| קיצור | תיאור |
|:---|:---|
| `ghcp` | קיצור דרך ל-Copilot |
| `ghs` | הצע קוד |
| `ghe` | הסבר קוד |

### Ollama

| קיצור | תיאור |
|:---|:---|
| `ol` | קיצור דרך ל-Ollama |
| `olr` | הפעל מודל |
