# קיצורי Wget

הורד קבצים מהרשת עם תמיכה בהמשך הורדה.

## סקירה

קיצורים אלה מוגדרים ב-`wget.aliases.sh` ונטענים אוטומטית על ידי Chezmoi.

## הפניה

| קיצור | תיאור |
|:---|:---|
| `wg` | קיצור דרך ל-wget |
| `wgc` | wget עם המשך |
| `wge` | wget עם robots=off |
| `wget` | wget עם המשך |
