# קיצורי Mkdir

ייעל יצירת תיקיות עם מוסכמות שמות חכמות.

## סקירה

קיצורי mkdir מספקים קיצורים ליצירת תיקיות עם חותמות תאריך ושעה אוטומטיות. קיצורים אלה מוגדרים ב-`mkdir.aliases.sh` ונטענים אוטומטית על ידי chezmoi.

## הפניה

| קיצור | תיאור |
|-------|-------------|
| `md` | צור תיקיה |
| `mdd` | צור תיקיה עם תאריך |
| `mdt` | צור תיקיה עם שעה |
| `mde` | צור תיקיית דוגמה עם תאריך נוכחי |
| `mdn` | צור תיקיית הערות עם תאריך נוכחי |
| `mdw` | צור תיקיית עבודה עם תאריך נוכחי |
