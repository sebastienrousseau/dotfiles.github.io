# קיצורי Fonts

קיצורי מעטפת לניהול גופני מערכת ומטמונים.

## סקירה

קיצורי fonts מספקים קיצורי דרך נוחים למשימות ניהול גופנים נפוצות. פקודות אלה מפשטות עדכון מטמון הגופנים לאחר התקנת גופנים חדשים והצגת כל משפחות הגופנים הזמינות במערכת שלך.

## הפניה

### פקודות

| קיצור | פקודה | תיאור |
|-------|---------|-------------|
| `update-fonts` | `fc-cache -fv` | מעדכן את מטמון גופני המערכת |
| `list-fonts` | — | מציג את כל משפחות הגופנים המותקנות |

### גופנים מותקנים

הגופנים הבאים מוגדרים כברירת מחדל:

| גופן | מטרה |
|------|---------|
| JetBrainsMono Nerd Font | גופן טרמינל ראשי |
| Symbols Nerd Font | גיבוי לאייקונים |
