# קיצורי Python

קיצורים לפיתוח Python. מפרש, חבילות וסביבות וירטואליות.

## סקירה

קיצורים אלה מוגדרים ב-`python.aliases.sh` ונטענים אוטומטית על ידי Chezmoi.

## הפניה

| קיצור | פקודה | תיאור |
|:---|:---|:---|
| `py` | `python` | מפרש Python |
| `pip` | `pip3` | מתקין חבילות |
| `v` | `virtualenv` | סביבה וירטואלית |
| `uv` | `uv` | מנהל חבילות מהיר |
| `uvpi` | `uv pip install` | התקן עם uv |
