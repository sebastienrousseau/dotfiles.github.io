# קיצורי macOS

קיצורי מעטפת לניהול מערכת macOS ותחזוקה.

## סקירה

קיצורים אלה מוגדרים ב-`macOS.aliases.sh` ונטענים אוטומטית על ידי chezmoi. הם מספקים גישה מהירה לפעולות macOS נפוצות כולל שליטה ב-Finder, תחזוקת מערכת וכלי פיתוח.

## הפניה

### ניהול מערכת

| קיצור | תיאור |
|-------|-------------|
| `lockScreen` | נעל את המסך |
| `wifiOn` | הפעל Wi-Fi |
| `wifiOff` | כבה Wi-Fi |
| `vp` | אמת הרשאות macOS |
| `vv` | אמת נפח macOS |

### Finder

| קיצור | תיאור |
|-------|-------------|
| `ofd` | פתח את התיקיה הנוכחית ב-Finder |
| `finderShowHidden` | הצג קבצים מוסתרים ב-Finder |
| `finderHideHidden` | הסתר קבצים מוסתרים ב-Finder |
| `showDesktopIcons` | הצג את כל האייקונים על שולחן העבודה |
| `hideDesktopIcons` | הסתר את כל האייקונים על שולחן העבודה |

### ניקוי

| קיצור | תיאור |
|-------|-------------|
| `clds` | מחק רקורסיבית קבצי .DS_Store |
| `clls` | נקה LaunchServices להסרת כפילויות בתפריט Open With |
| `trash` | רוקן את האשפה בכל הנפחים המחוברים ובדיסק הראשי |

### כלי פיתוח

| קיצור | תיאור |
|-------|-------------|
| `xcode` | הפעל Xcode |
| `purge` | טהר DerivedData של Xcode |
| `iphone` | פתח סימולטורי מכשירים |
| `safariSafeMode` | הפעל Safari במצב בטוח |

### תצוגה

| קיצור | תיאור |
|-------|-------------|
| `screensaverDesktop` | הרץ שומר מסך על שולחן העבודה |
