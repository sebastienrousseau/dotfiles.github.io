# קיצורי Chmod

קיצורי דרך לניהול הרשאות קבצים ותיקיות.

## סקירה

קיצורי chmod מספקים דרך יעילה לנהל הרשאות קבצים ותיקיות משורת הפקודה. במקום לזכור קודי הרשאות מספריים, השתמש בקיצורים אינטואיטיביים כמו `chmod_755` או `chmod_u+x` להגדרת מצב הגישה שאתה צריך במהירות.

קיצורים אלה מוגדרים ב-`chmod.aliases.sh` ונטענים אוטומטית על ידי chezmoi.

## הפניה

### קיצורי הרשאות נפוצים

| קיצור | הרשאות | תיאור |
|-------|-------------|-------------|
| `chmod_000` | `----------` | אין הרשאות לאף אחד |
| `chmod_400` | `r--------` | קריאה בלבד לבעלים |
| `chmod_444` | `r--r--r--` | קריאה בלבד לכולם |
| `chmod_600` | `rw-------` | קריאה/כתיבה לבעלים |
| `chmod_644` | `rw-r--r--` | קריאה/כתיבה לבעלים, קריאה בלבד לאחרים |
| `chmod_666` | `rw-rw-rw-` | קריאה/כתיבה לכולם |
| `chmod_755` | `rwxr-xr-x` | הכל לבעלים, קריאה/הרצה לאחרים |
| `chmod_764` | `rwxrw-r--` | הכל לבעלים, קריאה/כתיבה לקבוצה |
| `chmod_777` | `rwxrwxrwx` | הרשאות מלאות לכולם |

### קיצורי הרשאות בעלים

| קיצור | תיאור |
|-------|-------------|
| `chmod_u+x` | הוסף הרשאת הרצה לבעלים |
| `chmod_u-x` | הסר הרשאת הרצה מהבעלים |
| `chmod_u+w` | הוסף הרשאת כתיבה לבעלים |
| `chmod_u-w` | הסר הרשאת כתיבה מהבעלים |
| `chmod_u+r` | הוסף הרשאת קריאה לבעלים |
| `chmod_u-r` | הסר הרשאת קריאה מהבעלים |

### קיצורי הרשאות קבוצה

| קיצור | תיאור |
|-------|-------------|
| `chmod_g+x` | הוסף הרשאת הרצה לקבוצה |
| `chmod_g-x` | הסר הרשאת הרצה מהקבוצה |
| `chmod_g+w` | הוסף הרשאת כתיבה לקבוצה |
| `chmod_g-w` | הסר הרשאת כתיבה מהקבוצה |
| `chmod_g+r` | הוסף הרשאת קריאה לקבוצה |
| `chmod_g-r` | הסר הרשאת קריאה מהקבוצה |

### קיצורי הרשאות אחרים

| קיצור | תיאור |
|-------|-------------|
| `chmod_o+x` | הוסף הרשאת הרצה לאחרים |
| `chmod_o-x` | הסר הרשאת הרצה מאחרים |
| `chmod_o+w` | הוסף הרשאת כתיבה לאחרים |
| `chmod_o-w` | הסר הרשאת כתיבה מאחרים |
| `chmod_o+r` | הוסף הרשאת קריאה לאחרים |
| `chmod_o-r` | הסר הרשאת קריאה מאחרים |

### קיצורי סוג קובץ

| קיצור | תיאור |
|-------|-------------|
| `chmod_755d` | הגדר הרשאות כל התיקיות ל-`rwxr-xr-x` |
| `chmod_644f` | הגדר הרשאות כל הקבצים ל-`rw-r--r--` |

### פונקציית הרשאות רקורסיבית

פונקציית `change_permission` מחילה הרשאות רקורסיבית עם הנחיית אישור:

```bash
change_permission 755 /path/to/directory -R
```
