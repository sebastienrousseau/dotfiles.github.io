# קיצורי עדכון

שמור על המערכת והחבילות שלך מעודכנים.

## סקירה

קיצורים אלה מוגדרים ב-`update.aliases.sh` ונטענים אוטומטית על ידי Chezmoi.

## הפניה

### macOS

| קיצור | תיאור |
|:---|:---|
| `upd` | עדכן תוכנת macOS, Homebrew, Casks, App Store, Rust, Ruby, Python ו-Node.js |

### Linux

| קיצור | תיאור |
|:---|:---|
| `open` | פתח קובץ או כתובת URL באפליקציה המועדפת |
| `pbcopy` | העתק ללוח |
| `pbpaste` | הדבק מהלוח |
| `upd` | עדכן חבילות Linux, pnpm, Rust, Ruby, Python ו-Node.js |
