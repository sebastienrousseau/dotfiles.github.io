# קיצורי Interactive

פקודות מעטפת בטוחות עם הנחיות אישור לפני פעולות הרסניות.

## סקירה

קיצורי interactive מוסיפים הנחיות אישור לפקודות מעטפת נפוצות שיכולות לשנות או למחוק קבצים לצמיתות. מוגדרים ב-`interactive.aliases.sh` ונטענים אוטומטית על ידי chezmoi, קיצורים אלה עוזרים למנוע אובדן נתונים בטעות על ידי דרישת אישור משתמש לפני ביצוע פעולות פוטנציאלית הרסניות.

## הפניה

### פעולות קבצים

| קיצור | תיאור |
|-------|-------------|
| `cp` | העתק קבצים ותיקיות באופן אינטראקטיבי עם פלט מפורט. מנחה לפני דריסת קבצים קיימים. |
| `mv` | העבר או שנה שם קבצים באופן אינטראקטיבי עם פלט מפורט. מנחה לפני דריסת קבצים קיימים. |
| `ln` | צור קישורים סימבוליים באופן אינטראקטיבי עם פלט מפורט. מנחה לפני דריסת קישורים קיימים. |

### הסרת קבצים

| קיצור | תיאור |
|-------|-------------|
| `rm` | הסר קבצים או תיקיות באופן אינטראקטיבי עם פלט מפורט. מנחה לפני כל הסרה. |
| `del` | הסר קבצים או תיקיות באופן אינטראקטיבי ורקורסיבי עם פלט מפורט. מנחה לפני כל הסרה. |
| `zap` | קיצור ל-`rm`. הסר קבצים או תיקיות באופן אינטראקטיבי עם פלט מפורט. |

### ניהול אשפה

| קיצור | תיאור |
|-------|-------------|
| `bin` | הסר את כל הקבצים בתיקיית האשפה (`~/.Trash`) בכוח ורקורסיבית. |

### הרשאות וכלי עזר

| קיצור | תיאור |
|-------|-------------|
| `chmod` | שנה הרשאות קובץ או תיקיה עם פלט מפורט. |
| `chown` | שנה בעלים וקבוצה של קובץ או תיקיה עם פלט מפורט. |
| `diff` | השווה והראה הבדלים בין שני קבצים בפורמט מאוחד. |
| `grep` | חפש דפוס בקבצים או פלט, הצג מספרי שורות וללא רגישות לאותיות. |
| `mkdir` | צור תיקיה חדשה עם תיקיות הורה לפי הצורך ופלט מפורט. |
