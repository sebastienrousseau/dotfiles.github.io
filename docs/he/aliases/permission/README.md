# קיצורי Permission

קיצורים לניהול הרשאות קבצים ותיקיות בסביבת המעטפת שלך.

## סקירה

קיצורים אלה מוגדרים ב-`permission.aliases.sh` ונטענים אוטומטית על ידי chezmoi. הם מספקים גישה מהירה לדפוסי הרשאות נפוצים ופקודות בעלות.

## הפניה

### קיצורי הרשאות מספריים

| קיצור | תיאור |
|-------|-------------|
| `000` | הגדר הרשאות ללא קריאה, כתיבה או הרצה לכולם |
| `400` | הגדר הרשאות ללא קריאה או כתיבה, אבל אפשר הרצה לבעלים בלבד |
| `444` | הגדר הרשאות ללא כתיבה או הרצה, אבל אפשר קריאה לכולם |
| `600` | הגדר הרשאות לקריאה וכתיבה לבעלים בלבד |
| `644` | הגדר הרשאות לקריאה לכולם, אבל כתיבה רק לבעלים |
| `666` | הגדר הרשאות לקריאה וכתיבה לכולם |
| `755` | הגדר הרשאות לקריאה, כתיבה והרצה לבעלים, רק קריאה והרצה לקבוצה ואחרים |
| `764` | הגדר הרשאות לקריאה וכתיבה לבעלים ולקבוצה, רק קריאה לאחרים |
| `777` | הגדר הרשאות לקריאה, כתיבה והרצה לכולם |

### פקודות מצב קובץ

| קיצור | תיאור |
|-------|-------------|
| `chmod` | שנה סיביות מצב קובץ |
| `chmodr` | שנה סיביות מצב קובץ רקורסיבית |
| `chmodu` | שנה סיביות מצב קובץ רקורסיבית למשתמש הנוכחי |
| `chmox` | הפוך קובץ להפעלה |

### פקודות בעלות

| קיצור | תיאור |
|-------|-------------|
| `chown` | שנה בעלים וקבוצה של קובץ |
| `chownr` | שנה בעלים וקבוצה של קובץ רקורסיבית |
| `chownu` | שנה בעלים וקבוצה של קובץ רקורסיבית למשתמש הנוכחי |
| `chgrp` | שנה בעלות קבוצה של קבצים או תיקיות |
| `chgrpr` | שנה בעלות קבוצה של קבצים או תיקיות רקורסיבית |
| `chgrpu` | שנה בעלות קבוצה של קבצים או תיקיות רקורסיבית למשתמש הנוכחי |
