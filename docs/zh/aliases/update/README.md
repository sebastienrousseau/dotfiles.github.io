# Update 别名

保持系统和软件包最新。

## 概述

这些别名定义在 `update.aliases.sh` 中，由 Chezmoi 自动加载。

## 参考

### macOS

| 别名 | 描述 |
|:---|:---|
| `upd` | 更新 macOS 软件、Homebrew、Casks、App Store、Rust、Ruby、Python 和 Node.js |

### Linux

| 别名 | 描述 |
|:---|:---|
| `open` | 在首选应用程序中打开文件或 URL |
| `pbcopy` | 复制到剪贴板 |
| `pbpaste` | 从剪贴板粘贴 |
| `upd` | 更新 Linux 包、pnpm、Rust、Ruby、Python 和 Node.js |
