# 列表别名

`list.aliases.sh` 文件为列出文件和目录创建了有用的快捷别名。

列表别名是一组别名，允许您以更交互式的方式与 `ls` 命令交互，以便显示目录的内容。

对于每个命名非目录类型文件的操作数，`ls` 显示其名称以及任何请求的相关信息。
对于每个命名目录类型文件的操作数，`ls` 显示该目录中包含的文件名称，以及任何请求的相关信息。

## ls

[ls](https://zh.wikipedia.org/wiki/Ls) 是一个列出目录内容的命令。

| 别名   | 命令                 | 描述                                         |
| ------ | -------------------- | -------------------------------------------- |
| l.     | `ls -dlhF .\* grep -v "^d"` | 列出隐藏文件。                               |
| l      | `ls -lFh`            | 大小，显示类型，人类可读。                   |
| l1     | `ls -1`              | 每行显示一个文件。                           |
| la     | `ls -lAFh`           | 所有文件，显示类型，人类可读。               |
| labc   | `ls -lap`            | 按字母顺序列出所有文件。                     |
| lc     | `wc -l`              | 统计文件中的行数。                           |
| lct    | `ls -lcrh`           | 按时间列出文件，最新的在前。                 |
| ld     | `ls -ltrh`           | 按日期排序，最旧的在前。                     |
| ldir   | `ls -l egrep '^d'`   | 仅列出目录。                                 |
| ldot   | `l.`                 | 列出隐藏文件。                               |
| left   | `ls -t -1`           | 按日期列出文件，最近的在后。                 |
| right  | `ls -t -1r`          | 按日期列出文件，最近的在前。                 |
| lf     | `ls -l egrep -v '^d'`| 仅列出文件。                                 |
| lk     | `ls -lSrh`           | 按大小排序，最大的在前。                     |
| ll     | `la`                 | 长列表，显示几乎所有，显示类型，人类可读。   |
| lla    | `ls -l -d $PWD/*`    | 列出当前目录中所有文件的完整路径。           |
| locale | `locale -a grep UTF-8` | 列出所有可用的区域设置。                     |
| lp     | `sudo lsof -i -T -n` | 列出所有打开的端口。                         |
| lr     | `ls -lRh`            | 递归列表，显示类型，人类可读。               |
| ls     | `ls --color`         | 着色输出。                                   |
| lS     | `ls -1FSsh`          | 根据最后修改时间和大小对文件进行排序。       |
| lt     | `tree`               | 以树状格式列出目录内容。                     |
| lu     | `ls -lurh`           | 按日期排序，最旧的在前。                     |
| lw     | `ls -xAh`            | 宽列表，显示几乎所有，显示类型，人类可读。   |
| lx     | `ls sort -k 1,1 -t .`| 按扩展名排序。                               |
| lz     | `ls -lSr`            | 按大小排序，最小的在前。                     |
