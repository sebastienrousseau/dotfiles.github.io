# Rsync 别名

`rsync.aliases.sh` 文件为 [rsync](https://zh.wikipedia.org/wiki/Rsync) 创建了有用的快捷别名。

## Rsync

[Rsync](https://zh.wikipedia.org/wiki/Rsync) 是一个将文件和目录复制到另一个位置的命令。

| 别名  | 命令         | 描述                         |
| ----- | ------------ | ---------------------------- |
| rs    | `rsync -avz` | 带有详细信息和进度的 Rsync。 |
| rsync | `rs`         | 带有详细信息和进度的 Rsync。 |
