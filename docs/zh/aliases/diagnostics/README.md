# 诊断别名

用于自愈与系统健康检查的工具。

## 🩺 健康与修复

| Alias | 说明 |
|-------|------|
| `doc`, `dot-doctor` | 运行系统健康检查（`doctor.sh`）。 |
| `drift`, `dot-drift` | 检查本地文件是否偏离受管状态（`chezmoi verify`）。 |
| `heal`, `dot-heal` | 应用受管状态修复偏离（`chezmoi apply`）。 |
| `doc-full` | 运行 doctor 并显示扩展路径调试信息。 |
