# 合规别名

用于监管合规和隐私执行的工具。

## 🛡️ 控制

| 别名           | 描述                                                       |
| -------------- | ---------------------------------------------------------- |
| `privacy-mode` | 为当前会话禁用 CLI 工具（Homebrew, Dotnet, Azure 等）的遥测。 |
| `dot-audit`    | 显示最近的配置审计跟踪（Git 历史记录或审计日志）。         |

## 📚 文档

有关完整的 SOC2/ISO 控制映射，请参阅 [COMPLIANCE.md](../../../COMPLIANCE.md)。
