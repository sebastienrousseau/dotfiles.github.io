# 合规别名

用于合规与隐私执行的工具。

## 🛡️ 控制

| Alias | 说明 |
|-------|------|
| `privacy-mode` | 禁用 CLI 工具的遥测（Homebrew、Dotnet、Azure 等），仅限当前会话。 |
| `dot-audit` | 显示近期配置审计轨迹（Git 历史或审计日志）。 |
