# Mkdir 别名

使用智能命名约定简化目录创建。

## 概述

mkdir 别名提供了创建带有自动日期和时间戳目录的快捷方式。这些别名定义在 `mkdir.aliases.sh` 中，由 chezmoi 自动加载。

## 参考

| 别名 | 描述 |
|:---|:---|
| `md` | 创建目录 |
| `mdd` | 创建带日期的目录 |
| `mdt` | 创建带时间的目录 |
| `mde` | 创建带当前日期的示例目录 |
| `mdn` | 创建带当前日期的笔记目录 |
| `mdw` | 创建带当前日期的工作目录 |
