# PS 别名

使用简单的快捷方式监控和管理运行中的进程。

## 概述

这些别名定义在 `ps.aliases.sh` 中，由 Chezmoi 自动加载。

## 参考

| 别名 | 描述 |
|:---|:---|
| `pid` | 显示 UID、PID、父 PID、CPU 使用率、启动时间、TTY 和命令 |
| `ps` | 显示所有进程 |
| `psa` | 列出所有进程 |
