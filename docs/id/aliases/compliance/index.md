# Alias Compliance

Alat shell untuk kepatuhan regulasi dan penegakan privasi.

## Ikhtisar

Alias Compliance menyediakan kontrol penting untuk manajemen privasi dan pemeliharaan jejak audit di lingkungan shell Anda. Alias ini membantu menonaktifkan telemetri di seluruh alat CLI umum dan melacak perubahan konfigurasi untuk persyaratan kepatuhan.

## Referensi

### Kontrol

| Alias | Deskripsi |
|-------|-------------|
| `privacy-mode` | Menonaktifkan telemetri untuk alat CLI (Homebrew, Dotnet, Azure, dll.) di sesi saat ini. |
| `dot-audit` | Menampilkan jejak audit konfigurasi terbaru (riwayat Git atau log audit). |
