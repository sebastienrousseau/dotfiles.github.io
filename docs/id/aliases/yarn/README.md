# Alias Yarn

Kelola alias Yarn (JavaScript). Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `yarn.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

- `y` - Pintasan Yarn
- `yi` - Install
- `ya` - Tambah dependensi
- `yad` - Tambah dependensi development
- `yrun` - Jalankan skrip
- `ystem` - Start
