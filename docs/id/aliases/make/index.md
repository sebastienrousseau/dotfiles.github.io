# Alias Make

Kelola alias Make. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `make.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk perintah `make`.
- `mk` - Make.
- `mkc` - Make clean.
- `mkd` - Make doc.
- `mkf` - Make format.
- `mkh` - Make help.
- `mki` - Make install.
- `mka` - Make all.
- `mkr` - Make run.
- `mkt` - Make test.
