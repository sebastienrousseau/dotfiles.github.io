# Alias macOS

Kelola alias macOS. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `macOS.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk macOS.
- `clds` Hapus file .DS_Store secara rekursif.
- `clls` Bersihkan LaunchServices untuk menghapus duplikat di menu "Open With".
- `finderHideHidden` Sembunyikan file tersembunyi di Finder.
- `finderShowHidden` Tampilkan file tersembunyi di Finder.
- `hideDesktopIcons` Sembunyikan semua ikon di desktop.
- `iphone` Buka simulator perangkat.
- `lockScreen` Kunci layar Mac.
- `ofd` Buka direktori saat ini di jendela Finder.
- `purge` Purge DerivedData Xcode.
- `safariSafeMode` Mulai Safari dalam safe mode.
- `showDesktopIcons` Tampilkan semua ikon di desktop.
- `screensaverDesktop` Jalankan screensaver di desktop.
- `trash` Kosongkan Trash di semua volume yang terpasang dan HDD utama.
- `vp` Verifikasi izin macOS.
- `vv` Verifikasi volume macOS.
- `wifiOn` Nyalakan Wi-Fi.
- `wifiOff` Matikan Wi-Fi.
- `xcode` Luncurkan aplikasi Xcode di macOS.
