# Alias Mkdir

Kelola alias Mkdir. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `mkdir.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk perintah `mkdir`.
- `mde` Buat direktori contoh dengan tanggal saat ini.
- `md` Buat direktori.
- `mdd` Buat direktori dengan tanggal.
- `mdn` Buat direktori catatan dengan tanggal saat ini.
- `mdw` Buat direktori kerja dengan tanggal saat ini.
- `mdt` Buat direktori dengan waktu.
