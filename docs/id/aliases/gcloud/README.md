# Alias GCloud

Pintasan shell untuk mengelola Google Cloud Platform.

## Ikhtisar

Alias ini didefinisikan di `gcloud.aliases.sh` dan dimuat secara otomatis oleh Chezmoi.

## Referensi

| Alias | Deskripsi |
|:---|:---|
| `gclb` | Kelola Google Cloud Build |
| `gcca` | Kelola alamat IP Compute Engine |
| `gccc` | Buat instance mesin virtual baru |
| `gcco` | Hubungkan ke mesin virtual via SSH |
| `gcd` | Atur proyek default ke nama direktori saat ini |
| `gcf` | Kelola Google Cloud Functions |
| `gci` | Kelola instance Compute Engine |
| `gck` | Daftar semua konfigurasi |
| `gcn` | Kelola Google Cloud Networks |
| `gcp` | Kelola proyek Google Cloud |
| `gcpd` | Hapus proyek Google Cloud |
| `gcps` | Kelola Google Cloud Pub/Sub |
| `gcs` | Kelola cluster Kubernetes Engine |
| `gcsa` | Atur akun untuk konfigurasi saat ini |
| `gcso` | Buka Cloud Console untuk proyek saat ini |
| `gcsq` | Kelola Google Cloud SQL |
| `gcss` | Kelola Google Cloud Storage |
| `gcst` | Aktifkan atau nonaktifkan layanan Google Cloud |
