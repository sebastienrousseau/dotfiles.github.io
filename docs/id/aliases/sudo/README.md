# Alias Sudo

Kelola alias Sudo. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `sudo.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk perintah `sudo`.
- `root` Jalankan perintah sebagai superuser.
- `s` Jalankan perintah sebagai superuser.
- `su` Jalankan perintah sebagai superuser.
- `sudo` Jalankan perintah sebagai superuser.
