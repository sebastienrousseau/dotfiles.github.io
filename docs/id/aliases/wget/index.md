# Alias Wget

Kelola alias Wget. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `wget.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk perintah `wget`.
- `wg` wget.
- `wgc` wget dengan continue.
- `wge` wget dengan robots=off.
- `wget` wget dengan continue.
