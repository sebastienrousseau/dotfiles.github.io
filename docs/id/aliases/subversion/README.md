# Alias Subversion

Kelola alias Subversion. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `subversion.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk command line Subversion.
- `sad` Letakkan file dan direktori baru di bawah version control.
- `sau` Kelola kredensial yang di-cache.
- `sbl` Tampilkan kapan setiap baris file terakhir (atau selanjutnya) diubah.
- `sci` Kirim perubahan dari working copy ke repositori.
- `scl` Pulihkan dari operasi yang terganggu atau hapus file yang tidak diperlukan.
- `sco` Checkout working copy dari repositori.
- `scp` Salin file dan direktori di working copy atau repositori.
- `sdi` Tampilkan perubahan lokal atau perbedaan antara dua revisi atau path.
- `sdl` Hapus file dan direktori dari version control.
- `sin` Tampilkan informasi tentang item lokal atau remote.
- `slg` Tampilkan pesan log untuk set revisi atau path.
- `sls` List entri direktori di repositori.
- `smg` Merge perubahan ke working copy.
- `smv` Pindahkan (rename) item di working copy atau repositori.
- `sst` Cetak status file dan direktori working copy.
- `sup` Ambil perubahan dari repositori ke working copy.
