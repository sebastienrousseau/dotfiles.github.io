# Alias Go

Kelola alias bahasa Go. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `go.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

- `go` - Pintasan Go
- `gor` - `go run`
- `gob` - `go build`
- `got` - `go test`
- `gomt` - `go mod tidy`
- `goget` - `go get`
