# Alias PNPM

Kelola alias PNPM. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `pnpm.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk package manager [pnpm](https://pnpm.js.org/).
- `pni` Inisialisasi proyek baru.
- `pnin` Install dependensi proyek.
- `pnb` Build proyek.
- `pnrun` Jalankan skrip dari proyek.
- `pnd` Jalankan proyek dalam mode development.
- `pns` Jalankan proyek dalam mode service.
- `pnst` Start proyek.
- `pnln` Lint proyek.
- `pnls` List semua dependensi.
- `pnout` Periksa dependensi yang outdated.
- `pnt` Test proyek.
- `pntc` Test proyek dengan coverage.
- `pndoc` Generate dokumentasi proyek.
- `pnpub` Publish proyek.
- `pnwhy` Periksa mengapa dependensi terinstal.
- `pnui` Update dependensi secara interaktif.
- `pnup` Update dependensi.
- `pnad` Tambah dependensi development ke proyek.
- `pnap` Tambah dependensi peer ke proyek.
- `pnrm` Remove dependensi dari proyek.
- `pnun` Uninstall dependensi proyek.
- `pnga` Tambah dependensi global.
- `pngls` List semua dependensi global.
- `pngrm` Remove dependensi global.
- `pngu` Update dependensi global.
- `pnh` Tampilkan bantuan.
- `pnc` Buat proyek baru.
- `pnsv` Jalankan proyek dalam mode server.
- `pnx` Pintasan pnpx.
