# Alias Disk Usage

Pintasan untuk menampilkan penggunaan disk dalam format yang mudah dibaca.

## Ikhtisar

Alias ini didefinisikan di `disk-usage.aliases.sh` dan dimuat secara otomatis oleh Chezmoi.

## Referensi

| Alias | Deskripsi |
|:---|:---|
| `du` | Tampilkan penggunaan disk direktori saat ini |
| `du1` | Tampilkan ukuran file dan direktori di direktori saat ini |
| `duh` | Tampilkan ukuran file dan direktori |
| `ducks` | Tampilkan 10 file dan direktori terbesar |
| `dus` | Tampilkan dalam format mudah dibaca diurutkan berdasarkan ukuran |
| `dusym` | Tampilkan ukuran termasuk symlink |
| `dut` | Tampilkan ukuran total direktori saat ini |
