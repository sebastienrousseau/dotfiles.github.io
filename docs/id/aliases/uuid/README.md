# Alias UUID

Kelola alias UUID. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `uuid.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk perintah `uuidgen`.
- `uuid` Generate UUID dan salin ke clipboard.
