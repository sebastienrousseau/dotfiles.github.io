# Alias PS

Kelola alias PS. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `ps.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk [ps](https://www.gnu.org/software/ps/).
- `pid` Tampilkan uid, pid, parent pid, penggunaan CPU terbaru, waktu mulai proses, controlling tty, penggunaan CPU yang telah berlalu, dan perintah terkait.
- `ps` Tampilkan semua proses.
- `psa` List semua proses.
