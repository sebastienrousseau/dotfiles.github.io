# Alias NPM

Kelola alias NPM. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `npm.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk package manager [npm](https://www.npmjs.com/).
- `npa` Audit paket npm.
- `npb` Build skrip npm.
- `npc` Cache paket npm.
- `npd` Develop skrip npm.
- `npg` Paket npm global.
- `npi` Install paket npm.
- `npl` List paket npm.
- `npp` Publish paket npm.
- `nprm` Remove paket npm.
- `npr` Run skrip npm.
- `nprw` Run skrip npm dalam mode watch.
- `nps` Start skrip npm.
- `npsv` Serve skrip npm.
- `npt` Test skrip npm.
- `npu` Update paket npm.
- `npx` Execute paket npm.
- `npy` Mengapa paket npm terinstal.
