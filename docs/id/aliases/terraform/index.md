# Alias Terraform

Kelola alias Infrastructure as Code (Terraform, OpenTofu, Ansible). Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `terraform.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

### Terraform
- `tf` - Pintasan Terraform
- `tfi` - Init
- `tfp` - Plan
- `tfa` - Apply
- `tfaa` - Apply (auto-approve)
- `tfd` - Destroy
- `tff` - Format
- `tfv` - Validate

### Ansible
- `ans` - Pintasan Ansible
- `ansp` - Ansible Playbook
- `anslint` - Ansible Lint
