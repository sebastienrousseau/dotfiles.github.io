# Alias Rsync

Kelola alias Rsync. Bagian dari konfigurasi **Universal Dotfiles**.

## Ikhtisar

Alias ini didefinisikan di `rsync.aliases.sh` dan dimuat secara otomatis oleh `chezmoi`.

## Alias

Kode ini menyediakan set alias untuk perintah `rsync`.
- `rs` Rsync dengan verbose dan progress.
