# Alias Installer

Alat untuk bootstrap dan deployment remote.

## Perintah

| Alias | Deskripsi |
|-------|-------------|
| `dot-install` | Jalankan skrip bootstrap `install.sh` secara lokal. |
| `dot-teleport` | Teleport konfigurasi ke server remote (`dot-teleport user@host`). |
