# Alias Clear

Percepat alur kerja terminal dengan pintasan pembersihan layar cepat.

## Ikhtisar

Alias clear menyediakan cara efisien untuk mereset tampilan terminal sambil opsional menampilkan konteks berguna seperti isi direktori, riwayat perintah, atau tree file. Alias ini didefinisikan di `clear.aliases.sh` dan dimuat secara otomatis oleh chezmoi.

## Referensi

| Alias | Deskripsi |
|-------|-------------|
| `c` | Bersihkan layar terminal |
| `cdw` | Pindah ke direktori workspace |
| `cl` | Bersihkan layar terminal (alias dari `c`) |
| `clc` | Bersihkan layar dan tampilkan isi direktori |
| `clh` | Bersihkan layar dan tampilkan riwayat perintah |
| `clp` | Bersihkan layar dan tampilkan direktori kerja |
| `clt` | Bersihkan layar dan tampilkan tree direktori |
| `cpl` | Bersihkan layar dan tampilkan direktori kerja beserta isinya |
| `cplt` | Bersihkan layar dan tampilkan direktori kerja beserta tree |
