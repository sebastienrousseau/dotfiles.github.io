# Alias Find

Pintasan untuk pencarian file dan direktori dengan cepat.

## Ikhtisar

Alias ini didefinisikan di `find.aliases.sh` dan dimuat secara otomatis oleh Chezmoi. Menggunakan utilitas `fd` (alternatif find yang lebih cepat).

## Referensi

| Alias | Deskripsi |
|:---|:---|
| `fd` | Daftar semua file dengan output berwarna |
| `fda` | Daftar semua file dengan path absolut |
| `fdc` | Pencarian tidak sensitif huruf besar/kecil |
| `fdd` | Daftar semua file dengan detail |
| `fde` | Daftar file dengan ekstensi tertentu |
| `fdf` | Daftar dengan mengikuti symlink |
| `fdh` | Tampilkan bantuan fd |
| `fdn` | Daftar file yang cocok dengan pola glob |
| `fdo` | Daftar dengan informasi pemilik |
| `fds` | Daftar dengan ukuran |
| `fdu` | Daftar dengan aturan pengecualian |
| `fdv` | Tampilkan versi fd |
| `fdx` | Jalankan perintah untuk setiap hasil pencarian |
