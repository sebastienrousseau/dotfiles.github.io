# Clear-aliassen

Stroomlijn je terminal-workflow met snelle schermwis-snelkoppelingen.

## Overzicht

De clear-aliassen bieden efficiente manieren om je terminalweergave te resetten terwijl optioneel nuttige context wordt getoond zoals mapinhoud, commandogeschiedenis of bestandsbomen. Deze aliassen zijn gedefinieerd in `clear.aliases.sh` en worden automatisch geladen door chezmoi.

## Referentie

| Alias | Beschrijving |
|-------|-------------|
| `c` | Wis het terminalscherm |
| `cdw` | Ga naar de werkruimtemap |
| `cl` | Wis het terminalscherm (alias voor `c`) |
| `clc` | Wis scherm en toon mapinhoud |
| `clh` | Wis scherm en toon commandogeschiedenis |
| `clp` | Wis scherm en toon werkmap |
| `clt` | Wis scherm en toon mapboom |
| `cpl` | Wis scherm en toon werkmap met inhoud |
| `cplt` | Wis scherm en toon werkmap met boom |
