# Mkdir-aliassen

Stroomlijn het aanmaken van mappen met intelligente naamgevingsconventies.

![Dotfiles banner][banner]

## Overzicht

De mkdir-aliassen bieden snelkoppelingen voor het aanmaken van mappen met automatische datum- en tijdstempels. Deze aliassen zijn gedefinieerd in `mkdir.aliases.sh` en worden automatisch geladen door chezmoi.

## Referentie

| Alias | Beschrijving |
|-------|-------------|
| `md` | Maak map |
| `mdd` | Maak map met datum |
| `mdt` | Maak map met tijd |
| `mde` | Maak voorbeeldmap met huidige datum |
| `mdn` | Maak notitiesmap met huidige datum |
| `mdw` | Maak werkmap met huidige datum |
