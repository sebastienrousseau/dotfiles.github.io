# Terraform-aliassen

Snelkoppelingen voor Infrastructure as Code met Terraform en Ansible.

## Overzicht

Deze aliassen zijn gedefinieerd in `terraform.aliases.sh` en worden automatisch geladen door Chezmoi.

## Referentie

### Terraform

| Alias | Beschrijving |
|:---|:---|
| `tf` | Terraform-snelkoppeling |
| `tfi` | Initialiseer configuratie |
| `tfp` | Plan wijzigingen |
| `tfa` | Pas wijzigingen toe |
| `tfaa` | Pas toe met auto-approve |
| `tfd` | Vernietig infrastructuur |
| `tff` | Formatteer configuratie |
| `tfv` | Valideer configuratie |

### Ansible

| Alias | Beschrijving |
|:---|:---|
| `ans` | Ansible-snelkoppeling |
| `ansp` | Voer Ansible Playbook uit |
| `anslint` | Lint Ansible-bestanden |
