# Tmux-aliassen

Beheer terminalsessies, vensters en panelen met tmux.

## Overzicht

Deze aliassen zijn gedefinieerd in `tmux.aliases.sh` en worden automatisch geladen door Chezmoi.

## Referentie

| Alias | Beschrijving |
|:---|:---|
| `tm` | Start tmux |
| `tma` | Koppel aan laatste sessie |
| `tmat` | Koppel aan specifieke sessie |
| `tmks` | Sluit alle sessies behalve huidige |
| `tmka` | Sluit alle sessies |
| `tml` | Toon alle sessies |
| `tmn` | Nieuwe naamloze sessie |
| `tms` | Nieuwe benoemde sessie |
| `tmr` | Herlaad tmux-configuratie |
| `tmls` | Toon vensters |
| `tmlp` | Toon panelen |
| `tmi` | Toon tmux-info |
