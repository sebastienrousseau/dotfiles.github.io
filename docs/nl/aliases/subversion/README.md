# Subversion-aliassen

Snelkoppelingen voor Subversion versiebeheer-operaties.

## Overzicht

Deze aliassen zijn gedefinieerd in `subversion.aliases.sh` en worden automatisch geladen door Chezmoi.

## Referentie

| Alias | Beschrijving |
|:---|:---|
| `sad` | Voeg bestanden en mappen toe aan versiebeheer |
| `sau` | Beheer gecachte authenticatiegegevens |
| `sbl` | Toon wanneer elke regel het laatst is gewijzigd |
| `scg` | Associeer changelist met bestanden |
| `sci` | Commit wijzigingen naar repository |
| `scl` | Ruim werkkopie op |
| `sco` | Check out een werkkopie |
| `scp` | Kopieer bestanden en mappen |
| `sct` | Geef inhoud van bestanden of URL's weer |
| `sdi` | Toon lokale wijzigingen of verschillen |
| `sdl` | Verwijder bestanden van versiebeheer |
| `shp` | Toon help voor commando's |
| `sin` | Toon informatie over een item |
| `sip` | Importeer ongeversioned bestand naar repository |
| `slg` | Toon logberichten |
| `slk` | Vergrendel werkkopie-paden |
| `sls` | Toon map-entries |
| `smd` | Maak nieuwe map onder versiebeheer |
| `smg` | Merge wijzigingen naar werkkopie |
| `smgi` | Toon merge-gerelateerde informatie |
| `smv` | Verplaats of hernoem een item |
| `sst` | Print status van werkkopie |
| `ssw` | Wissel werkkopie naar andere URL |
| `sulk` | Ontgrendel werkkopie-paden |
| `sup` | Werk werkkopie bij van repository |
| `supg` | Upgrade metadata-opslagformaat |
| `sxp` | Maak ongeversioned kopie van boom |
