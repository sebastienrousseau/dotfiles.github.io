# Compliance-aliassen

Shell-tools voor regelgevingsnaleving en privacy-handhaving.

## Overzicht

Compliance-aliassen bieden essentiele controles voor het beheren van privacy en het onderhouden van audittrails in je shell-omgeving. Deze aliassen helpen telemetrie uit te schakelen voor gangbare CLI-tools en configuratiewijzigingen bij te houden voor compliance-vereisten.

## Referentie

### Controles

| Alias | Beschrijving |
|-------|-------------|
| `privacy-mode` | Schakelt telemetrie uit voor CLI-tools (Homebrew, Dotnet, Azure, etc.) voor de huidige sessie. |
| `dot-audit` | Toont de recente configuratie-audittrail (Git-geschiedenis of Auditlogboek). |
