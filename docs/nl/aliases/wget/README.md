# Wget-aliassen

Download bestanden van het web met hervat-ondersteuning.

## Overzicht

Deze aliassen zijn gedefinieerd in `wget.aliases.sh` en worden automatisch geladen door Chezmoi.

## Referentie

| Alias | Beschrijving |
|:---|:---|
| `wg` | Wget-snelkoppeling |
| `wgc` | Wget met continue |
| `wge` | Wget met robots=off |
| `wget` | Wget met continue |
