# Yarn-aliassen

Snelkoppelingen voor de Yarn JavaScript-pakketbeheerder.

## Overzicht

Deze aliassen zijn gedefinieerd in `yarn.aliases.sh` en worden automatisch geladen door Chezmoi.

## Referentie

| Alias | Beschrijving |
|:---|:---|
| `y` | Yarn-snelkoppeling |
| `yi` | Installeer afhankelijkheden |
| `ya` | Voeg afhankelijkheid toe |
| `yad` | Voeg dev-afhankelijkheid toe |
| `yrun` | Voer script uit |
| `ystem` | Start project |
