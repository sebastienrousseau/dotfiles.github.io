# PS-aliassen

Monitor en beheer draaiende processen met eenvoudige snelkoppelingen.

## Overzicht

Deze aliassen zijn gedefinieerd in `ps.aliases.sh` en worden automatisch geladen door Chezmoi.

## Referentie

| Alias | Beschrijving |
|:---|:---|
| `pid` | Toon UID, PID, parent PID, CPU-gebruik, starttijd, TTY en commando |
| `ps` | Toon alle processen |
| `psa` | Toon alle processen |
