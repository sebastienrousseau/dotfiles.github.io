# Python-aliassen

Snelkoppelingen voor Python-ontwikkeling. Interpreter, pakketten en virtuele omgevingen.

## Overzicht

Deze aliassen zijn gedefinieerd in `python.aliases.sh` en worden automatisch geladen door Chezmoi.

## Referentie

| Alias | Commando | Beschrijving |
|:---|:---|:---|
| `py` | `python` | Python-interpreter |
| `pip` | `pip3` | Pakket-installeerder |
| `v` | `virtualenv` | Virtuele omgeving |
| `uv` | `uv` | Snelle pakketbeheerder |
| `uvpi` | `uv pip install` | Installeren met uv |
