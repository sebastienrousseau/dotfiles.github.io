# Update-aliassen

Houd je systeem en pakketten up-to-date.

## Overzicht

Deze aliassen zijn gedefinieerd in `update.aliases.sh` en worden automatisch geladen door Chezmoi.

## Referentie

### macOS

| Alias | Beschrijving |
|:---|:---|
| `upd` | Werk macOS-software, Homebrew, Casks, App Store, Rust, Ruby, Python en Node.js bij |

### Linux

| Alias | Beschrijving |
|:---|:---|
| `open` | Open bestand of URL in voorkeur-applicatie |
| `pbcopy` | Kopieer naar klembord |
| `pbpaste` | Plak van klembord |
| `upd` | Werk Linux-pakketten, pnpm, Rust, Ruby, Python en Node.js bij |
