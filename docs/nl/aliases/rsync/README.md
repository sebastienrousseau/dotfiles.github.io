# Rsync-aliassen

Synchroniseer bestanden en mappen tussen locaties met voortgangsweergave.

## Overzicht

Deze aliassen zijn gedefinieerd in `rsync.aliases.sh` en worden automatisch geladen door Chezmoi.

|:---|:---|
| `rs` | Rsync met uitgebreide uitvoer en voortgangsweergave |
