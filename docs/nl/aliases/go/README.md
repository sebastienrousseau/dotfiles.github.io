# Go-aliassen

Stroomlijn je Go-ontwikkelworkflow met intuitive shell-snelkoppelingen.

![Dotfiles banner](https://kura.pro/dotfiles/v2/images/titles/title-dotfiles.svg "Dotfiles banner")

## Overzicht

Deze aliassen zijn gedefinieerd in `go.aliases.sh` en worden automatisch geladen door chezmoi. Ze bieden snelle toegang tot gangbare Go-commando's voor uitvoeren, bouwen, testen en beheren van modules.

## Referentie

| Alias    | Commando       | Beschrijving                        |
|----------|---------------|------------------------------------|
| `go`     | `go`          | Go-snelkoppeling                   |
| `gor`    | `go run`      | Voer Go-programma uit              |
| `gob`    | `go build`    | Bouw Go-pakket                     |
| `got`    | `go test`     | Voer Go-tests uit                  |
| `gomt`   | `go mod tidy` | Ruim Go-module afhankelijkheden op |
| `goget`  | `go get`      | Download en installeer Go-pakketten |
