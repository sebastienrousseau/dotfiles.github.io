# Installer-aliassen

Tools voor bootstrappen en remote deployment.

## Overzicht

De installer-aliassen bieden gestroomlijnde commando's voor het opzetten van je dotfiles-omgeving lokaal en het deployen van je configuratie naar remote servers.

## Referentie

| Alias | Beschrijving |
|-------|-------------|
| `dot-install` | Voer het `install.sh` bootstrap-script lokaal uit. |
| `dot-teleport` | Teleporteer je config naar een remote server (`dot-teleport user@host`). |
