# Alias Compliance

Outils shell pour la conformité réglementaire et la protection de la confidentialité.

## Aperçu

Les alias compliance offrent des contrôles essentiels pour la confidentialité et le suivi d’audit dans votre environnement shell. Ils aident à désactiver la télémétrie de CLI courantes et à suivre les changements de configuration pour les exigences de conformité.

## Référence

### Contrôles

| Alias | Description |
|-------|-------------|
| `privacy-mode` | Désactive la télémétrie des outils CLI (Homebrew, Dotnet, Azure, etc.) pour la session en cours. |
| `dot-audit` | Affiche l’historique d’audit de configuration (Git ou journal d’audit). |
