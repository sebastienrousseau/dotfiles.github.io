# Alias Terraform

Raccourcis pour l’Infrastructure as Code avec Terraform et Ansible.

## Aperçu

Ces alias sont définis dans `terraform.aliases.sh` et chargés automatiquement par Chezmoi.

## Référence

### Terraform

| Alias | Description |
|:---|:---|
| `tf` | Raccourci Terraform |
| `tfi` | Initialiser la configuration |
| `tfp` | Planifier les changements |
| `tfa` | Appliquer les changements |
| `tfaa` | Appliquer avec auto-approve |
| `tfd` | Détruire l’infrastructure |
| `tff` | Formater la configuration |
| `tfv` | Valider la configuration |

### Ansible

| Alias | Description |
|:---|:---|
| `ans` | Raccourci Ansible |
| `ansp` | Exécuter un playbook Ansible |
| `anslint` | Linter des fichiers Ansible |
