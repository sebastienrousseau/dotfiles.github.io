# Alias Yarn

Raccourcis pour le gestionnaire de paquets JavaScript Yarn.

## Aperçu

Ces alias sont définis dans `yarn.aliases.sh` et chargés automatiquement par Chezmoi.

## Référence

| Alias | Description |
|:---|:---|
| `y` | Raccourci Yarn |
| `yi` | Installer les dépendances |
| `ya` | Ajouter une dépendance |
| `yad` | Ajouter une dépendance de dev |
| `yrun` | Exécuter un script |
| `ystem` | Démarrer le projet |
