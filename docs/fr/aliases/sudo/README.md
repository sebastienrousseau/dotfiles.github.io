# Alias Sudo

Exécutez des commandes avec des privilèges élevés.

## Aperçu

Ces alias sont définis dans `sudo.aliases.sh` et chargés automatiquement par Chezmoi.

## Référence

| Alias | Description |
|:---|:---|
| `root` | Exécuter une commande en superutilisateur |
| `s` | Exécuter une commande en superutilisateur |
| `su` | Exécuter une commande en superutilisateur |
| `sudo` | Exécuter une commande en superutilisateur |
