# Alias UUID

Générez des identifiants universellement uniques.

## Aperçu

Ces alias sont définis dans `uuid.aliases.sh` et chargés automatiquement par Chezmoi.

## Référence

| Alias | Description |
|:---|:---|
| `uuid` | Générer un UUID et le copier dans le presse-papiers |
