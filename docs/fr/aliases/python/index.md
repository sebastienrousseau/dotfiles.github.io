# Alias Python

Raccourcis pour le développement Python. Interpréteur, paquets et environnements virtuels.

## Aperçu

Ces alias sont définis dans `python.aliases.sh` et chargés automatiquement par Chezmoi.

## Référence

| Alias | Commande | Description |
|:---|:---|:---|
| `py` | `python` | Interpréteur Python |
| `pip` | `pip3` | Installeur de paquets |
| `v` | `virtualenv` | Environnement virtuel |
| `uv` | `uv` | Gestionnaire de paquets rapide |
| `uvpi` | `uv pip install` | Installer avec uv |
