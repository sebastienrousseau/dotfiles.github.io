# Alias Mkdir

Simplifiez la création de répertoires avec des conventions de nommage intelligentes.

![Dotfiles banner][banner]

## Aperçu

Les alias mkdir offrent des raccourcis pour créer des répertoires avec des horodatages automatiques. Ces alias sont définis dans `mkdir.aliases.sh` et chargés automatiquement par chezmoi.

## Référence

| Alias | Description |
|-------|-------------|
| `md` | Créer un répertoire |
| `mdd` | Créer un répertoire avec la date |
| `mdt` | Créer un répertoire avec l’heure |
| `mde` | Créer un répertoire d’exemple avec la date du jour |
| `mdn` | Créer un répertoire de notes avec la date du jour |
| `mdw` | Créer un répertoire de travail avec la date du jour |
