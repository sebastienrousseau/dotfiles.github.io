# Alias Rsync

Synchronisez fichiers et répertoires entre emplacements avec suivi de progression.

## Aperçu

Ces alias sont définis dans `rsync.aliases.sh` et chargés automatiquement par Chezmoi.

| Alias | Description |
|:---|:---|
| `rs` | Rsync avec sortie verbeuse et progression |
