# Alias Wget

Téléchargez des fichiers depuis le web avec reprise.

## Aperçu

Ces alias sont définis dans `wget.aliases.sh` et chargés automatiquement par Chezmoi.

## Référence

| Alias | Description |
|:---|:---|
| `wg` | Raccourci wget |
| `wgc` | Wget avec reprise |
| `wge` | Wget avec robots=off |
| `wget` | Wget avec reprise |
