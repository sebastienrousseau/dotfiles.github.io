# Alias Subversion

Raccourcis pour les opérations Subversion.

## Aperçu

Ces alias sont définis dans `subversion.aliases.sh` et chargés automatiquement par Chezmoi.

## Référence

| Alias | Description |
|:---|:---|
| `sad` | Ajouter fichiers et répertoires au contrôle de versions |
| `sau` | Gérer les identifiants d’authentification en cache |
| `sbl` | Afficher la dernière modification de chaque ligne |
| `scg` | Associer une changelist aux fichiers |
| `sci` | Commit des changements |
| `scl` | Nettoyer la copie de travail |
| `sco` | Checkout d’une copie de travail |
| `scp` | Copier des fichiers et répertoires |
| `sct` | Afficher le contenu de fichiers ou URLs |
| `sdi` | Afficher les changements locaux |
| `sdl` | Supprimer des fichiers du contrôle de versions |
| `shp` | Afficher l’aide des commandes |
| `sin` | Afficher les infos d’un élément |
| `sip` | Importer un fichier non versionné |
| `slg` | Afficher les logs |
| `slk` | Verrouiller des chemins |
| `sls` | Lister les entrées d’un répertoire |
| `smd` | Créer un nouveau répertoire versionné |
| `smg` | Fusionner des changements |
| `smgi` | Afficher les infos de merge |
| `smv` | Déplacer ou renommer un élément |
| `sp` | Définir une propriété |
| `spdl` | Supprimer une propriété |
| `spdt` | Éditer une propriété avec un éditeur externe |
| `spgt` | Afficher la valeur d’une propriété |
| `sph` | Appliquer un patch |
| `spls` | Lister toutes les propriétés |
| `srl` | Relocaliser la copie de travail vers une autre URL |
| `srs` | Résoudre des conflits |
| `srsd` | Retirer l’état conflictuels |
| `srv` | Restaurer l’état pristine |
| `sst` | Afficher le statut |
| `ssw` | Basculer la copie de travail vers une autre URL |
| `sulk` | Déverrouiller des chemins |
| `sup` | Mettre à jour la copie de travail |
| `supg` | Mettre à jour le format de métadonnées |
| `sxp` | Créer une copie non versionnée de l’arborescence |
