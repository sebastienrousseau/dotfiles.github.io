# Alias Installer

Outils pour le bootstrap et le déploiement distant.

## Aperçu

Les alias installer fournissent des commandes simplifiées pour mettre en place votre environnement dotfiles en local et déployer votre configuration sur des serveurs distants.

## Référence

| Alias | Description |
|-------|-------------|
| `dot-install` | Exécuter le script `install.sh` localement. |
| `dot-teleport` | Téléporter votre config vers un serveur distant (`dot-teleport user@host`). |
