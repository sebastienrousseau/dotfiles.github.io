# Alias Tmux

Gérez vos sessions, fenêtres et panneaux avec tmux.

## Aperçu

Ces alias sont définis dans `tmux.aliases.sh` et chargés automatiquement par Chezmoi.

## Référence

| Alias | Description |
|:---|:---|
| `tm` | Démarrer tmux |
| `tma` | Attacher à la dernière session |
| `tmat` | Attacher à une session spécifique |
| `tmks` | Tuer toutes les sessions sauf la courante |
| `tmka` | Tuer toutes les sessions |
| `tml` | Lister toutes les sessions |
| `tmn` | Nouvelle session sans nom |
| `tms` | Nouvelle session nommée |
| `tmr` | Recharger la configuration tmux |
| `tmls` | Lister les fenêtres |
| `tmlp` | Lister les panneaux |
| `tmi` | Afficher les infos tmux |
