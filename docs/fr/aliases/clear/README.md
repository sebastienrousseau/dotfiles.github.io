# Alias Clear

Simplifiez votre workflow terminal avec des raccourcis pour nettoyer l’écran.

## Aperçu

Les alias clear permettent de réinitialiser l'affichage du terminal tout en affichant, si besoin, le contenu du répertoire, l’historique des commandes ou l’arborescence. Ces alias sont définis dans `clear.aliases.sh` et chargés automatiquement par chezmoi.

## Référence

| Alias | Description |
|-------|-------------|
| `c` | Effacer l’écran du terminal |
| `cdw` | Aller au répertoire de travail |
| `cl` | Effacer l’écran (alias de `c`) |
| `clc` | Effacer l’écran et lister le contenu du répertoire |
| `clh` | Effacer l’écran et afficher l’historique |
| `clp` | Effacer l’écran et afficher le répertoire courant |
| `clt` | Effacer l’écran et afficher l’arborescence |
| `cpl` | Effacer l’écran et afficher le répertoire courant avec contenu |
| `cplt` | Effacer l’écran et afficher le répertoire courant avec arborescence |
