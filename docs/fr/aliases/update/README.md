# Alias Update

Gardez votre système et vos paquets à jour.

## Aperçu

Ces alias sont définis dans `update.aliases.sh` et chargés automatiquement par Chezmoi.

## Référence

### macOS

| Alias | Description |
|:---|:---|
| `upd` | Mettre à jour macOS, Homebrew, Casks, App Store, Rust, Ruby, Python et Node.js |

### Linux

| Alias | Description |
|:---|:---|
| `open` | Ouvrir un fichier ou une URL dans l’application préférée |
| `pbcopy` | Copier dans le presse-papiers |
| `pbpaste` | Coller depuis le presse-papiers |
| `upd` | Mettre à jour les paquets Linux, pnpm, Rust, Ruby, Python et Node.js |
