# Accessibilité

- La navigation au clavier est prise en charge.
- Les indicateurs de focus sont visibles.
- Les préférences de mouvement réduit sont respectées.

Signaler des problèmes d'accessibilité: [info@dotfiles.io](mailto:info@dotfiles.io).
