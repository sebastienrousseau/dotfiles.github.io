# Aliasuri Rust

Comenzi scurte pentru Rust.

Această pagină prezintă un grup de aliasuri și scurtăturile sale de comandă obișnuite.
Citiți-o mai întâi pentru a înțelege convențiile de denumire, intrările așteptate și tiparele de execuție sigure.
Apoi deschideți anexa de referință pentru inventarul complet de comenzi și sintaxa exactă.
Aceste aliasuri reduc tastarea repetitivă și mențin un comportament consistent pe macOS, Linux și WSL.

## Când se utilizează

Utilizați această pagină înainte de lucrul zilnic în shell când aveți nevoie de tipare de comandă previzibile.

## Ce este inclus

- Aliasuri: 75
- Funcții helper: 0
- Fișiere sursă: 1

## Exemple rapide

- `cg` → `'cargo'          # Scurtătură Cargo`
- `cga` → `'cg add'        # Adaugă dependență`
- `cgad` → `'cg add --dev' # Adaugă dependență de dezvoltare`
- `cgaud` → `'cg audit'    # Verificare vulnerabilități de securitate`
- `cgb` → `'cg build'            # Compilare debug`
- `cgba` → `'cg build --all-targets'       # Compilare toate țintele`
- `cgbh` → `'cg bench'                    # Rulează benchmark-uri`
- `cgbl` → `'cg bloat'      # Analiză dimensiune binar`
- `cgbp` → `'cg build --release --profile' # Compilare cu profil specific`
- `cgbr` → `'cg build --release' # Compilare release`
- `cgbt` → `'cg build --all-features'      # Compilare cu toate funcționalitățile`
- `cgc` → `'cg check'      # Verificare compilare`
