# Dostępność

- Nawigacja klawiaturą jest obsługiwana.
- Wskaźniki fokusa są widoczne.
- Preferencje ograniczenia ruchu są respektowane.

Zgłoś problemy z dostępnością: [info@dotfiles.io](mailto:info@dotfiles.io).
