# Alias macOS

Scorciatoie shell per gestione sistema macOS e manutenzione.

## Panoramica

Questi alias sono definiti in `macOS.aliases.sh` e caricati automaticamente da chezmoi. Forniscono accesso rapido alle operazioni macOS comuni incluso controllo Finder, manutenzione sistema e strumenti sviluppatore.

## Riferimento

### Gestione sistema

| Alias | Descrizione |
|-------|-------------|
| `lockScreen` | Blocca lo schermo |
| `wifiOn` | Attiva Wi-Fi |
| `wifiOff` | Disattiva Wi-Fi |
| `vp` | Verifica permessi macOS |
| `vv` | Verifica volume macOS |

### Finder

| Alias | Descrizione |
|-------|-------------|
| `ofd` | Apri directory corrente nel Finder |
| `finderShowHidden` | Mostra file nascosti nel Finder |
| `finderHideHidden` | Nascondi file nascosti nel Finder |
| `showDesktopIcons` | Mostra tutte le icone sul Desktop |
| `hideDesktopIcons` | Nascondi tutte le icone sul Desktop |

### Pulizia

| Alias | Descrizione |
|-------|-------------|
| `clds` | Elimina ricorsivamente file .DS_Store |
| `clls` | Pulisci LaunchServices per rimuovere duplicati nel menu Apri con |
| `trash` | Svuota il Cestino su tutti i volumi montati e l'HDD principale |

### Strumenti sviluppatore

| Alias | Descrizione |
|-------|-------------|
| `xcode` | Avvia Xcode |
| `purge` | Elimina Xcode DerivedData |
| `iphone` | Apri simulatori dispositivo |
| `safariSafeMode` | Avvia Safari in Modalita Sicura |

### Display

| Alias | Descrizione |
|-------|-------------|
| `screensaverDesktop` | Esegui screensaver sul Desktop |
