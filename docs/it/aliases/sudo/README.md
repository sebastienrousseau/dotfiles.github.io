# Alias Sudo

Scorciatoie per eseguire comandi con privilegi elevati.

## Panoramica

Questi alias sono definiti in `sudo.aliases.sh` e caricati automaticamente da chezmoi. Forniscono comandi brevi per le operazioni superuser comuni.

## Riferimento

| Alias | Descrizione |
|-------|-------------|
| `s` | Esegui comando come superuser |
| `su` | Esegui comando come superuser |
| `sudo` | Esegui comando come superuser |
| `root` | Esegui comando come superuser |
