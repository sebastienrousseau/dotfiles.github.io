# Alias Go

Scorciatoie per sviluppo linguaggio Go.

## Panoramica

Questi alias sono definiti in `go.aliases.sh` e caricati automaticamente da Chezmoi. Forniscono comandi brevi per le operazioni comuni di sviluppo Go.

## Riferimento

| Alias | Comando | Descrizione |
|-------|---------|-------------|
| `go` | `go` | Scorciatoia Go |
| `gor` | `go run` | Esegui programma Go |
| `gob` | `go build` | Compila pacchetti e dipendenze |
| `got` | `go test` | Esegui test |
| `gomt` | `go mod tidy` | Pulisci dipendenze modulo |
| `goget` | `go get` | Scarica e installa pacchetti |
