# Alias NPM

Scorciatoie per il gestore pacchetti npm.

## Panoramica

Questi alias sono definiti in `npm.aliases.sh` e caricati automaticamente da chezmoi. Forniscono comandi brevi per le operazioni npm di uso comune.

## Riferimento

| Alias | Descrizione |
|-------|-------------|
| `npi` | Installa pacchetto npm |
| `npa` | Controlla pacchetti npm |
| `npb` | Esegui script build npm |
| `npc` | Cache pacchetto npm |
| `npd` | Script sviluppo npm |
| `npg` | Pacchetto globale npm |
| `npl` | Elenca pacchetti npm |
| `npp` | Pubblica pacchetto npm |
| `npr` | Esegui script npm |
| `nprm` | Rimuovi pacchetto npm |
| `nprw` | Esegui npm in modalita watch |
| `nps` | Avvia script npm |
| `npsv` | Servi script npm |
| `npt` | Testa script npm |
| `npu` | Aggiorna pacchetto npm |
| `npx` | Esegui pacchetto npm |
| `npy` | Perche pacchetto npm |
