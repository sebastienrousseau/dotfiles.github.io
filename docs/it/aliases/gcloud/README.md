# Alias GCloud

Ottimizza le operazioni Google Cloud Platform con alias shell memorizzabili.

## Panoramica

Questi alias forniscono scorciatoie per i comandi Google Cloud SDK di uso comune. Definiti in `gcloud.aliases.sh`, vengono caricati automaticamente da chezmoi e semplificano la gestione di risorse cloud come VM, storage, IAM e Kubernetes.

## Riferimento

### Compute Engine

| Alias | Descrizione |
|-------|-------------|
| `gci` | Gestisci istanze Compute Engine |
| `gcca` | Gestisci indirizzi IP Compute Engine |
| `gccc` | Crea una nuova istanza VM |
| `gcco` | Connettiti a una VM via SSH |
| `gcpha` | Visualizza dettagli indirizzo IP |

### Progetti e configurazione

| Alias | Descrizione |
|-------|-------------|
| `gcd` | Imposta progetto predefinito al nome della directory corrente |
| `gcp` | Gestisci progetti Google Cloud |
| `gcpd` | Elimina un progetto Google Cloud |
| `gck` | Elenca tutte le configurazioni |
| `gcsa` | Imposta account per la configurazione corrente |
| `gcso` | Apri Google Cloud Console per il progetto corrente |

### Servizi

| Alias | Descrizione |
|-------|-------------|
| `gclb` | Gestisci Google Cloud Build |
| `gcdb` | Gestisci Google Cloud Datastore |
| `gcdp` | Gestisci Google Cloud Dataproc |
| `gce` | Gestisci Google Cloud Endpoints |
| `gcf` | Gestisci Google Cloud Functions |
| `gcro` | Gestisci Google Cloud Run |
| `gcs` | Gestisci cluster GKE |
| `gcsq` | Gestisci Google Cloud SQL |
| `gcss` | Gestisci Google Cloud Storage |
| `gcst` | Abilita o disabilita servizi Google Cloud |
| `gct` | Gestisci Google Cloud Tasks |
| `gcu` | Gestisci Google Cloud App Engine |
