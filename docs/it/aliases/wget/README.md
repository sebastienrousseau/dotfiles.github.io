# Alias Wget

Scorciatoie per download file da linea di comando.

## Panoramica

Questi alias sono definiti in `wget.aliases.sh` e caricati automaticamente da chezmoi. Forniscono comandi ottimizzati per wget.

## Riferimento

| Alias | Descrizione |
|-------|-------------|
| `wg` | wget |
| `wgc` | wget con continua (ripresa download) |
| `wge` | wget con robots=off |
| `wget` | wget con continua |
