# Alias Python

Scorciatoie per sviluppo Python e gestione ambienti virtuali.

## Panoramica

Questi alias sono definiti in `python.aliases.sh` e caricati automaticamente da chezmoi. Forniscono comandi brevi per le operazioni Python comuni.

## Riferimento

| Alias | Descrizione |
|-------|-------------|
| `py` | Esegui python |
| `pip` | pip3 |
| `v` | virtualenv |
| `uv` | Gestore pacchetti uv |
| `uvpi` | uv pip install |
