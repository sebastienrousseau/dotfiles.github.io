# Alias Compliance

Strumenti shell per conformita normativa e applicazione privacy.

## Panoramica

Gli alias compliance forniscono controlli essenziali per gestire la privacy e mantenere tracce di audit nel tuo ambiente shell. Questi alias aiutano a disabilitare la telemetria nei comuni strumenti CLI e tracciare le modifiche di configurazione per requisiti di conformita.

## Riferimento

### Controlli

| Alias | Descrizione |
|-------|-------------|
| `privacy-mode` | Disabilita la telemetria per strumenti CLI (Homebrew, Dotnet, Azure, ecc.) per la sessione corrente. |
| `dot-audit` | Visualizza la traccia audit recente della configurazione (cronologia Git o log Audit). |
