# Alias Mkdir

Crea directory con template e convenzioni di naming.

## Panoramica

Questi alias sono definiti in `mkdir.aliases.sh` e caricati automaticamente da chezmoi. Semplificano la creazione di directory con timestamp e prefissi di naming.

## Riferimento

| Alias | Descrizione |
|-------|-------------|
| `md` | Crea directory |
| `mdd` | Crea directory con data |
| `mde` | Crea directory di esempio con data corrente |
| `mdn` | Crea directory per note con data corrente |
| `mdt` | Crea directory con ora |
| `mdw` | Crea directory di lavoro con data corrente |
