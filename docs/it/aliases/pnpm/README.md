# Alias PNPM

Scorciatoie per pnpm, il gestore pacchetti veloce ed efficiente.

## Panoramica

Questi alias sono definiti in `pnpm.aliases.sh` e caricati automaticamente da chezmoi. Forniscono comandi brevi per le operazioni pnpm comuni.

## Riferimento

### Gestione progetto

| Alias | Descrizione |
|-------|-------------|
| `pni` | Inizializza nuovo progetto |
| `pnin` | Installa dipendenze progetto |
| `pnb` | Compila progetto |
| `pnrun` | Esegui script dal progetto |
| `pnd` | Esegui in modalita sviluppo |
| `pns` | Esegui in modalita servizio |
| `pnst` | Avvia progetto |

### Test e qualita

| Alias | Descrizione |
|-------|-------------|
| `pnt` | Testa progetto |
| `pntc` | Testa con copertura |
| `pnln` | Lint progetto |
| `pndoc` | Genera documentazione |

### Gestione dipendenze

| Alias | Descrizione |
|-------|-------------|
| `pnad` | Aggiungi dipendenza sviluppo |
| `pnap` | Aggiungi dipendenza peer |
| `pnrm` | Rimuovi dipendenza |
| `pnun` | Disinstalla dipendenze |
| `pnup` | Aggiorna dipendenza |
| `pnui` | Aggiorna interattivamente |
| `pnuil` | Aggiorna all'ultima versione |
| `pnout` | Controlla dipendenze obsolete |
| `pnwhy` | Perche una dipendenza e installata |

### Dipendenze globali

| Alias | Descrizione |
|-------|-------------|
| `pnga` | Aggiungi dipendenza globale |
| `pngls` | Elenca dipendenze globali |
| `pngrm` | Rimuovi dipendenza globale |
| `pngu` | Aggiorna dipendenza globale |

### Altro

| Alias | Descrizione |
|-------|-------------|
| `pnls` | Elenca tutte le dipendenze |
| `pnpub` | Pubblica progetto |
| `pnh` | Mostra aiuto |
| `pnc` | Crea nuovo progetto |
| `pnx` | Scorciatoia pnpx |
