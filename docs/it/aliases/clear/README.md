# Alias Clear

Ottimizza il flusso di lavoro del terminale con scorciatoie rapide per pulire lo schermo.

## Panoramica

Gli alias clear forniscono modi efficienti per reimpostare la visualizzazione del terminale mostrando opzionalmente contesto utile come contenuti directory, cronologia comandi o alberi file. Questi alias sono definiti in `clear.aliases.sh` e caricati automaticamente da chezmoi.

## Riferimento

| Alias | Descrizione |
|-------|-------------|
| `c` | Pulisci lo schermo del terminale |
| `cdw` | Cambia alla directory workspace |
| `cl` | Pulisci lo schermo del terminale (alias per `c`) |
| `clc` | Pulisci schermo ed elenca contenuti directory |
| `clh` | Pulisci schermo e visualizza cronologia comandi |
| `clp` | Pulisci schermo e stampa directory corrente |
| `clt` | Pulisci schermo e visualizza albero directory |
| `cpl` | Pulisci schermo e mostra directory corrente con contenuti |
| `cplt` | Pulisci schermo e mostra directory corrente con albero |
