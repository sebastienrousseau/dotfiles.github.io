# Alias PS

Scorciatoie per visualizzare e gestire i processi di sistema.

## Panoramica

Questi alias sono definiti in `ps.aliases.sh` e caricati automaticamente da chezmoi. Forniscono comandi brevi per le operazioni comuni di stato processi.

## Riferimento

| Alias | Descrizione |
|-------|-------------|
| `ps` | Visualizza tutti i processi |
| `psa` | Elenca tutti i processi |
| `pid` | Visualizza uid, pid, pid padre, uso CPU, tempo avvio, tty e comando |
