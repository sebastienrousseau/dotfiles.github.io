# Alias Update

Un singolo comando per aggiornare tutto il sistema.

## Panoramica

Questi alias sono definiti in `update.aliases.sh` e caricati automaticamente da chezmoi. Forniscono aggiornamenti completi del sistema per macOS e Linux.

## Riferimento

### macOS

| Alias | Descrizione |
|-------|-------------|
| `upd` | Aggiorna tutto: software macOS, Homebrew, Cask, Mac App Store, Rust, Ruby gems, Python e Node.js |

### Linux

| Alias | Descrizione |
|-------|-------------|
| `upd` | Aggiorna tutto: pacchetti sistema, cache, pnpm, Rust, Ruby gems, Python e Node.js |
| `open` | Apri file o URL nell'applicazione preferita |
| `pbcopy` | Copia negli appunti |
| `pbpaste` | Incolla dagli appunti |
