# Alias Yarn

Scorciatoie per il gestore pacchetti Yarn.

## Panoramica

Questi alias sono definiti in `yarn.aliases.sh` e caricati automaticamente da chezmoi. Forniscono comandi brevi per le operazioni Yarn comuni.

## Riferimento

| Alias | Descrizione |
|-------|-------------|
| `y` | Scorciatoia Yarn |
| `yi` | Installa dipendenze |
| `ya` | Aggiungi dipendenza |
| `yad` | Aggiungi dipendenza di sviluppo |
| `yrun` | Esegui script |
| `ystem` | Avvia progetto |
