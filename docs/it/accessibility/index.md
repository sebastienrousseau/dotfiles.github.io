# Accessibilità

- La navigazione da tastiera è supportata.
- Gli indicatori di focus sono visibili.
- Le preferenze di movimento ridotto vengono rispettate.

Segnala problemi di accessibilità: [info@dotfiles.io](mailto:info@dotfiles.io).
