# Yarn Aliase

Schnelle Paketverwaltung mit Yarn fuer Node.js-Projekte.

## Uebersicht

Diese Aliase sind in `yarn.aliases.sh` definiert und werden automatisch von Chezmoi geladen. Sie bieten schnellen Zugriff auf gaengige Yarn-Operationen fuer Abhaengigkeitsverwaltung und Skriptausfuehrung.

## Referenz

| Alias | Befehl | Beschreibung |
|:---|:---|:---|
| `y` | `yarn` | Yarn-Verknuepfung |
| `yi` | `yarn install` | Abhaengigkeiten installieren |
| `ya` | `yarn add` | Abhaengigkeit hinzufuegen |
| `yad` | `yarn add -D` | Dev-Abhaengigkeit hinzufuegen |
| `yrun` | `yarn run` | Skript ausfuehren |
| `ystem` | `yarn start` | Projekt starten |
