# Go Aliase

Optimieren Sie Ihren Go-Entwicklungs-Workflow mit intuitiven Shell-Verknuepfungen.

## Uebersicht

Diese Aliase sind in `go.aliases.sh` definiert und werden automatisch von Chezmoi geladen. Sie bieten schnellen Zugriff auf gaengige Go-Befehle zum Ausfuehren, Bauen, Testen und Verwalten von Modulen.

## Referenz

| Alias | Befehl | Beschreibung |
|:---|:---|:---|
| `go` | `go` | Go-Verknuepfung |
| `gor` | `go run` | Go-Programm ausfuehren |
| `gob` | `go build` | Go-Paket bauen |
| `got` | `go test` | Go-Tests ausfuehren |
| `gomt` | `go mod tidy` | Go-Modul-Abhaengigkeiten aufraeumen |
| `goget` | `go get` | Go-Pakete herunterladen und installieren |
