# Heroku-Aliase

Heroku-Aliase verwalten. Teil der **Universal Dotfiles** Konfiguration.

![Dotfiles banner][banner]

Diese Aliase sind in `heroku.aliases.sh` definiert und werden automatisch von `chezmoi` geladen.

## âš¡ Aliase

Diese Aliase helfen beim Verwalten von Heroku-Apps und zugehoerigen Ressourcen.
- `hk` Allgemeiner Heroku-Kurzbefehl.
- `hkb` Browser fuer aktuelle App oeffnen.
- `hkba` Addons der App anzeigen.
- `hkconf` App-Konfiguration anzeigen.
- `hkcs` Dynos der App skalieren.
- `hkinfo` App-Informationen anzeigen.
- `hkl` Letzte Logs anzeigen.
- `hklf` Logs in Echtzeit folgen.
- `hklt` Logs mit Tail anzeigen.
- `hkop` Heroku-Dashboard oeffnen.
- `hkps` Prozesse auflisten.
- `hkpsa` Apps auflisten.
- `hkpsoff` Alle Dynos stoppen.
- `hkpso` Dynos starten.
- `hkrestart` App neu starten.
- `hkstack` Stack der App anzeigen.
- `hkstackset` Stack setzen.
- `hkt` Tests ausfuehren.
- `hktl` Pipelines auflisten.
- `hktla` Apps in einer Pipeline auflisten.
- `hktop` Prozessnutzung anzeigen.
- `hkreleases` Releases auflisten.
- `hkrl` Aktuelles Release promoten.
- `hkrb` Auf vorheriges Release zurueckrollen.
- `hkrf` Release erzwingen (Rollback bei Fehler).
- `hkrfi` Release erzwingen (Rollback, interaktiv).
