# Clear 별칭

빠른 화면 지우기 단축키로 터미널 워크플로를 간소화합니다.

## 개요

clear 별칭은 디렉토리 내용, 명령 히스토리 또는 파일 트리와 같은 유용한 컨텍스트를 선택적으로 표시하면서 터미널 디스플레이를 재설정하는 효율적인 방법을 제공합니다. 이 별칭은 `clear.aliases.sh`에 정의되어 있으며 chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `c` | 터미널 화면 지우기 |
| `cdw` | 작업 공간 디렉토리로 변경 |
| `cl` | 터미널 화면 지우기 (`c`의 별칭) |
| `clc` | 화면 지우고 디렉토리 내용 나열 |
| `clh` | 화면 지우고 명령 히스토리 표시 |
| `clp` | 화면 지우고 작업 디렉토리 출력 |
| `clt` | 화면 지우고 디렉토리 트리 표시 |
| `cpl` | 화면 지우고 작업 디렉토리와 내용 표시 |
| `cplt` | 화면 지우고 작업 디렉토리와 트리 표시 |
