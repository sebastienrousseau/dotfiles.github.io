# Update 별칭

시스템과 패키지를 최신 상태로 유지합니다.

## 개요

이 별칭은 `update.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

### macOS

| 별칭 | 설명 |
|:---|:---|
| `upd` | macOS 소프트웨어, Homebrew, Casks, App Store, Rust, Ruby, Python, Node.js 업데이트 |

### Linux

| 별칭 | 설명 |
|:---|:---|
| `open` | 선호하는 애플리케이션으로 파일 또는 URL 열기 |
| `pbcopy` | 클립보드에 복사 |
| `pbpaste` | 클립보드에서 붙여넣기 |
| `upd` | Linux 패키지, pnpm, Rust, Ruby, Python, Node.js 업데이트 |
