# Rsync 별칭

진행률 추적과 함께 위치 간 파일 및 디렉토리를 동기화합니다.

## 개요

이 별칭은 `rsync.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `rs` | 상세 출력 및 진행률 표시와 함께 Rsync |
