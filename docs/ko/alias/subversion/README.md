# Subversion 별칭

Subversion 버전 관리 작업을 위한 단축키.

## 개요

이 별칭은 `subversion.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `sad` | 파일과 디렉토리를 버전 관리에 추가 |
| `sau` | 캐시된 인증 정보 관리 |
| `sbl` | 각 줄이 마지막으로 변경된 시기 표시 |
| `scg` | 파일과 변경 목록 연결 |
| `sci` | 저장소에 변경 커밋 |
| `scl` | 작업 복사본 정리 |
| `sco` | 작업 복사본 체크아웃 |
| `scp` | 파일과 디렉토리 복사 |
| `sct` | 파일 또는 URL의 내용 출력 |
| `sdi` | 로컬 변경 또는 차이점 표시 |
| `sdl` | 버전 관리에서 파일 제거 |
| `shp` | 명령어 도움말 표시 |
| `sin` | 항목 정보 표시 |
| `sip` | 버전 관리되지 않은 파일을 저장소에 가져오기 |
| `slg` | 로그 메시지 표시 |
| `slk` | 작업 복사본 경로 잠금 |
| `sls` | 디렉토리 항목 나열 |
| `smd` | 버전 관리 하에 새 디렉토리 생성 |
| `smg` | 작업 복사본에 변경 병합 |
| `smgi` | 병합 관련 정보 표시 |
| `smv` | 항목 이동 또는 이름 변경 |
| `sp` | 속성 값 설정 |
| `spdl` | 속성 제거 |
| `spdt` | 외부 편집기로 속성 편집 |
| `spgt` | 속성 값 출력 |
| `sph` | 패치 적용 |
| `spls` | 모든 속성 나열 |
| `srl` | 작업 복사본을 다른 URL로 재배치 |
| `srs` | 충돌 해결 |
| `srsd` | 충돌 상태 제거 |
| `srv` | 원본 작업 복사본 상태 복원 |
| `sst` | 작업 복사본 상태 출력 |
| `ssw` | 작업 복사본을 다른 URL로 전환 |
| `sulk` | 작업 복사본 경로 잠금 해제 |
| `sup` | 저장소에서 작업 복사본 업데이트 |
| `supg` | 메타데이터 저장 형식 업그레이드 |
| `sxp` | 버전 관리되지 않은 트리 복사본 생성 |
