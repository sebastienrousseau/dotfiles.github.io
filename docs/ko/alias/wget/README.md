# Wget 별칭

이어받기 지원과 함께 웹에서 파일을 다운로드합니다.

## 개요

이 별칭은 `wget.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `wg` | Wget 단축형 |
| `wgc` | 이어받기와 함께 Wget |
| `wge` | robots=off와 함께 Wget |
| `wget` | 이어받기와 함께 Wget |
