# macOS 별칭

macOS 시스템 관리 및 유지보수를 위한 셸 단축키.

## 개요

이 별칭은 `macOS.aliases.sh`에 정의되어 있으며 chezmoi에 의해 자동으로 로드됩니다. Finder 제어, 시스템 유지보수, 개발자 도구를 포함한 일반적인 macOS 작업에 빠르게 접근할 수 있습니다.

## 참조

### 시스템 관리

| 별칭 | 설명 |
|:---|:---|
| `lockScreen` | 화면 잠금 |
| `wifiOn` | Wi-Fi 켜기 |
| `wifiOff` | Wi-Fi 끄기 |
| `vp` | macOS 권한 확인 |
| `vv` | macOS 볼륨 확인 |

### Finder

| 별칭 | 설명 |
|:---|:---|
| `ofd` | Finder에서 현재 디렉토리 열기 |
| `finderShowHidden` | Finder에서 숨김 파일 표시 |
| `finderHideHidden` | Finder에서 숨김 파일 숨기기 |
| `showDesktopIcons` | 바탕화면의 모든 아이콘 표시 |
| `hideDesktopIcons` | 바탕화면의 모든 아이콘 숨기기 |

### 정리

| 별칭 | 설명 |
|:---|:---|
| `clds` | .DS_Store 파일 재귀적 삭제 |
| `clls` | 다음으로 열기 메뉴에서 중복 제거를 위해 LaunchServices 정리 |
| `trash` | 모든 마운트된 볼륨과 메인 HDD의 휴지통 비우기 |

### 개발자 도구

| 별칭 | 설명 |
|:---|:---|
| `xcode` | Xcode 실행 |
| `purge` | Xcode DerivedData 삭제 |
| `iphone` | 장치 시뮬레이터 열기 |
| `safariSafeMode` | 안전 모드로 Safari 실행 |

### 디스플레이

| 별칭 | 설명 |
|:---|:---|
| `screensaverDesktop` | 바탕화면에서 화면 보호기 실행 |
