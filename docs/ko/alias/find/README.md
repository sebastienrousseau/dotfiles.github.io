# Find 별칭

`fd` 유틸리티를 사용한 빠른 파일 검색. `find` 명령의 현대적 대안.

## 개요

이 별칭은 `find.aliases.sh`에 정의되어 있으며 chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `fd` | 컬러 출력으로 모든 파일 나열 |
| `fda` | 절대 경로로 모든 파일 나열 |
| `fdc` | 대소문자 구분 없이 모든 파일 나열 |
| `fdd` | 상세 정보와 함께 모든 파일 나열 |
| `fde` | 특정 확장자의 모든 파일 나열 |
| `fdf` | 심볼릭 링크를 따라 모든 파일 나열 |
| `fdh` | fd 도움말 표시 |
| `fdn` | 특정 glob과 일치하는 모든 파일 나열 |
| `fdo` | 소유자 정보와 함께 모든 파일 나열 |
| `fds` | 크기와 함께 모든 파일 나열 |
| `fdu` | 제외 규칙과 함께 모든 파일 나열 |
| `fdv` | fd 버전 표시 |
| `fdx` | 각 검색 결과에 대해 명령 실행 |
