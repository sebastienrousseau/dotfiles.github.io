# Sudo 별칭

상승된 권한으로 명령어를 실행합니다.

## 개요

이 별칭은 `sudo.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `root` | 슈퍼유저로 명령어 실행 |
| `s` | 슈퍼유저로 명령어 실행 |
| `su` | 슈퍼유저로 명령어 실행 |
| `sudo` | 슈퍼유저로 명령어 실행 |
