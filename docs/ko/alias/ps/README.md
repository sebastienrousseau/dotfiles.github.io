# PS 별칭

간단한 단축키로 실행 중인 프로세스를 모니터링하고 관리합니다.

## 개요

이 별칭은 `ps.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `pid` | UID, PID, 부모 PID, CPU 사용량, 시작 시간, TTY, 명령어 표시 |
| `ps` | 모든 프로세스 표시 |
| `psa` | 모든 프로세스 나열 |
