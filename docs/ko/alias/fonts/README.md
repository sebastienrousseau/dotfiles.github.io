# 폰트 별칭

시스템 폰트와 캐시 관리를 위한 단축키.

## 개요

이 별칭은 `fonts.aliases.sh`에 정의되어 있으며 chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `update-fonts` | 시스템 폰트 캐시 업데이트 (`fc-cache -fv`) |
| `list-fonts` | 설치된 모든 폰트 패밀리 나열 |

## 설치된 폰트

- **JetBrainsMono Nerd Font** - 기본 터미널 폰트
- **Symbols Nerd Font** - 아이콘 폴백
