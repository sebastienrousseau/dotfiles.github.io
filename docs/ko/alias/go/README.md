# Go 별칭

직관적인 셸 단축키로 Go 개발 워크플로를 간소화합니다.

## 개요

이 별칭은 `go.aliases.sh`에 정의되어 있으며 chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 명령 | 설명 |
|:---|:---|:---|
| `go` | `go` | Go 단축키 |
| `gor` | `go run` | Go 프로그램 실행 |
| `gob` | `go build` | Go 패키지 빌드 |
| `got` | `go test` | Go 테스트 실행 |
| `gomt` | `go mod tidy` | Go 모듈 의존성 정리 |
| `goget` | `go get` | Go 패키지 다운로드 및 설치 |
