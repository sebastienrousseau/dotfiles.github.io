# UUID 별칭

범용 고유 식별자를 생성합니다.

## 개요

이 별칭은 `uuid.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `uuid` | UUID를 생성하고 클립보드에 복사 |
