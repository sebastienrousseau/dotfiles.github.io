# Terraform 별칭

Terraform 및 Ansible을 사용한 Infrastructure as Code를 위한 단축키.

## 개요

이 별칭은 `terraform.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

### Terraform

| 별칭 | 설명 |
|:---|:---|
| `tf` | Terraform 단축형 |
| `tfi` | 구성 초기화 |
| `tfp` | 변경 계획 |
| `tfa` | 변경 적용 |
| `tfaa` | 자동 승인과 함께 적용 |
| `tfd` | 인프라 파괴 |
| `tff` | 구성 포맷 |
| `tfv` | 구성 검증 |

### Ansible

| 별칭 | 설명 |
|:---|:---|
| `ans` | Ansible 단축형 |
| `ansp` | Ansible Playbook 실행 |
| `anslint` | Ansible 파일 린트 |
