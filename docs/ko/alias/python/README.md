# Python 별칭

Python 개발을 위한 단축키. 인터프리터, 패키지, 가상 환경.

## 개요

이 별칭은 `python.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 명령 | 설명 |
|:---|:---|:---|
| `py` | `python` | Python 인터프리터 |
| `pip` | `pip3` | 패키지 설치 프로그램 |
| `v` | `virtualenv` | 가상 환경 |
| `uv` | `uv` | 빠른 패키지 관리자 |
| `uvpi` | `uv pip install` | uv로 설치 |
