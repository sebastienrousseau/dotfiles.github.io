# AI 별칭

AI 및 LLM 도구 단축키. AI 지원 워크플로를 가속화합니다.

## 개요

이 별칭은 `ai.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

### GitHub Copilot

| 별칭 | 설명 |
|:---|:---|
| `ghcp` | Copilot 단축키 |
| `ghs` | 코드 제안 |
| `ghe` | 코드 설명 |

### Ollama

| 별칭 | 설명 |
|:---|:---|
| `ol` | Ollama 단축키 |
| `olr` | 모델 실행 |
