# Installer 별칭

부트스트랩 및 원격 배포를 위한 도구.

## 개요

installer 별칭은 dotfiles 환경을 로컬에서 설정하고 원격 서버에 구성을 배포하기 위한 간소화된 명령을 제공합니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `dot-install` | 로컬에서 `install.sh` 부트스트랩 스크립트 실행. |
| `dot-teleport` | 원격 서버로 설정 전송 (`dot-teleport user@host`). |
