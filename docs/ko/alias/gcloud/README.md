# Google Cloud 별칭

Google Cloud 서비스 관리를 위한 직관적인 단축키.

## 개요

이 별칭은 `gcloud.aliases.sh`에 정의되어 있으며 chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `gclb` | Google Cloud Build 관리 |
| `gcca` | Compute Engine IP 주소 관리 |
| `gccc` | 새 가상 머신 인스턴스 생성 |
| `gcco` | SSH를 사용하여 가상 머신 인스턴스에 연결 |
| `gcd` | 기본 프로젝트를 현재 디렉토리 이름으로 설정 |
| `gcf` | Google Cloud Functions 관리 |
| `gci` | Google Cloud Compute Engine 인스턴스 관리 |
| `gck` | 모든 설정 나열 |
| `gcn` | Google Cloud Networks 관리 |
| `gcp` | Google Cloud 프로젝트 관리 |
| `gcpd` | Google Cloud 프로젝트 삭제 |
| `gcr` | Google Container Registry에서 컨테이너 이미지 삭제 |
| `gcro` | Google Cloud Run 관리 |
| `gcs` | Google Cloud Kubernetes Engine 클러스터 관리 |
| `gcsa` | 현재 설정의 계정 설정 |
| `gcso` | 현재 프로젝트의 Google Cloud Console 열기 |
| `gcsq` | Google Cloud SQL 관리 |
| `gcss` | Google Cloud Storage 관리 |
| `gct` | Google Cloud Tasks 관리 |
