# Yarn 별칭

Yarn JavaScript 패키지 관리자를 위한 단축키.

## 개요

이 별칭은 `yarn.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `y` | Yarn 단축형 |
| `yi` | 의존성 설치 |
| `ya` | 의존성 추가 |
| `yad` | 개발 의존성 추가 |
| `yrun` | 스크립트 실행 |
| `ystem` | 프로젝트 시작 |
