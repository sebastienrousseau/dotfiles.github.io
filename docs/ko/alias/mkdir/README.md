# Mkdir 별칭

지능형 명명 규칙으로 디렉토리 생성을 간소화합니다.

## 개요

mkdir 별칭은 자동 날짜 및 시간 스탬프로 디렉토리를 생성하는 단축키를 제공합니다. `mkdir.aliases.sh`에 정의되어 있으며 chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `md` | 디렉토리 생성 |
| `mdd` | 날짜가 포함된 디렉토리 생성 |
| `mdt` | 시간이 포함된 디렉토리 생성 |
| `mde` | 현재 날짜로 예제 디렉토리 생성 |
| `mdn` | 현재 날짜로 노트 디렉토리 생성 |
| `mdw` | 현재 날짜로 작업 디렉토리 생성 |
