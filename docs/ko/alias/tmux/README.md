# Tmux 별칭

tmux로 터미널 세션, 창, 패널을 관리합니다.

## 개요

이 별칭은 `tmux.aliases.sh`에 정의되어 있으며 Chezmoi에 의해 자동으로 로드됩니다.

## 참조

| 별칭 | 설명 |
|:---|:---|
| `tm` | tmux 시작 |
| `tma` | 마지막 세션에 연결 |
| `tmat` | 특정 세션에 연결 |
| `tmks` | 현재를 제외한 모든 세션 종료 |
| `tmka` | 모든 세션 종료 |
| `tml` | 모든 세션 나열 |
| `tmn` | 새 이름 없는 세션 |
| `tms` | 새 명명된 세션 |
| `tmr` | tmux 구성 다시 로드 |
| `tmls` | 창 나열 |
| `tmlp` | 패널 나열 |
| `tmi` | tmux 정보 표시 |
