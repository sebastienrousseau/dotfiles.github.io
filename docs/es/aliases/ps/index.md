# Alias de PS

Monitorea y gestiona procesos en ejecución con atajos simples.

## Descripción general

Estos alias están definidos en `ps.aliases.sh` y se cargan automáticamente por Chezmoi.

## Referencia

| Alias | Descripción |
|:---|:---|
| `pid` | Mostrar UID, PID, PID padre, uso de CPU, hora de inicio, TTY y comando |
| `ps` | Mostrar todos los procesos |
| `psa` | Listar todos los procesos |
