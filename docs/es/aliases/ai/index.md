# Alias IA

Atajos para herramientas de IA y LLM. Acelera tu flujo de trabajo asistido por IA.

## Descubrir

Estos alias se definen en `ai.aliases.sh` y se cargan automáticamente por Chezmoi.

## Referencia

### GitHub Copilot

| Alias | Descripción |
|:---|:---|
| `ghcp` | Atajo de Copilot |
| `ghs` | Sugerir código |
| `ghe` | Explicar código |

### Ollama

| Alias | Descripción |
|:---|:---|
| `ol` | Atajo de Ollama |
| `olr` | Ejecutar modelo |
