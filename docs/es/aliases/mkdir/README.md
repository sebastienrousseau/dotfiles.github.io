# Alias de Mkdir

Simplifica la creación de directorios con convenciones de nombres inteligentes.

## Descripción general

Los alias de mkdir proporcionan atajos para crear directorios con marcas automáticas de fecha y hora. Estos alias están definidos en `mkdir.aliases.sh` y se cargan automáticamente por chezmoi.

## Referencia

| Alias | Descripción |
|-------|-------------|
| `md` | Crear directorio |
| `mdd` | Crear directorio con fecha |
| `mdt` | Crear directorio con hora |
| `mde` | Crear directorio de ejemplo con fecha actual |
| `mdn` | Crear directorio de notas con fecha actual |
| `mdw` | Crear directorio de trabajo con fecha actual |
