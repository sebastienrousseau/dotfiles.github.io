# Alias Installer

Herramientas para bootstrap y despliegue remoto.

## Descubrir

Los alias installer proporcionan comandos simplificados para configurar tu entorno dotfiles en local y desplegar tu configuración en servidores remotos.

## Referencia

| Alias | Descripción |
|-------|-------------|
| `dot-install` | Ejecutar el script `install.sh` localmente. |
| `dot-teleport` | Teletransportar tu configuración a un servidor remoto (`dot-teleport user@host`). |
