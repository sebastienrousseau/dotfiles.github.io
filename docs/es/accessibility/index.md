# Accesibilidad

- Se admite la navegación por teclado.
- Los indicadores de enfoque son visibles.
- Se respetan las preferencias de movimiento reducido.

Informar problemas de accesibilidad: [info@dotfiles.io](mailto:info@dotfiles.io).
