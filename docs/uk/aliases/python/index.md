# Аліаси Python

Короткі команди Python.

Ця сторінка знайомить з однією групою аліасів та її поширеними скороченнями команд.
Прочитайте її спочатку, щоб зрозуміти угоди про іменування, очікувані вхідні дані та безпечні шаблони виконання.
Потім відкрийте довідковий додаток для повного переліку команд та точного синтаксису.
Ці аліаси зменшують повторний набір та зберігають послідовну поведінку робочого процесу на macOS, Linux та WSL.

## Коли використовувати

Використовуйте цю сторінку перед щоденною роботою в оболонці, коли вам потрібні передбачувані шаблони команд.

## Швидкі приклади

- `black` → `'python -m black' # Форматування коду з black`
- `deact` → `'deactivate'                     # Деактивація venv`
- `ipy` → `'ipython'           # Інтерактивна оболонка Python`
- `lint` → `'pylint'           # Лінтинг коду`
- `mkvenv` → `'python -m venv .venv'          # Створити .venv у поточному каталозі`
- `mkvenvlocal` → `'python -m venv ./venv'    # Створити ./venv у поточному каталозі`
- `mypy` → `'python -m mypy'   # Статична перевірка типів`
- `pep8` → `'autopep8'         # Форматування коду`
- `pipf` → `'pip freeze'                      # Показати заморожені залежності`
- `pipi` → `'pip install'                     # Встановити пакети`
- `pipl` → `'pip list'                        # Список встановлених пакетів`
- `pipout` → `'pip freeze > requirements.txt' # Зберегти залежності`
