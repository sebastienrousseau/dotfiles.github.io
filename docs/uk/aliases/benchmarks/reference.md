# Довідковий додаток Benchmarks

## Вихідні файли

- `.chezmoitemplates/aliases/benchmarks/benchmarks.aliases.sh`

## Довідник псевдонімів

| Псевдонім | Команда |
| :--- | :--- |
| `bench` | `hyperfine` |
| `benchw` | `hyperfine --warmup 3` |

## Перевірка

- Виконайте `chezmoi apply` після оновлення псевдонімів.
- Виконайте `dot doctor` для перевірки стану оболонки.
