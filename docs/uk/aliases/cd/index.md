# Аліаси Cd

Короткі команди переміщення по каталогах.

Ця сторінка знайомить з однією групою аліасів та її поширеними скороченнями команд.
Прочитайте її спочатку, щоб зрозуміти угоди про іменування, очікувані вхідні дані та безпечні шаблони виконання.
Потім відкрийте довідковий додаток для повного переліку команд та точного синтаксису.
Ці аліаси зменшують повторний набір та зберігають послідовну поведінку робочого процесу на macOS, Linux та WSL.

## Коли використовувати

Використовуйте цю сторінку перед щоденною роботою в оболонці, коли вам потрібні передбачувані шаблони команд.

## Що включено

- Аліаси: 16
- Допоміжні функції: 6
- Файли джерел: 12

## Швидкі приклади

- `bm` → `'bm add'            # Створити закладку`
- `bmg` → `'cd "$(bm goto $1)"' # Перейти до закладки (потребує shell cd)`
- `bml` → `'bm list'           # Список закладок`
- `bmr` → `'bm remove'         # Видалити закладку`
- `bmu` → `'bm update'         # Оновити закладку`
- `cd` → `'cd_with_history' # Замінити типову команду cd`
- `cdh` → `'cd_with_history' # Явний покращений хелпер cd`
- `cdhelp` → `cd_aliases_help`
- `cdversion` → `cd_aliases_version`
- `dh` → `'dirhistory' # Показати історію каталогів`
- `dirs` → `'dirs -v'        # Список стеку каталогів з індексами`
- `ld` → `'lwd'        # Повернутися до останнього каталогу`
