# Аліаси архівів

Короткі імена команд архівування.

Ця сторінка знайомить з однією групою аліасів та її поширеними скороченнями команд.
Прочитайте її спочатку, щоб зрозуміти угоди про іменування, очікувані вхідні дані та безпечні шаблони виконання.
Потім відкрийте довідковий додаток для повного переліку команд та точного синтаксису.
Ці аліаси зменшують повторний набір та зберігають послідовну поведінку робочого процесу на macOS, Linux та WSL.

## Коли використовувати

Використовуйте цю сторінку перед щоденною роботою в оболонці, коли вам потрібні передбачувані шаблони команд.

## Що включено

- Аліаси: 40
- Допоміжні функції: 3
- Файли джерел: 1

## Швидкі приклади

- `ac` → `'compress'        # Загальне стиснення (Archive Create)`
- `acl` → `'compress_large' # Застаріле compress_large`
- `bak` → `'backup'         # Швидке резервне копіювання`
- `c7z` → `'7z a' # Створити архів 7z`
- `cbz` → `'bzip2 -zk' # Стиснути з bzip2`
- `cgz` → `'gzip -cv' # Стиснути з gzip`
- `clz4` → `'lz4 -zc' # Стиснути з lz4`
- `crar` → `'rar a' # Створити архів rar`
- `ctar` → `'tar -cvf'         # Створити архів tar`
- `ctbz` → `'tar -jcvf'        # Створити архів tar.bz2`
- `ctgz` → `'tar -zcvf'        # Створити архів tar.gz`
- `ctxz` → `'tar -Jcvf'        # Створити архів tar.xz`
