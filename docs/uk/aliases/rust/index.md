# Аліаси Rust

Короткі команди Rust.

Ця сторінка знайомить з однією групою аліасів та її поширеними скороченнями команд.
Прочитайте її спочатку, щоб зрозуміти угоди про іменування, очікувані вхідні дані та безпечні шаблони виконання.
Потім відкрийте довідковий додаток для повного переліку команд та точного синтаксису.
Ці аліаси зменшують повторний набір та зберігають послідовну поведінку робочого процесу на macOS, Linux та WSL.

## Коли використовувати

Використовуйте цю сторінку перед щоденною роботою в оболонці, коли вам потрібні передбачувані шаблони команд.

## Що включено

- Аліаси: 75
- Допоміжні функції: 0
- Файли джерел: 1

## Швидкі приклади

- `cg` → `'cargo'          # Скорочення Cargo`
- `cga` → `'cg add'        # Додати залежність`
- `cgad` → `'cg add --dev' # Додати dev-залежність`
- `cgaud` → `'cg audit'    # Перевірка безпекових вразливостей`
- `cgb` → `'cg build'            # Збірка debug`
- `cgba` → `'cg build --all-targets'       # Збірка всіх цілей`
- `cgbh` → `'cg bench'                    # Запуск бенчмарків`
- `cgbl` → `'cg bloat'      # Аналіз розміру бінарника`
- `cgbp` → `'cg build --release --profile' # Збірка з профілем`
- `cgbr` → `'cg build --release' # Збірка release`
- `cgbt` → `'cg build --all-features'      # Збірка з усіма feature`
- `cgc` → `'cg check'      # Перевірка компіляції`
