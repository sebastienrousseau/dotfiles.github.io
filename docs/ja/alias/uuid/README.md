# UUIDエイリアス

ユニバーサリー一意識別子を生成。

## 概要

これらのエイリアスは`uuid.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `uuid` | UUIDを生成してクリップボードにコピー |
