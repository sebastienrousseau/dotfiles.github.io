# Wgetエイリアス

レジュームサポート付きでWebからファイルをダウンロード。

## 概要

これらのエイリアスは`wget.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `wg` | Wgetショートカット |
| `wgc` | 続行付きWget |
| `wge` | robots=offでWget |
| `wget` | 続行付きWget |
