# Goエイリアス

直感的なシェルショートカットでGo開発ワークフローを効率化。

## 概要

これらのエイリアスは`go.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。実行、ビルド、テスト、モジュール管理の一般的なGoコマンドに素早くアクセスできます。

## リファレンス

| エイリアス | コマンド | 説明 |
|:---|:---|:---|
| `go` | `go` | Goショートカット |
| `gor` | `go run` | Goプログラムを実行 |
| `gob` | `go build` | Goパッケージをビルド |
| `got` | `go test` | Goテストを実行 |
| `gomt` | `go mod tidy` | Goモジュールの依存関係を整理 |
| `goget` | `go get` | Goパッケージをダウンロードしてインストール |
