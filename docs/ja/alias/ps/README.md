# PSエイリアス

シンプルなショートカットで実行中のプロセスを監視・管理。

## 概要

これらのエイリアスは`ps.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `pid` | UID、PID、親PID、CPU使用率、開始時刻、TTY、コマンドを表示 |
| `ps` | すべてのプロセスを表示 |
| `psa` | すべてのプロセスを一覧表示 |
