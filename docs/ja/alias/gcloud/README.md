# GCloudエイリアス

Google Cloudプラットフォームを管理するためのシェルショートカット。

## 概要

これらのエイリアスは`gcloud.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `gclb` | Google Cloud Buildを管理 |
| `gcca` | Compute EngineのIPアドレスを管理 |
| `gccc` | 新しい仮想マシンインスタンスを作成 |
| `gcco` | SSHで仮想マシンに接続 |
| `gcd` | デフォルトプロジェクトを現在のディレクトリ名に設定 |
| `gcf` | Google Cloud Functionsを管理 |
| `gci` | Compute Engineインスタンスを管理 |
| `gck` | すべての構成を一覧表示 |
| `gcn` | Google Cloud Networksを管理 |
| `gcp` | Google Cloudプロジェクトを管理 |
| `gcpd` | Google Cloudプロジェクトを削除 |
| `gcps` | Google Cloud Pub/Subを管理 |
| `gcs` | Kubernetes Engineクラスタを管理 |
| `gcsa` | 現在の構成のアカウントを設定 |
| `gcso` | 現在のプロジェクトのCloud Consoleを開く |
| `gcsq` | Google Cloud SQLを管理 |
| `gcss` | Google Cloud Storageを管理 |
| `gcst` | Google Cloudサービスを有効または無効にする |
