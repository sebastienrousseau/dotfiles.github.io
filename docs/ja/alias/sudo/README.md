# Sudoエイリアス

昇格された権限でコマンドを実行。

## 概要

これらのエイリアスは`sudo.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `root` | スーパーユーザーとしてコマンドを実行 |
| `s` | スーパーユーザーとしてコマンドを実行 |
| `su` | スーパーユーザーとしてコマンドを実行 |
| `sudo` | スーパーユーザーとしてコマンドを実行 |
