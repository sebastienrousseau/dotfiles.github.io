# Disk Usageエイリアス

ディスク使用量を人間が読める形式で表示するためのショートカット。

## 概要

これらのエイリアスは`disk-usage.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `du` | 現在のディレクトリのディスク使用量を表示 |
| `du1` | 現在のディレクトリ内のファイルとディレクトリのサイズを表示 |
| `duh` | ファイルとディレクトリのサイズを表示 |
| `ducks` | 上位10個の最大ファイルとディレクトリを表示 |
| `dus` | サイズでソートして人間が読める形式で表示 |
| `dusym` | シンボリックリンクを含むサイズを表示 |
| `dut` | 現在のディレクトリの合計サイズを表示 |
