# Tmuxエイリアス

tmuxでターミナルセッション、ウィンドウ、ペインを管理。

## 概要

これらのエイリアスは`tmux.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `tm` | tmuxを開始 |
| `tma` | 最後のセッションにアタッチ |
| `tmat` | 特定のセッションにアタッチ |
| `tmks` | 現在以外のすべてのセッションを終了 |
| `tmka` | すべてのセッションを終了 |
| `tml` | すべてのセッションを一覧表示 |
| `tmn` | 無名の新しいセッション |
| `tms` | 名前付きの新しいセッション |
| `tmr` | tmux設定をリロード |
| `tmls` | ウィンドウを一覧表示 |
| `tmlp` | ペインを一覧表示 |
| `tmi` | tmux情報を表示 |
