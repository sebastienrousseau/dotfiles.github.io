# AIエイリアス

AIおよびLLMツールのショートカット。AI支援ワークフローを加速します。

## 概要

これらのエイリアスは`ai.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

### GitHub Copilot

| エイリアス | 説明 |
|:---|:---|
| `ghcp` | Copilotショートカット |
| `ghs` | コードを提案 |
| `ghe` | コードを説明 |

### Ollama

| エイリアス | 説明 |
|:---|:---|
| `ol` | Ollamaショートカット |
| `olr` | モデルを実行 |
