# インストーラーエイリアス

ブートストラップとリモートデプロイメントのためのツール。

## 概要

インストーラーエイリアスは、dotfiles環境をローカルでセットアップし、リモートサーバーに設定をデプロイするためのストリームラインされたコマンドを提供します。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `dot-install` | `install.sh`ブートストラップスクリプトをローカルで実行 |
| `dot-teleport` | リモートサーバーに設定をテレポート（`dot-teleport user@host`） |
