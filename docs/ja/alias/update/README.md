# 更新エイリアス

システムとパッケージを最新の状態に保つ。

## 概要

これらのエイリアスは`update.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

### macOS

| エイリアス | 説明 |
|:---|:---|
| `upd` | macOSソフトウェア、Homebrew、Cask、App Store、Rust、Ruby、Python、Node.jsを更新 |

### Linux

| エイリアス | 説明 |
|:---|:---|
| `open` | 好みのアプリケーションでファイルまたはURLを開く |
| `pbcopy` | クリップボードにコピー |
| `pbpaste` | クリップボードから貼り付け |
| `upd` | Linuxパッケージ、pnpm、Rust、Ruby、Python、Node.jsを更新 |
