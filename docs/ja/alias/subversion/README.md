# Subversionエイリアス

Subversionバージョン管理操作のためのショートカット。

## 概要

これらのエイリアスは`subversion.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `sad` | ファイルとディレクトリをバージョン管理に追加 |
| `sau` | キャッシュされた認証情報を管理 |
| `sbl` | 各行が最後に変更された時期を表示 |
| `scg` | チェンジリストをファイルに関連付け |
| `sci` | リポジトリに変更をコミット |
| `scl` | ワーキングコピーをクリーンアップ |
| `sco` | ワーキングコピーをチェックアウト |
| `scp` | ファイルとディレクトリをコピー |
| `sct` | ファイルまたはURLの内容を出力 |
| `sdi` | ローカルな変更または差分を表示 |
| `sdl` | バージョン管理からファイルを削除 |
| `shp` | コマンドのヘルプを表示 |
| `sin` | アイテムに関する情報を表示 |
| `sip` | バージョン管理されていないファイルをリポジトリにインポート |
| `slg` | ログメッセージを表示 |
| `slk` | ワーキングコピーパスをロック |
| `sls` | ディレクトリエントリを一覧表示 |
| `smd` | バージョン管理下に新しいディレクトリを作成 |
| `smg` | ワーキングコピーに変更をマージ |
| `smgi` | マージ関連情報を表示 |
| `smv` | アイテムを移動または名前変更 |
| `sp` | プロパティ値を設定 |
| `spdl` | プロパティを削除 |
| `spdt` | 外部エディタでプロパティを編集 |
| `spgt` | プロパティ値を表示 |
| `sph` | パッチを適用 |
| `spls` | すべてのプロパティを一覧表示 |
| `srl` | ワーキングコピーを別のURLに再配置 |
| `srs` | 競合を解決 |
| `srsd` | 競合状態を削除 |
| `srv` | クリーンなワーキングコピー状態を復元 |
| `sst` | ワーキングコピーのステータスを表示 |
| `ssw` | ワーキングコピーを別のURLに切り替え |
| `sulk` | ワーキングコピーパスのロックを解除 |
| `sup` | リポジトリからワーキングコピーを更新 |
| `supg` | メタデータストレージ形式をアップグレード |
| `sxp` | ツリーのバージョン管理されていないコピーを作成 |
