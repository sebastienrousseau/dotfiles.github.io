# Mkdirエイリアス

インテリジェントな命名規則でディレクトリ作成を効率化。

## 概要

mkdirエイリアスは、自動的な日付と時刻スタンプ付きでディレクトリを作成するショートカットを提供します。これらのエイリアスは`mkdir.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `md` | ディレクトリを作成 |
| `mdd` | 日付付きでディレクトリを作成 |
| `mdt` | 時刻付きでディレクトリを作成 |
| `mde` | 現在の日付でサンプルディレクトリを作成 |
| `mdn` | 現在の日付でノートディレクトリを作成 |
| `mdw` | 現在の日付で作業ディレクトリを作成 |
