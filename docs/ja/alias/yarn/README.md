# Yarnエイリアス

Yarn JavaScriptパッケージマネージャーのためのショートカット。

## 概要

これらのエイリアスは`yarn.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|:---|:---|
| `y` | Yarnショートカット |
| `yi` | 依存関係をインストール |
| `ya` | 依存関係を追加 |
| `yad` | 開発依存関係を追加 |
| `yrun` | スクリプトを実行 |
| `ystem` | プロジェクトを開始 |
