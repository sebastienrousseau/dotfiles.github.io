# Pythonエイリアス

Python開発用ショートカット。インタープリター、パッケージ、仮想環境。

## 概要

これらのエイリアスは`python.aliases.sh`で定義され、Chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | コマンド | 説明 |
|:---|:---|:---|
| `py` | `python` | Pythonインタープリター |
| `pip` | `pip3` | パッケージインストーラー |
| `v` | `virtualenv` | 仮想環境 |
| `uv` | `uv` | 高速パッケージマネージャー |
| `uvpi` | `uv pip install` | uvでインストール |
