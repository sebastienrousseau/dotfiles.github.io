# Clearエイリアス

クイックスクリーンクリアショートカットでターミナルワークフローを効率化。

## 概要

clearエイリアスは、ディレクトリ内容、コマンド履歴、ファイルツリーなどの有用なコンテキストをオプションで表示しながら、ターミナル表示をリセットする効率的な方法を提供します。これらのエイリアスは`clear.aliases.sh`で定義され、chezmoiによって自動的にロードされます。

## リファレンス

| エイリアス | 説明 |
|-------|-------------|
| `c` | ターミナル画面をクリア |
| `cdw` | ワークスペースディレクトリに移動 |
| `cl` | ターミナル画面をクリア（`c`のエイリアス） |
| `clc` | 画面をクリアしてディレクトリ内容を一覧表示 |
| `clh` | 画面をクリアしてコマンド履歴を表示 |
| `clp` | 画面をクリアして作業ディレクトリを表示 |
| `clt` | 画面をクリアしてディレクトリツリーを表示 |
| `cpl` | 画面をクリアして作業ディレクトリと内容を表示 |
| `cplt` | 画面をクリアして作業ディレクトリとツリーを表示 |
