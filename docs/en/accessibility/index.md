# Accessibility Statement

Dotfiles is designed to meet high accessibility standards for keyboard, screen reader, and low-vision users.

## Standards target

- WCAG 2.2 AA baseline across all documentation templates.
- AAA intent for contrast, focus visibility, and reduced motion where practical.

## Current limitations

- Legacy content tables and code examples may include occasional formatting patterns that need manual refinement.
- Third-party browser extensions can affect color and focus rendering.

## Report an issue

If you find an accessibility issue, contact [info@dotfiles.io](mailto:info@dotfiles.io) with:

- page URL
- assistive technology used
- reproduction steps

We prioritize accessibility defects as release blockers.
