# Privacy Policy

This documentation site is a static site intended to minimize personal data collection.

## Data we may process

- Standard web server logs (IP address, user agent, request path, timestamp).
- Optional GitHub analytics or hosting metrics in aggregate form.

## Data use

- Site reliability and security monitoring.
- Performance tuning and documentation quality improvements.

## Data sharing

No personal data is sold. Data may be processed by infrastructure providers required to host the site.

## Contact

For privacy requests, email [info@dotfiles.io](mailto:info@dotfiles.io).
