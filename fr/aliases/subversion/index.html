# Les alias Subversion

Le fichier subversion.aliases.sh crée des raccourcis utiles pour travailler avec les commandes Subversion (opens new window).

Les alias Subversion sont une collection d'alias qui vous permettent d'interagir avec la commande svn. Subversion est un système de contrôle de version, qui vous permet de garder des anciennes versions de fichiers et de répertoires (généralement du code source), de garder un journal de qui, quand et pourquoi les modifications ont eu lieu.
