# Les alias Heroku

Le fichier heroku.aliases.zsh crée des raccourcis utiles pour de nombreuses commandes courantes de Heroku (opens new window).

Les alias Heroku sont des enveloppes autour de la CLI Heroku. Heroku est une plateforme de cloud computing qui permet de déployer des applications web.

# Raccourcis Heroku
