# Les alias GNU Coreutils

Le fichier gnucoreutils.aliases.bash contient des alias pour les utilitaires GNU Coreutils (opens new window), qui sont installés par défaut sur la plupart des systèmes Linux.

Les alias des utilitaires de base GNU, fournissent des commandes puissantes qui créent des raccourcis vers des commandes d'utilitaires de base fréquemment utilisées. Toutes ces fonctions et alias sont des enveloppes autour des GNU Core Utilities qui sont les outils de base, fondamentaux fondamentaux d'un système GNU/Linux.
